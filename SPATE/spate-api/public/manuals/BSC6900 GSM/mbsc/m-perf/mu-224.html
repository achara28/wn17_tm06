PPP Link Measurement(PPP)

Overview

The quality of transmission links directly affects the system performance. The PPP link measurement helps analyze system transmission quality so as to maintain system stability and reliability.

Figure 1 The four layers of the TCP/IP protocol suite

PPP link belongs to the Link layer of the TCP/IP protocol suite, as figure 1 shows.

Figure 2 Physical bearer of the PPP Link

In the configuration structure, PPP links are between physical ports and logical applications, as shown in the figure 2.
