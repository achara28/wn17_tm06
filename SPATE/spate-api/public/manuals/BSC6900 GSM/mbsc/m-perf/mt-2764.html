Number of Downlink GPRS TBF Normal Releases

Measurement Counters

ID Counter Description
1279174421 DOWN.GPRS.TBF.NORMAL.REL A9105:Number of Downlink GPRS TBF Normal Releases

Description

This measurement provides the number of normal downlink GPRS TBF releases in a granularity period. A low measurement value indicates that the radio environment, network parameter configuration, or network resource configuration is not satisfactory, and thus requires optimization. Further analysis should be made based on the causes of GPRS TBF abnormal release.

Measurement point

The BSC initiates the downlink GPRS TBF release procedure by identifying FBI = 1 contained in the last downlink block and reserves the blocks for the MS to respond with the assignment acknowledgement message. When the BSC receives the PACKET DOWNLINK ACK/NACK message with FAI = 1 from the MS on the reserved block, it indicates that the MS receives all the downlink blocks. Then, the BSC releases the downlink TBF. Figure 1 shows the successful downlink GPRS TBF release procedure. Each time the BSC receives a PACKET DOWNLINK ACK/NACK message with FAI = 1 from the MS (see measurement point A), the counter is incremented by one.
