Number Of PDCH Application Attempts

Measurement Counters

ID Counter Description
1282411898 ATTEMPT.APPLY.PDCH R9393:Number Of PDCH Application Attempts

Description

This counter provides the number of attempts to apply for the PDCH within a measurement period.

Measurement point

The application for the PDCH is performed in the following cases:

1. The number of the assigned PDCHs does not meet the requirement for the multislot capability of the MS.

2. The number of assigned PDCHs reaches the threshold of the dynamic channel conversion.

3. The application for the PDCH is triggered according to the setting of the Number of Dynamic Channel Pre-Converted parameter.

4. The application for the PDCH is triggered due to NACC cell reselection.

5. The application for the PDCH is triggered due to single block assignment failure.
