Number of Failed Downlink DTM Assignments Received on DCCH

Measurement Counters

ID Counter Description
1279189426 DTM.FAIL.DCCH.DOWNLINK.DTM.ASS A9810:Number of Failed Downlink DTM Assignments Received on DCCH

Description

This counter provides the number of downlink DTM assignment failures after a BSC sends a downlink DTM assignment command on the DCCH. The causes are as follows:

The MS sends a DTM ASS FAIL message to the BSS.

The INTRAHO CMP timer fails to start after the BSS sends a downlink DTM ASS CMD message to the MS.

The INTRAHO CMP timer expires after the BSS sends a downlink DTM ASS CMD message to the MS.

Measurement point

If the MS does not respond with the DTM assignment message after the BSS sends a downlink DTM ASS CMD message due to various causes, the counter is incremented by one respectively according to the specific cause.

After the BSC receives the DTM ASS FAIL message from the MS, the counter is incremented by one.

If the start of the INTRAHO CMP timer fails after the BSC sends the DTM ASS CMD message to the MS, the counter is incremented by one.

If the INTRAHO CMP timer starts and then expires after the BSC sends the DTM ASS CMD message to the MS, the counter is incremented by one.
