Number of Downlink EGPRS TBF Abnormal Releases due to N3105 Overflow

Measurement Counters

ID Counter Description
1279176422 DOWN.EGPRS.TBF.ABNORM.REL.N3105.OVERFLOW A9306:Number of Downlink EGPRS TBF Abnormal Releases due to N3105 Overflow

Description

This measurement provides the number of downlink EGPRS TBF abnormal releases due to N3105 overflow for services including the DTM service during a measurement period. A high measurement value is yielded because the radio quality of the cell is bad.

Measurement point

During downlink EGPRS TBF transmission, the BSC periodically sends the block with the RRBP flag to reserve the blocks for the MS to respond with the PACKET DOWNLINK ACK/NACK message. The purpose is to check the downlink receiving conditions.

If the BSC receives the PACKET DOWNLINK ACK/NACK message from the MS in the reserved blocks during a period defined by the BSC timer, then the N3105 is reset; otherwise, the N3105 is incremented by one, and the BSC sends the block with RRBP flag again. If the N3105 overflows, the BSC initiates the downlink EGPRS TBF release procedure.

The measurement is triggered at point A in figure 1 (the process of abnormal release of downlink EGPRS TBFs due to N3105 overflow). This counter is incremented by one each time N3105 overflows.
