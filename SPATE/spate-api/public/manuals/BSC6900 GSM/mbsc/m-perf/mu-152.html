Uplink GPRS TBF Establish and Release Capability Measurement(PSCALL.UpGTbfEstbRel.Cell)

Overview

The measurement unit Uplink GPRS TBF Establish and Release Capability Measurement provides the measurement counters related to the uplink GPRS TBF establishment and release. This measurement unit indicates the PS service load of a cell and is used as a reference for PS service model establishment.
