Number of MRs Based on TA per TRX Location Group(MR.TaDistribOrig.TRX_LOCGRP)

Overview

This measurement unit provides the number of MRs from the BTS with a certain TA range. This provides the distribution of traffic volume based on the distance between the MS and the BTS and facilitates network maintenance and optimization.
