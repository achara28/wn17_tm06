Description

These counters provide the numbers of valid MRs or preprocessing MRs with the uplink receive quality rank ranging from 0 to 7 on the TCHHs in one location group.

Measurement point

These counters are measured when the BSC receives valid MRs or preprocessing MRs from the BTS based on the uplink receive level on the TCHHs in one location group.
