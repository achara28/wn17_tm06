Measurement Related to BTS IPPM(BTSM.BTSIPPM)

Overview

The BTS IP Performance Monitor (PM) uses Forward Monitoring (FM) and Backward Reporting (BR) to check for packet loss over channels. For measurement on the BTS side, the BTS periodically sends FM messages to indicate the number of outgoing packets, the number of bytes, and sending time. The BSC responds with BR messages after receiving the FM message to report the number of received packets, the number of received bytes, the receiving time of PM message, the sending time of BR response. The BTS calculates the packet loss rate, transmission delay and jitter according to the BR response returned from the BSC.
