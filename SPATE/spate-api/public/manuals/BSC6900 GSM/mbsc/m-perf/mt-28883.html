Successful Immediate Assignments

Measurement Counters

ID Counter Description
1278074430 TRX.IMM.ASS.SUCC CR440B:Successful Immediate Assignments

Description

In the immediate assignment procedure, the MS successfully seizes the channel assigned by the BSC. The MS reports an SABM frame on the assigned channel to the BTS to notify the BTS that the radio link layer has entered the status of multiframe link establishment. The BTS returns a UA frame to the MS and at the same time sends an EST IND message to the BSC to notify the BSC that a link is normally established on the air interface. Upon receipt of the EST IND message, the BSC considers that the immediate assignment is successful and the counter CR440B:TRX.IMM.ASS.SUCC is incremented by one. The assigned channel can be an SDCCH or TCH. The counter covers the number of EST IND messages received on any channel of the measured TRX.

Measurement point

As shown in Figure 1, each time the BSC receives an EST IND message from the BTS (see measurement point C), the counter is incremented by one.

Formula

Successful Immediate Assignments = [Successful Immediate Assignments (SDCCH)] + [Successful Immediate Assignments (TCH)]
