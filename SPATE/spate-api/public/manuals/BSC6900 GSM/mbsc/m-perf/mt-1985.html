Description

This measurement provides the number of successful VGCS TCH handovers,according to the different handover reasons. The reasons include frequency shift,quick PBGT,uplink quality,downlink quality,uplink strength,downlink strength,timing advance,better cell,traffic load,rapid Rxlevel drop,response to MSC intervention,OM intervention and other causes.

Measurement point

(1) In the case of the intra-BSC intra-cell handover, this counter is incremented by one when the originating BSC receives a HANDOVER SUCCEEDED message from the MSC. The measurement point is B, as shown in Figure 1.

(2) In the case of the intra-BSC inter-cell handover, this counter is incremented by one when the target cell receives a HANDOVER COMPLETE message from the BTS. The measurement point is B, as shown in Figure 2.

(3) In the case of the inter-BSC handover, this counter is incremented by one when the originating BSC receives a CLEAR COMMAND message from the MSC. The measurement point is B, as shown in Figure 3.
