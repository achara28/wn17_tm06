Channel Assignment Request Measurement per Cell(CHAN.AllocReq.CELL)

Overview

The BSS channel assignment is performed in the following procedures: immediate assignment procedure, assignment procedure, internal intra-cell handover procedure, incoming internal inter-cell handover procedure, and incoming external inter-cell handover procedure.

The measurement unit Channel Assignment Request Measurement per Cell is used to measure the assignment requests for the SDCCH and TCH channels in the preceding procedures.
