Number of Attempts to Return to Normal Call from BTS local switching (Service)

Measurement Counters

ID Counter Description
1278089504 CELL.LOCALSWITCH.RESTORE.TRY.BTS.LOOP.SERVICE R3843:Number of Attempts to Return to Normal Call from BTS local switching (Service)

Description

During the process of starting up the BTS, the BTS local switching recovery procedure is started because the BSC detects a supplementary service or another service that needs to be carried and processed on the core network.

Measurement point

1. After the BSS receives a service that needs to be carried and processed on the core network in the stable BTS local switching state, the BTS local switching recovery procedure is started. This counter is measured on both the calling MS and the called MS.

2. After the BSS receives a service that needs to be carried and processed on the core network in the BTS local switching handover process, the local switching is not restarted and the BTS local switching recovery procedure is started if the handover fails. This counter is measured on both the calling MS and the called MS.
