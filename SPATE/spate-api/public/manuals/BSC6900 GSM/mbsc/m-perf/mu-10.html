Ater Interface Circuit Measurement per TC Subrack(ATER_RES)

Overview

Ater Interface Circuit Measurement per BM Subrack is used to measure the faulty, idle, busy, block and maintain circuits on the Ater interface. This measurement is only used to measure user plane time-slot, but the signaling plane time-slot nor the semipermanent time-slot. If the Ater Interface Circuits are in various states, the respective number of Ater Interface Circuits in each state are measured and the measurement results are provided in the sequence of busy, maintenance(faulty and blocked), faulty, blocked and idle. A single Ater Interface Circuit is in only one state and measured only once.
