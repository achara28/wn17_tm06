Downlink EGPRS RLC Data Transfer Capability Measurement(PSCALL.DownERlcDataTran.Cell)

Overview

The Downlink EGPRS RLC Data Transfer Capability measurement unit provides the measurement counters related to the downlink EGPRS RLC transmission performance in a cell in a granularity period. This measurement unit indicates the PS service load of a cell and is used as a reference for PS service model establishment.
