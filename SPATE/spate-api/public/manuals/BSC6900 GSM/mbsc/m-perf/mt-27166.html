Number of Failed Handovers from Underlaid Subcell to Overlaid Subcell

Measurement Counters

ID Counter Description
1279172422 ULCELL.INTO.OLCELL.FAIL.NUM CH9006:Number of Failed Handovers from Underlaid Subcell to Overlaid Subcell

Description

This counter provides the number of failed handovers of packet service from the underlaid subcell to the overlaid subcell in a measurement period.

Measurement point

Number of Failed Handovers from Underlaid Subcell to Overlaid Subcell is equal to Number of Underlaid Subcell to Overlaid Subcell Handover Requests minus Number of Successful Handovers from Underlaid Subcell to Overlaid Subcell. If the load in the underlaid subcell is lower than the load in the overlaid subcell or no PDCH is available in the overlaid subcell, a handover from the underlaid subcell to the overlaid subcell fails. As a result, the MS remains in the underlaid subcell, and its PS service is not affected.

Formula

Number of Failed Handovers from Underlaid Subcell to Overlaid Subcell = [Number of Underlaid Subcell to Overlaid Subcell Handover Requests] - [Number of Successful Handovers from Underlaid Subcell to Overlaid Subcell]
