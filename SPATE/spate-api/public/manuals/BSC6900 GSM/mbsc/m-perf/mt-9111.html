1282411919 TRX.LOCGRP.IMM.ASS.SUCC.SD TR4400B:Successful Immediate Assignments (SDCCH) in One Location Group

Description

In the immediate assignment (SDCCH) procedure, the MS successfully seizes the SDCCH assigned by the BSC. The MS reports an SABM frame on the SDCCH to the BTS to notify the BTS that the radio link layer has entered the status of multiframe link establishment. The BTS returns a UA frame to the MS and at the same time sends an EST IND message to the BSC to notify the BSC that a link is normally established on the Um interface. Upon receipt of the EST IND message, the BSC considers that the immediate assignment of the SDCCH is successful. This counter provides the number of successful immediate assignments (SDCCH) in the TRX location group where the SDCCH is carried.

Measurement point

Each time the BSC receives an EST IND message from the BTS, the related counter is incremented by one based on the specific type of the channel successfully assigned.
