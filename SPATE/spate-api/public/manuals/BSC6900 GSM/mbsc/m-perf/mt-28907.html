Measurement Counters

ID Counter Description
1278278440 TRX.HR.DROP.DL.QLTY.AVR AS4338D:Mean Downlink Quality during Radio Link Failure (TCHH)

Description

After receiving a valid MR or pre-processed MR, the BSC takes statistics of some parameter values based on the requirements. The statistical results can help users better understand the network performance. The measurement provides the mean downlink quality during radio link failure on half-rate TCH. The counter value is calculated based on the counters S4338D: TRX.HR.DROP.DL.QLT and S4358D: TRX.HR.DROP.TOTAL.NUM.

Measurement point

See measurement points of the counters in the formula.

Formula

Mean Downlink Quality during Radio Link Failure (TCHH) = [Downlink Quality during Radio Link Failure (TCHH)]/[Radio Link Failures (TCHH)]
