Outgoing Inter-Cell Handover Commands in One Cell Location Group

Measurement Counters

ID Counter Description
1282411998 CELL.LOCGRP.OUT.GROUP.HO.CMD TO313:Outgoing Inter-Cell Handover Commands in One Cell Location Group

Description

This counter provides the number of outgoing inter-cell handover commands from the cell configured with one or more location groups. Along with Failed Outgoing Inter-cell Handovers in One Loaction Group, this counter indicates the performance of outgoing inter-cell handovers.

Measurement point

If the originating cell and the target cell are GSM cells, this counter is measured in the following situations (note that the information on the cell location group must be identified during the measurement):
  • After receiving an MR, the BSC detects that the MS needs to be handed over to another cell in the BSC. Then, the BSC sends a Channel Activation message to BTS2 in the target cell to activate the channel. If the channel type is correct after receiving the Channel Activation message, BTS2 switches on the power amplifier for the specified channel to receive the uplink message, and sends a Channel Activation Acknowledge message to the BSC. After receiving the Channel Activation Acknowledge message from BTS2, the BSC sends a Handover CMD message to BTS1. In this case, the related counters are measured based on the handover causes, such as Uplink Signal Quality, Downlink Signal Quality, Uplink Signal Strength, Downlink Signal Strength, Timing Advance, Better Cell, Load, Rapid Level Drop, MSC Intervention, OM Intervention, EDGE Coverage.
  • After receiving a Handover REQ ACK message, the MSC sends a Handover CMD message to BSC1. Then, BSC1 sends the MS the Handover CMD message, requesting the MS to access the new channel. In this case, the related counters are measured based on the handover cause, such as Uplink Signal Quality, Downlink Signal Quality, Uplink Signal Strength, Downlink Signal Strength, Timing Advance, Better Cell, Load, Rapid Level Drop, MSC Intervention, OM Intervention, EDGE Coverage.
