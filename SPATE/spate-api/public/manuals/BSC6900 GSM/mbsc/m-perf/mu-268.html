BTS FlexAbis Measurement per BTS(BTSM.FlexAbis.Site)

Overview

If the Abis resources of the BTS are dynamically assigned, the call connection may fail due to the assignment of the Abis resources. Thus, the counters related to the assignment of the Abis resources should be measured to indicate the Abis resources.

These counters are measured only when the Abis resource assignment mode of the BTS is set to FlexAbis.
