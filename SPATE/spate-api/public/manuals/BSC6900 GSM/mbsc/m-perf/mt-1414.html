Number of Failed Intra-BSC Handovers with Codec Change (Rejected by the MSC) (Other Causes)

Measurement Counters

ID Counter Description
1282411937 CELL.INTRABSC.CODECHO.HO.FAIL.CN.REJ.OTHER H4100J:Number of Failed Intra-BSC Handovers with Codec Change (Rejected by the MSC) (Other Causes)

Description

This counter provides the number of failed intra-BSC handovers with codec change in A over IP in a measurement period (rejected by the MSC) (other causes).

Measurement point

This counter is incremented by one when an intra-BSC handover with codec change fails in A over IP in a measurement period because the MSC responds with an INTERNAL HANDOVER REQUIRED REJECT message with the cause value that is not "requested A-Interface Type not supported", "requested Codec Type or Codec Configuration not supported", "requested A-Interface Type temporarily unavailable", "requested Codec Type or Codec Configuration temporarily unavailable."
