Description

This measurement provides the number of MRs with the RQI in the preceding range in the granularity period.

Measurement point

This counter is incremented by one when the BSC receives an MR or pre-processed MR in which N <= RQI < M from the BTS in the serving cell. N ranges from 0 to 45, M ranges from 1 to 55.
