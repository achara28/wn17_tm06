Description

The BSC decides whether to initiate a handover based on the MR and the handover algorithm. If the target cell of the handover is a TD-SCDMA cell, an outgoing inter-RAT inter-cell handover (TDD) is initiated. The specific counter provides the number of outgoing inter-RAT inter-cell handover requests (TDD) in the originating cell initiated based on the handover algorithm. Together with the Outgoing Inter-RAT Inter-Cell Handover Commands (TDD) and the Failed Outgoing Inter-RAT Inter-Cell Handovers (TDD), this counter determines the performance of the outgoing inter-RAT inter-cell handover (TDD).

The following counter provides the number of outgoing inter-RAT inter-cell handover requests (TDD) (excluding directed retry) if the MS is seizing an SDCCH before the handover:
  • H35001:CELL.INTERRAN.TDD.OUTCELL.HO.REQ.NOT.INCLUDE.DR.SD
The following counter provides the number of outgoing inter-RAT inter-cell handover requests (TDD) (including directed retry) if the MS is seizing an SDCCH before the handover:
  • H35001A:CELL.INTERRAN.TDD.OUTCELL.HO.REQ.DR.SD
The following two counters provide respectively the number of outgoing inter-RAT inter-cell handover requests (TDD) when the MS seizes a TCHF or a TCHH before the handover. In this measurement, both the handovers initiated by directed retry and those initiated according to the handover algorithms based on the measurement reports are included.
  • H35071:CELL.INTERRAN.TDD.OUTCELL.HO.REQ.TCHF
  • H35081:CELL.INTERRAN.TDD.OUTCELL.HO.REQ.TCHH
