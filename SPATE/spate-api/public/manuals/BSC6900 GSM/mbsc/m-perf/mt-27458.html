Forward Packet Loss Rate of IPPM

Measurement Counters

ID Counter Description
67204127 VS.IPPM.Forword.DropMeans T7676:Average Forward Packet Loss Rate of IPPM
67204128 VS.IPPM.Forword.Peak.DropRates T7677:Peak Forward Packet Loss Rate of IPPM
73441211 VS.IPPM.Forward.Precise.Peak.DropRates T7681:High-Precision Peak Forward Packet Loss Rate of IPPM

Description

This counter provides the Forward Packet Loss Rate of the IPPATH in the specified measurement period.The packet loss rate can be obtained by dividing the difference between the number of service packets sent by the local IP address of the IPPM and the number of service packets received by the peer IP address of the IPPM by the number of service packets sent by the local IP address of the IPPM.

Measurement point

Samples are taken every five seconds in the specified measurement period. The system obtains the number of discarded packtes once every five seconds. With this, the system obtains the total number of discarded packtes in one measurement period. The loss rate is calculated by dividing the discarded packtes with total packtes.

Formula

None

Unit

%
