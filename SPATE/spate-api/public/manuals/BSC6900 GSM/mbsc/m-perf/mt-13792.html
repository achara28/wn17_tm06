Throughput of Valid Downlink EGPRS RLC Data Blocks (Overlaid Subcell)

Measurement Counters

ID Counter Description
1282432240 DOWN.RLC.EGPRS.BLK.VALID.THROUGHPUT.OVERLAY L9360:Throughput of Valid Downlink EGPRS RLC Data Blocks (Overlaid Subcell)

Description

This counter provides the throughput of valid downlink EGPRS RLC data blocks in the overlaid subcell of a concentric cell in a measurement period. By comparing the throughput at different periods, you can identify the off-peak and peak hours for data services.

Measurement point

This counter is incremented by 22 when the following conditions are met:

1. The BSC receives the EGPRS PACKET DOWNLINK ACK/NACK message from the MS in the overlaid subcell of a concentric cell.

2. The BSC determines that the MS receives the corresponding downlink EGPRS RLC data block based on the bitmap in the message for the first time.

3. The downlink EGPRS RLC data block uses an MCS1 coding scheme.

If the RLC data blocks use MCS2, this counter is incremented by 28.

If the RLC data blocks use MCS3, this counter is incremented by 37.

If the RLC data blocks use MCS4, this counter is incremented by 44.

If the RLC data blocks use MCS5, this counter is incremented by 56.

If the RLC data blocks use MCS6, this counter is incremented by 74.

If the RLC data blocks use MCS7, this counter is incremented by 56.

If the RLC data blocks use MCS8, this counter is incremented by 68.

If the RLC data blocks use MCS9, this counter is incremented by 74.
