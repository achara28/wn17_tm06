Mean RTT for RTP Packets Received Between an IP Address Pair

Measurement Counters

ID Counter Description
73441284 VS.IPPOOL.RTP.AverageRtt P141012:Mean RTT for RTP Packets Received Between an IP Address Pair

Description

This counter provides the mean Round Trip Time (RTT) for packets received over the Iu/A interface between a certain pair of IP addresses in a measurement period.

Measurement point

This counter is calculated by using the following formula in scenarios where Iu/A over IP is applied and RTCP is supported: Mean RTT = Sum of RTT/Number of RTP traffic statistic reporting times

Formula

None

Unit

ms
