Measurement of Failed Outgoing Internal Inter-Cell Handovers (HSCSD Services-Based)

Measurement Counters

ID Counter Description
1282432005 CELL.INTRABSC.OUTCELL.HO.FAIL.HSCSD A3139E:Number of Failed Outgoing Internal Inter-Cell Handovers (HSCSD Service)

Description

This counter provides the number of failed intra-BSC handovers for HSCSD calls. The measurement is performed on the basis of the source cell of the handover.

Measurement point

The BSC measures this counter in the source cell when the internal inter-cell handover for an HSCSD call initiated by the BSC fails because of various reasons.
