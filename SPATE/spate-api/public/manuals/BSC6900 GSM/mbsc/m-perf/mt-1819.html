Counters Related to Rejected Location Update Requests

Measurement Counters

ID Counter Description
1282412150 CELL.LUA.REJ.TIMES.NORMAL.FAIL LA305:Number of Rejected Location Update Requests (Normal Failure)
1282412151 CELL.LUA.REJ.TIMES.PLMN.NOT.ALLOWED LA306:Number of Rejected Location Update Requests (Not Allowed by the PLMN)
1282412152 CELL.LUA.REJ.TIMES.INTFACE.UNAVIL LA307:Number of Rejected Location Update Requests (Interface Unavailable)
1282412153 CELL.LUA.REJ.TIMES.CONGESTION LA308:Number of Rejected Location Update Requests (Congestion)

Description

These counters provide the number of rejected location update requests based on the specific cause.The counter LA305 is increased by 1 if the MSC sends a location update rejection message with one of the following causes:

  • IMSI unknown in HLR
  • Illegal MS
  • Illegal ME
  • Location Area not allowed
  • Roaming not allowed in this location area
  • No Suitable Cells In Location Area
  • PLMN not allowed
The counter LA306 is increased by 1 if the MSC sends a location update rejection message with the cause "PLMN not allowed."

The counter LA307 is increased by 1 if the MSC sends a location update rejection message with the cause "Network failure."

The counter LA308 is increased by 1 if the MSC sends a location update rejection message with the cause "Congestion."

Measurement point

A specific counter is incremented by one based on the cause when the BSC receives a Location Update Reject message.
