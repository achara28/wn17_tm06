Description

When the transmit power of all the calls on the timeslots with the same number in a TRX group (the TRXs in the group share the power amplifiers in the same resource pool) is greater than the maximum transmit power of the BTS, the BTS decreases the power of all the calls to ensure its normal operation. This counter measures the number of timeslots with the power decreased by the BTS.

Measurement point

The BTS periodically sends the number of timeslots with certain power decreased to the BSC for measurement. "Number of Timeslots with Power Decreased by 0.5 dB" provides the number of timeslots with the power decreased by 0.2 to 0.6 dB. "Number of Timeslots with Power Decreased by 1.0 dB" provides the number of timeslots with the power decreased by 0.8 to 1.2 dB. "Number of Timeslots with Power Decreased by 1.5 dB" provides the number of timeslots with the power decreased by 1.4 to 1.8 dB. "Number of Timeslots with Power Decreased by 2.0 dB" provides the number of timeslots with the power decreased by 2.0 to 2.4 dB. "Number of Timeslots with Power Decreased by 2.5 dB" provides the number of timeslots with the power decreased by 2.6 to 2.8 dB. "Number of Timeslots with Power Decreased by 3.0 dB or Higher" provides the number of timeslots with the power decreased by 3.0 dB or higher.
