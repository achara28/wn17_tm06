Description

This measurement provides the throughput of the downlink EGPRS interactive service THPx LLC PDU in a granularity period.

Measurement point

Each time the transmission of interactive THPx service is complete on the downlink EGPRS TBF, the transmission rate of the LLC PDU is calculated. The transmission rate is equal to the number of transmitted bytes divided by the transmission duration. This transmission rate multiplied by the number of transmitted bytes yields the throughput of transmitted LLC PDUs. Then, the calculated throughput is added to the counter.

Note: THPx indicates THP1-THP3.

Formula

None

Unit
