Description

This measurement provides the retransmission rates of downlink RLC data blocks in CS1-CS4 coding schemes in a granularity period. If the retransmission rate is high, you can infer that the signal quality is poor, and thus the network needs to be optimized.

Measurement point

See measurement points of the counters in the formula.

Formula

Retransmission Rate of RLC Downlink Data Block Using CS1 = ( ([Total Number of Downlink RLC Data Blocks Using CS1] - [Total Number of Valid Downlink RLC Data Blocks Using CS1])/[Total Number of Downlink RLC Data Blocks Using CS1]) x 100

The formula of "Retransmission Rate of RLC Downlink Data Block Using CSx" is similar as "[Retransmission Rate of RLC Downlink Data Block Using CS1]".

Note: CSx indicates CS1-CS4.

%
