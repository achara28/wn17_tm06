Call Drops on TCH(Uplink FER)

Measurement Counters

ID Counter Description
1282411664 CELL.TCH.CALL.DROPS.UL.FER M3030E:Call Drops on TCH(Uplink FER)

Description

This counter provides the number of call drops on the TCH of a cell. The causes for call drops are air interface fault, no measurement report from the MS for a long time, Abis interface fault, equipment fault, and forced handover.

When a call drop occurs on the TCH, the counter, Call Drops on TCH(Uplink FER), is incremented by one if the uplink FER is greater than "UL FER Threshold for TCH Call Drop", the downlink FER is smaller than or equal to "DL FER Threshold for TCH Call Drop", and the conditions for measuring Call Drops on TCH(Uplink Received Level), Call Drops on TCH(Downlink Received Level), or Call Drops on TCH(Uplink and Downlink Received Level) are not met. This counter is not measured if the EMR is not used.

Measurement point

As shown at point A in figure 1, when a call fails and the BSC sends a Clear Request message to the MSC, this counter is measured.

As shown at point A in figure 2, if the cause in the Clear Command message that the BSC receives from the MSC is not Call Control or Handover Successful and the BSC does not send a Clear Request message to the MSC before receives the Clear Command message, this counter is measured.
