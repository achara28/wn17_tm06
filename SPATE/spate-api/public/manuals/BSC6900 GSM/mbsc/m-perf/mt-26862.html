Outgoing External Inter-Cell Handover Requests (900/850/810-1800/1900)

Measurement Counters

ID Counter Description
1278078502 CELL.INTERBSC.OUTCELL.HO.REQ.900.1800 RH3035G:Outgoing External Inter-Cell Handover Requests (900/850/810-1800/1900)

Description

Based on the MR sent by the MS, the BSC uses the handover algorithm to determine whether a handover is required. If a handover is required, and if the originating cell (current cell) and the target cell are managed by different BSCs, the BSC initiates an external inter-cell handover. This counter provides the total number of outgoing external inter-cell handover requests. The measurement is performed in the originating cell and covers the requests on the SDCCH, TCHF and TCHH. In addition, the frequency band of the originating cell can be GSM900, GSM900&DCS1800, GSM850, GSM850&DCS1800, GSM850&PCS1900 or TGSM810, and the frequency band of the target cell can be DCS1800 or PCS1900.

Measurement point

See measurement points of the counters in the formula.

Formula

Outgoing External Inter-Cell Handover Requests (900/850/810-1800/1900) = [Outgoing External Inter-Cell Handover Requests (SDCCH) (Excluding Directed Retry) (900/850/810-1800/1900)] + [Outgoing External Inter-Cell Handover Requests (TCHF) (Excluding Directed Retry) (900/850/810-1800/1900)] + [Outgoing External Inter-Cell Handover Requests (TCHH) (Excluding Directed Retry) (900/850/810-1800/1900)] + [Outgoing External Inter-Cell Handover Requests (Directed Retry) (900/850/810-1800/1900)]
