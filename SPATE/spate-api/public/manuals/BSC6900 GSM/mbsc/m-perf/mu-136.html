Intra-CELL Handover Measurement(PSCALL.CellHo.Cell)

Overview

The Intra-CELL Handover Measurement provides the measurement counters related to intra-cell handover in a granularity period. This measurement unit indicates the PS service load of a cell and is used as a reference for PS service model establishment.
