Mean Duration of Busy State for Channels (SDCCH/TCHF/TCHH)

Measurement Counters

ID Counter Description
1278471423 CELL.CH.BUSY.TIME.SD.AVR AR3530:Mean Duration of Busy State for Channels (SDCCH)
1278471424 CELL.CH.BUSY.TIME.FR.AVR AR3531:Mean Duration of Busy State for Channels (TCHF)
1278471425 CELL.CH.BUSY.TIME.HR.AVR AR3532:Mean Duration of Busy State for Channels (TCHH)

Description

These counters provides the mean duration of the state for seized SDCCHs, TCHFs, or TCHHs configured in a cell in a granularity period, that is, the mean duration of the busy state of each SDCCH, TCHF, or TCHH channel.

Measurement point

Mean Duration of Busy State for Channels (SDCCH) is the result of "Duration of Busy State for Channels (SDCCH)/((Number of Dynamically Configured Channels (SDCCH) (900/850/810 Band)/Number of Sampling Times)+(Number of Dynamically Configured Channels (SDCCH) (1800/1900 Band)/Number of Sampling Times))".

Duration of Busy State for Channels (SDCCH):sum of the time when SDCCH is in being used.

Number of Dynamically Configured Channels (SDCCH):the BSC periodically get the total number of configured SDCCH.

Number of Sampling Times:every time the BSC calculates the Number of Dynamically Configured Channels,this value added by one.

The measure of Mean Duration of Busy State for Channels (TCHF/TCHH) is similar to above.
