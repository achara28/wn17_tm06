Intra-BSC Handovers with Codec Change Measurement per Cell(CALL.IntraBscCodecHo.CELL)

Overview

This measurement unit provides measurements about handover performance of the source cell in A over IP in a measurement period, if the codec of the target channel and that of the source channel are different or incompatible when an intra-BSC handover (intra- or inter-cell handover) occurs.
