Description

The MS sends the target BTS an HO ACC message to attempt to access a new channel of the serving cell. The target BTS forwards this message through an HO DETECT message to the BSC and sends a PHY INFO message to the MS. The MS sends a FIRST SABM frame to the target BTS (the serving cell) and the target BTS sends an EST IND message to the BSC, indicating that the MS has successfully accessed the target cell(the serving cell). The MS also sends an HO CMP message to the BSC. The BSC sends an HO PERFORMED message to the MSC.

The counters provide the numbers of HO DETECT messages the BSC receives from the BTS based on the type of the channel that the MS accesses and the type of the call. They can be combined with the Internal Intra-Cell Handover Commands and the Failed Internal Intra-Cell Handovers to reflect the performance of internal intra-cell handovers.

The counter H3040 indicates the number of HO DETECT messages received by the BSC from the BTS for an internal intra-cell handover when the MS accesses an SDCCH. The counter H3047 indicates the number of HO DETECT messages received by the BSC from the BTS for an internal intra-cell handover when the MS accesses a TCHF. The counter H3048 indicates the number of HO DETECT messages received by the BSC from the BTS for an internal intra-cell handover when the MS accesses a TCHH. Each time the BSC receives an HO DETECT message from the BTS, the counter H3040, H3047, or H3048 is incremented by one based on the type of the channel the MS accesses.
  • H3040:CELL.INTRACELL.HO.DETECT.SD
  • H3047:CELL.INTRACELL.HO.DETECT.TCHF
  • H3048:CELL.INTRACELL.HO.DETECT.TCHH
The counter H304a indicates the number of HO DETECT messages received by the BSC from the BTS during an internal intra-cell handover for a call that uses the local switching function of the BSC. The counter H304b indicates the number of HO DETECT messages received by the BSC from the BTS during an internal intra-cell handover for a call that uses the local switching function of the BTS. Each time the BSC receives an HO DETECT message from the BTS during an internal intra-cell handover for a call that uses the local switching function of the BSC or the BTS, the counter H304a or H304b is incremented by one.
  • H304a: CELL.INTRACELL.HO.DETECT.BSC.LOOP
  • H304b: CELL.INTRACELL.HO.DETECT.BTS.LOOP

Measurement point

As shown in Figure 1, each time the BSC in the target cell (current cell) receives an HO DETECT message from the BTS (see measurement point C), one of the above counters is incremented by one based on the channel type and the call type.
