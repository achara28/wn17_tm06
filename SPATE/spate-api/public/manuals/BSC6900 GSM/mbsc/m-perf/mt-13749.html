Number of Downlink GPRS TBF Abnormal Releases due to N3105 Overflow (Overlaid Subcell)

Measurement Counters

ID Counter Description
1282432197 DOWN.GPRS.TBF.ABNORM.REL.N3105.OVERFLOW.OVERLAY A9141:Number of Downlink GPRS TBF Abnormal Releases due to N3105 Overflow (Overlaid Subcell)

Description

This counter provides the number of downlink GPRS TBF abnormal releases due to N3105 overflow in the overlaid subcell of a concentric cell in a measurement period. If the value of this counter is high, the radio network quality of the overlaid subcell is bad.

Measurement point

During downlink GPRS TBF transmission, the BSC periodically sends blocks with the RRBP flag to reserve the blocks for the MS to respond with a PACKET DOWNLINK ACK/NACK message. The purpose is to check the downlink receiving conditions.

If the BSC receives the PACKET DOWNLINK ACK/NACK message from the MS in the reserved blocks within the period specified by the BSC timer, the counter N3105 is reset. Otherwise, N3105 is incremented by one and the BSC sends the block with RRBP flag again. If N3105 overflows, the BSC initiates the downlink GPRS TBF release procedure.
