Number of Circuit Paging Requests Received from the Gb Interface

Measurement Counters

ID Counter Description
1279171418 RX.GB.CS.PAGING.REQ A9602:Number of Circuit Paging Requests Received from the Gb Interface

Description

If the Gs interface is configured between the MSC and the SGSN, the MSC/VLR can instruct the SGSN to initiate circuit paging for the circuit switching establishment. The SGSN then sends one or more CS PAGING PDUs to the BSS. This measurement provides the number of circuit paging requests received from the Gb interface in a granularity period.

Measurement point

Figure 1 shows the procedure of the CS paging request initiated by the SGSN. Each time the BSC receives the CS PAGING REQUEST message from the Gb interface (see measurement point A), the counter is incremented by one. In addition, the SGSN can determine which cell to initiate the paging according to the NSEI or BVCI information element carried in the CS PAGING REQUEST message.
