Number of local switching Start Attempts

Measurement Counters

ID Counter Description
1278089418 CELL.LOCALSWITCH.LOOP.START.TRY R381:Number of local switching Start Attempts

Description

After the incoming BSC handover or the intra-BSC handover for the speech service is complete, or the call is put through, the end initiating local switching start sends a local switching request to the peer end if the BSC/BTS local switch start conditions are met and if it is detected that the calling and called MSs are served by the same BSC. This counter provides the number of local switching start attempts.

Measurement point

After the BSC receives the CONNECT ACK message from the MSC in the MS terminated call procedure, as shown in Figure 1 (A1), or after the BSC receives the HANDOVER CMP message from the MS in the incoming BSC handover procedure for the speech service, as shown in Figure 2 (B1), or after the BSC receives the HANDOVER CMP message from the MS in the intra-BSC handover procedure for the speech service, as shown in Figure 3 (C1), the BSC selects the BSS local switch policy, determines whether the conditions for initiating BSC/BTS local switch are met, and performs local switch call detection. If the calling and called MSs are served by the same BSC, the call processing module initiating local switching start in the BSC sends a local switching request message to the call processing module at the peer end in the BSC. In this case, this counter is measured when the calling MS sends a local switching request message or when the called MS receives a local switching request message.
