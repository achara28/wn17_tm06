Clear Requests Sent on the A interface (TCH) (Signaling Channel)

Measurement Counters

ID Counter Description
1278072525 CELL.A.CLR.REQ.TCH.SIG CM369:Clear Requests Sent on the A interface (TCH) (Signaling Channel)

Description

After the MS seizes a TCH to carry signaling, the BSC sends a CLEAR REQ message to the MSC if the BSC releases a call due to various causes. This measurement provides number of clear requests sent on the A interface when the channel type is TCHF or TCHH and the service type is signaling channel.

Measurement point

When a TCH is seized as a signaling channel during a call, this counter is measured when the BSC sends a CLEAR REQ message to the MSC.

This counter is calculated at A5, as shown in Figure 1.
