Call Drops on TCHF Due to Failures to Return to Normal Call from BSC local switching (Terrestrial Resource Request Failure)

Measurement Counters

ID Counter Description
1278089494 CELL.CALL.DROPS.LOOP.RESTORE.APPLY.TRSL.FAIL.TIMES.BSC.LOOP.TCHF M3151:Call Drops on TCHF Due to Failures to Return to Normal Call from BSC local switching (Terrestrial Resource Request Failure)

Description

During the process where the BSC local switching is returned to a normal call, the TC resources should be requested and the transmission channel between the Ater interface and the Abis interface should be reconnected at the same time. If the reconnection or request fails, the cause value in the local switching acknowledgement message is Terrestrial Resource Request Failure. Then, the BSC regards it as a call drop and then releases the call. This counter provides the number of call drops on TCHF due to failures to return to normal call from BSC local switching (terrestrial resource request failure).

Measurement point

During the process where the BSC local switching is returned to a normal call, the TC resources should be requested and the transmission channel between the Ater interface and the Abis interfaces should be reconnected at the same time. If the reconnection or request fails, the cause value in the local switching acknowledgement message is Terrestrial Resource Request Failure. Then, the BSC regards it as a call drop and then releases the call. If the channel that carries the call during the call drop is a TCHF, this counter is measured.
