Description

This measurement provides the total number of downlink EGPRS RLC data blocks using MCS-7, MCS-8, and MCS-9 coding schemes in a cell in a granularity period.

The following table describes the meaning of each counter.

Counter Description
DOWN.EGPRS.MCS7.RLC.DATA.BLK Total Number of Downlink EGPRS MCS7 RLC Data Blocks
DOWN.EGPRS.MCS8.RLC.DATA.BLK Total Number of Downlink EGPRS MCS8 RLC Data Blocks
DOWN.EGPRS.MCS9.RLC.DATA.BLK Total Number of Downlink EGPRS MCS9 RLC Data Blocks

Measurement point

Figure 1 shows the procedure in which the BSC sends a downlink EGPRS RLC data block to the MS. Every time the BSC sends a DOWNLINK EGPRS RLC data block using MCS-7 coding scheme (see measurement point A), the counter is incremented by one.

Figure 1 Transmission of downlink EGPRS RLC data blocks using MCS-7 coding scheme

The measurement point of the number of downlink EGPRS RLC data blocks using MCS-8 or MCS-9 coding scheme is the same as that of the number of downlink EGPRS RLC data blocks using MCS-7 coding scheme. The measurement is triggered after the data block using the corresponding coding scheme is sent.
