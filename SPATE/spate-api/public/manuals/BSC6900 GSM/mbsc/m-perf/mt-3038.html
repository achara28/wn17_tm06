Description

This measurement provides the number of times that UPLINK_GMSK_MEAN_BEP = N carried in the data blocks that the BTS sends to the BSC. This value indicates the Um interface quality on the downlink in GMSK modulation mode. The greater the value of N, the better the Um interface quality.

Measurement point

Figure 1 shows the process of sending a data block from the BTS to the BSC. Each time the BSC receives the data block in GMSK modulation mode contains the UPLINK_GMSK_MEAN_BEP field of value N from the BTS (see measurement point A), the counter is incremented by one.The value range for N is from 0 to 31.
