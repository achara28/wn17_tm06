Outgoing Inter-RAT Inter-Cell Handover Requests (TDD)

Measurement Counters

ID Counter Description
1278092468 CELL.INTERRAN.TDD.OUTCELL.HO.REQ CH3501:Outgoing Inter-RAT Inter-Cell Handover Requests (TDD)

Description

The BSC uses the handover algorithm to decide whether a handover is required based on the measurement reports. If a handover is required, and if the target cell is in a TD-SCDMA system, an outgoing inter-RAT inter-cell handover (TDD) is initiated.

This measurement provides the outgoing inter-RAT inter-cell handover requests (TDD) initiated by the BSC. The measurement is performed based on the originating cell and involves handovers on the SDCCH, TCHF, and TCHH.

Measurement point

As shown in Figure 1 and Figure 2, the counter is measured at A when the BSC decides that an outgoing inter-RAT inter-cell handover (TDD) is required based on the measurement report.

Formula

Outgoing Inter-RAT Inter-Cell Handover Requests (TDD) = [Outgoing Inter-RAT Inter-Cell Handover Requests (SDCCH) (Excluding Directed Retry) (TDD)] + [Outgoing Inter-RAT Inter-Cell Handover Requests (TCHF) (TDD)] + [Outgoing Inter-RAT Inter-Cell Handover Requests (TCHH) (TDD)]
