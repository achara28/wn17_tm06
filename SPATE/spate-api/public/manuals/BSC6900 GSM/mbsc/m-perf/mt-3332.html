Outgoing Inter-Cell Handover Fail Reconn Success

Measurement Counters

ID Counter Description
1278085477 CELLCELL.OUT.CELL.HO.FAIL.RECONN.SUCC.TIMES H375A:Outgoing Inter-Cell Handover Fail Reconn Success

Description

This counter provides the number of times that the handover fails but the MS successfully returns to the old channel during the intra-BSC and inter-BSC outgoing cell handover.

In the intra-BSC outgoing cell handover procedure or the inter-BSC outgoing cell handover procedure, the BSC starts the timer for the HO CMP message after sending the HO CMD message to the MS in the original cell. Before the timer expires, the BSC receives the HO FAIL message from the original cell and the MS returns to the old channel.

Measurement point

When the intra-BSC outgoing cell handover fails, the BSC triggers the measurement once the MS returns to the old channel.

When the inter-BSC outgoing cell handover fails, the BSC triggers the measurement once the MS returns to the old channel.
