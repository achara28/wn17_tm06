Measurement of Completed Handovers

Measurement Counters

ID Counter Description
1278074427 TRX.HO.CMPL.SD R4430B:Completed Handovers (SDCCH)
1278074428 TRX.HO.CMPL.TCH R4439B:Completed Handovers (TCH)

Description

In the SDCCH handover procedure, the MS sends an HO ACC message to the BTS of the target cell after receiving an HO CMD message to attempt to access the new cell. The BTS notifies the HO ACC message in an HO DETECT message to the BSC. In the case of asynchronous handover (the BTS of the source cell is not the BTS of the target cell), the BTS of the target cell sends a PHY INFO message to the MS. After receiving the PHY INFO message, the MS sends a FIRST SABM frame, sends an EST IND message through the target cell to the BSC to notify the BSC that the MS has successfully accessed the target cell, and sends an HO CMP message to the BSC. When the BSC receives the HO CMP message, the counter R4430B:TRX.HO.CMPL.SD is incremented by one for the TRX where the SDCCH of the target cell resides.

In the TCH handover procedure, the MS sends an HO ACC message to the BTS of the target cell after receiving an HO CMD message to attempt to access the new cell. The BTS notifies the HO ACC message in an HO DETECT message to the BSC. In the case of asynchronous handover (the BTS of the source cell is not the BTS of the target cell), the BTS of the target cell sends a PHY INFO message to the MS. After receiving the PHY INFO message, the MS sends a FIRST SABM frame, sends an EST IND message through the target cell to the BSC to notify the BSC that the MS has successfully accessed the target cell, and sends an HO CMP message to the BSC. When the BSC receives the HO CMP message, the counter R4439B:TRX.HO.CMPL.TCH is incremented by one for the TRX where the TCH of the target cell resides.

Measurement point

This measurement counter is involved in the Intra-Cell Handover Measurement, Incoming Internal Inter-Cell Handover Measurement, Incoming External Inter-Cell Handover Measurement, and Incoming Inter-RAT Inter-Cell Handover Measurement. See the corresponding sections for the flow charts. Each time the BSC receives an HO CMP message from an MS, the counter is incremented by one.
