Average Payload of Single Uplink EGPRS TBF

Measurement Counters

ID Counter Description
1279179449 UP.EGPRS.RLC.SGL.TBF.AVG.THRPUT TL9233:Average Payload of Single Uplink EGPRS TBF

Description

This measurement provides the average payload of a single EGPRS TBF at the uplink RLC in a cell in a granularity period. This measurement indicates the type of the service processed by the user. The measurements of uploading and downloading services are higher than those of the WAP services.

Measurement point

This counter is calculated based on the total number of uplink EGPRS RLC data blocks using MCS-1, MCS-2, MCS-3, MCS-4, MCS-5, MCS-6, MCS-7, MCS-8, and MCS-9 coding schemes and the total number of uplink EGPRS TBFs in a measurement period.

Formula

Average Payload of Single Uplink EGPRS TBF = (Total Number of Uplink EGPRS MCS1 RLC Data Blocks x 22 + Total Number of Uplink EGPRS MCS2 RLC Data Blocks x 28 + Total Number of Uplink EGPRS MCS3 RLC Data Blocks x 37 + Total Number of Uplink EGPRS MCS4 RLC Data Blocks x 44 + Total Number of Uplink EGPRS MCS5 RLC Data Blocks x 56 + Total Number of Uplink EGPRS MCS6 RLC Data Blocks x 74 + Total Number of Uplink EGPRS MCS7 RLC Data Blocks x 56 + Total Number of Uplink EGPRS MCS8 RLC Data Blocks x 68 + Total Number of Uplink EGPRS MCS9 RLC Data Blocks x 74)/(Total Number of Uplink EGPRS TBFs x 1024)

Unit

Kbytes
