Description

The measurement of the preceding counters provides the number of times in the measurement report with the downlink receive quality rank ranging from 0 to 7 over the TCHF.

Measurement point

The BSC measures these counters based on the downlink receive level (0-7) on the TCHF when receiving the valid MR or pre-processing MR, and stops to measure these counters when receiving the DISCONNECT or CLEAR COMMAND.
