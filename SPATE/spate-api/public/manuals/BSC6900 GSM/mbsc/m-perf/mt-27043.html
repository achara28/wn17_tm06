Description

AR3024A:CELL.CH.AVAIL.NUM.PDCH.FROM.DYN.PDCH.900.AVR provides the average number of available PDTCHs converted from dynamic PDCHs in the GSM900, GSM900/DCS1800, GSM850, GSM850/DCS1800, GSM850/PCS1900 and TGSM810 cells controlled by the BSC in a measurement period.

AR3024B:CELL.CH.AVAIL.NUM.PDCH.FROM.DYN.PDCH.1800.AVR provides the average number of available PDTCHs on the DCS1800 and PCS1900 frequency bands that are converted from dynamic PDCHs in DCS1800, PCS1900, GSM900&DCS1800, GSM850&DCS1800, GSM850&PCS1900 cells controlled by the BSC in a measurement period.

CR3024:CELL.CH.AVAIL.NUM.PDCH.FROM.DYN.PDCH.AVR provides the number of available PDTCHs converted from dynamic PDCHs in the cells on all the frequency bands controlled by the BSC in real-time measurement.

Measurement point

The number of available channels(from dynamic PDCH convert to PDCH) is sampled and accumulated in the measurement period. At the end of the measurement period, the value of this counter is obtained through the division of the accumulated value by the number of sampling times.The BSC does not measure this type of counter if dynamic PDCH cannot be configured because the BSC is configured with a built-in PCU.
