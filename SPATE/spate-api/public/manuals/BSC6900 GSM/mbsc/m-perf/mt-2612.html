Number of N-Bit Single Block Channel Requests Received on RACH

Description

This measurement provides the number of N-bit single block request messages received on the RACH in a granularity period. This measurement indicates the type of the N-bit single block service performed on the MS. Where, N can be 8 or 11.

Measurement point

The following figure shows the procedure for the MS initiating a single block channel request for packet access on the RACH. The CHANNEL REQUEST message carries a parameter indicating the access request type is single-block packet access. Each time the BSC receives an N-bit CHANNEL REQUEST message indicating single-block packet access from the MS on the RACH (see measurement point A in Figure 1), the counter is incremented by one. Where, N can be 8 or 11.
