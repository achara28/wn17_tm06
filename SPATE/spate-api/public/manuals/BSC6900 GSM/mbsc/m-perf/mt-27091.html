Description

AR3551A:CELL.CH.BUSY.NUM.FR.TRAF.900.AVR provides the average number of busy TCHs (TCHF) in the GSM900, GSM900/DCS1800, GSM850, GSM850/DCS1800, GSM850/PCS1900 and TGSM810 cells controlled by the BSC at each sampling point.

AR3551B:CELL.CH.BUSY.NUM.FR.TRAF.1800.AVR provides the average number of busy TCHs (TCHF) in the DCS1800 and PCS1900 cells controlled by the BSC at each sampling point.

AR3552A:CELL.CH.BUSY.NUM.HR.TRAF.900.AVR provides the average number of busy TCHs (TCHH) in the GSM900, GSM900/DCS1800, GSM850, GSM850/DCS1800, GSM850/PCS1900 and TGSM810 cells controlled by the BSC at each sampling point.

AR3552B:CELL.CH.BUSY.NUM.HR.TRAF.1800.AVR provides the average number of busy TCHs (TCHH) in the DCS1800 and PCS1900 cells controlled by the BSC at each sampling point.

Measurement point

The average number of busy TCHFs (TCHHs) in the GSM900, GSM850, TGSM810, DCS1800, and PCS1900 cells controlled by the BSC is sampled and accumulated in a sampling period of 5s. At the end of the measurement period, the value of this counter is obtained through dividing the accumulated value by the number of sampling times.
