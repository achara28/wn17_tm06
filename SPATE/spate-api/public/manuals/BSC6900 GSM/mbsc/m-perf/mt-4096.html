Number of REJ Frames Sent Through LAPD Link

Measurement Counters

ID Counter Description
1275079419 OS.LAPD.Lnk.Q921.REJ.Tx L5003:Number of REJ frames sent through LAPD link

Description

This counter provides the number of REJ frames sent by the BSC through the LAPD link. In the technical specifications for L2, a reject (REJ) frame is a monitoring frame, which means a rejection of the received information (I) frame. A REJ frame contains the sequence number of the I frame that is expected to be sent from the peer. I frames are numbered in sequence and the sequence numbers of I frames received and sent by the LAPD entity of the BSC are all recorded. Normally, the value of this counter should be 0. When the sequence number of an I frame received by the BSC is not the expected one, the BSC sends an REJ frame to the peer end and this counter is incremented by one.

Measurement point

When an unexpected frame is received though the LAPD link, the BSC sends an REJ frame to the peer end. In this case, the measurement is triggered at the BSC side and this counter is incremented by one.
