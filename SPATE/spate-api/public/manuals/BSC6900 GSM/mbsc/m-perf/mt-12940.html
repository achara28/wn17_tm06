Total Bytes of Downlink LLC PDUs of Downlink Dual-Carrier Users

Measurement Counters

ID Counter Description
1282431388 TX.DOWN.DLDC.LLC.PDU.TOTAL.BYTE L9571:Total Bytes of Downlink LLC PDUs of Downlink Dual-Carrier Users

Description

This counter provides the total bytes of downlink LLC PDUs of downlink dual-carrier users in a measurement period.

This counter indicates the total bytes of downlink LLC PDUs received by all downlink dual-carrier users in a cell in a measurement period.

This counter, along with the counter Downlink Throughput of Downlink Dual-Carrier Users at the LLC Layer, is used to calculate the downlink data rate of downlink dual-carrier users at the LLC layer. This counter, along with the counters Downlink Throughput of Downlink Dual-Carrier Users at the LLC Layer, Downlink Data Rate of Downlink Dual-Carrier Users at the LLC Layer, and Maximum Downlink Data Rate of Downlink Dual-Carrier Users at the LLC Layer, is used to analyze the downlink transmission performance of downlink dual-carrier users at the LLC layer.

Measurement point

When the SGSN sends a Downlink LLC PDU to the BSC, the bytes of the Downlink LLC PDU are accumulated to the counter Total Bytes of Downlink LLC PDUs of Downlink Dual-Carrier Users when the following conditions are met:

The Downlink LLC PDU is destined for a downlink dual-carrier user.

The BSC receives the latest PACKET DOWNLINK ACK/NACK message from MS for the Downlink LLC PDU (see measurement point A in figure 1).

The Downlink LLC PDU is not a signaling message such as ATTACH, DETACH, PDP context activation, PDP context deactivation, and routing area update.

Figure 1 This measurement is triggered when the BSC receives a PACKET DOWNLINK ACK/NACK message
