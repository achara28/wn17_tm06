Description

In the assignment procedure, the BSS assigns a radio channel to an MS for signaling transmission. After a radio resource connection is established between the MS and the BSS, a mobility management and call control connection to the MSC is established through the radio channel. Based on the call control connection request sent from the MS, the MSC assigns terrestrial resources to the MS and specifies the channel type that is required in the assignment procedure in the ASS REQ message. If TCHFs or TCHHs are specified in the ASS REQ message, the BSC measures this counter after receiving the message.

Channel Assignment Requests (TCHF), Channel Assignment Requests (TCHH), and Channel Assignment Requests (TCH) are measured on the basis of channel assignment. Channel Assignment Requests (TCHF) and Channel Assignment Requests (TCHH) are measured if the type of the channel to be allocated is specified. Channel Assignment Requests (TCH) is measured if the type of the channel to be allocated is not specified.

Measurement point

As shown at point A in figure 1, the previous counters are measured when the BSC receives the ASS REQ message from the MSC and the channel type that the message carries is TCHF or TCHH.

In the channel assignment procedure, if TCHFs need to be assigned preferentially but TCHHs are actually assigned, the following settings are required:
  • Channel Assignment Requests (TCH) is incremented by two.
  • Channel Assignment Requests (TCHH) is incremented by one.
  • Channel Assignment Requests (TCHF) is incremented by one.
