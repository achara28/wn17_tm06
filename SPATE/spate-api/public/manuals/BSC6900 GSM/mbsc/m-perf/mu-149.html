Uplink EGPRS TBF Establish and Release Capability Measurement(PSCALL.UpETbfEstbRel.Cell)

Overview

The measurement unit Uplink EGPRS TBF Establish and Release Capability Measurement provides the measurement counters related to the uplink EGPRS TBF establishment and release in a granularity period. This measurement unit indicates the PS services load of a cell and is used as a reference for PS services model establishment.
