Counters Related to the Rate for Receiving Data on the HDLC Link

Measurement Counters

ID Counter Description
73415492 VS.HDLCLINK.PEAK.RXRATE T7808:Maximum Rate for Receiving Data on the HDLC Link
73415662 VS.HDLCLINK.MEAN.RXRATE T7810:Average Rate for Receiving Data on the HDLC Link

Description

These counters provide the receive rate of an HDLC link in the specified measurement period.

Measurement point

Samples are taken every five seconds in the specified measurement period. The HDLC link receives data from the circuit side. The system obtains the received data(kbit) once every five seconds. Then, the counters are calculated in the following formulas:

Average receive rate:

HDLC_LINK_AVE_RCV_RATE = Ave(Rcvdata/PFM_TIME_LENGTH)

Maximum receive rate:

HDLC_LINK_MAX_RCV_RATE = Max(SampleRcvdata/SAMPLING_TIMER_LENGTH)

PFM_TIME_LENGTH: Measurement period.

SAMPLING_TIMER_LENGTH: Sampling period (five seconds).

Rcvdata: Data received on the HDLC link within the PFM_TIME_LENGTH.

SampleRcvdata: Data received on the HDLC link within the SAMPLING_TIMER_LENGTH.

HDLC_LINK_AVE_RCV_RATE: Average receive rate within the measurement period.

HDLC_LINK_MAX_RCV_RATE: Maximum receive rate within the measurement period.

Formula

None

Unit

kbit/s
