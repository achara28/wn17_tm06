Downlink GPRS RLC Data Transfer Capability Measurement(PSCALL.DownGRlcDataTran.Cell)

Overview

The measurement unit Downlink GPRS RLC Data Transfer Capability Measurement provides the measurement counters related to the downlink GPRS RLC transmission performance in a cell in a granularity period. This measurement unit indicates the PS service load of a cell and is used as a reference for PS service model establishment.
