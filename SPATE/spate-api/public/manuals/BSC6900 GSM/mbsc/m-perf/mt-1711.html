1278077537 CELL.INTRACELL.HO.FAIL.NONE.AMR.TCHH.TCHF H3202B:Failed Internal Intra-Cell Handovers (Non-AMR) (TCHH-TCHF)

Description

  • For a non-AMR call, when the BSC determines that an intra-cell handover from TCHF to TCHH is required based on the handover algorithm, the BSC measures the H3200A:CELL.INTRACELL.HO.REQ.NONE.AMR.TCHF.TCHH counter if the handover is allowed.
  • For a non-AMR call, when the BSC determines that an intra-cell handover from TCHH to TCHF is required based on the handover algorithm, the BSC measures the H3200B:CELL.INTRACELL.HO.REQ.NONE.AMR.TCHH.TCHF counter if the handover is allowed.
  • For a non-AMR call, if the BSC determines that an intra-cell handover from TCHF to TCHH is required based on the handover algorithm, the BSC measures the H3201A:CELL.INTRACELL.HO.CMD.NONE.AMR.TCHF.TCHH counter when the handover procedure is initiated and a handover command is issued.
  • For a non-AMR call, if the BSC determines that an intra-cell handover from TCHH to TCHF is required based on the handover algorithm, the BSC measures the H3201B:CELL.INTRACELL.HO.CMD.NONE.AMR.TCHH.TCHF counter when the handover procedure is initiated and a handover command is issued.
  • For a non-AMR call, if the BSC determines that an intra-cell handover from TCHF to TCHH is required based on the handover algorithm, the BSC measures the H3202A:CELL.INTRACELL.HO.FAIL.NONE.AMR.TCHF.TCHH counter when the handover is initiated but the handover fails due to a certain cause.
  • For a non-AMR call, if the BSC determines that an intra-cell handover from TCHH to TCHF is required based on the handover algorithm, the BSC measures the H3202B:CELL.INTRACELL.HO.FAIL.NONE.AMR.TCHH.TCHF counter when the handover is initiated but the handover fails due to a certain cause.

Measurement point

For the description of the Measurement Points of the counters, refer to the Measurement Points of the common handovers. For details, see counters related to Intra-cell Handover Measurement per Cell.
