Measurement of Completed Mode Modify Procedures

Measurement Counters

ID Counter Description
1278074424 TRX.CHMD.CMPL.TCH R4429B:Completed Mode Modify Procedures (TCH)

Description

In the Mode Modify procedure, the MS sends an SABM frame to the BTS after receiving a CHAN MODE MODIFY message to indicate that the air interface is already in the status of multi-frame link establishment. The BTS returns a UA frame to the MS as an acknowledgement and sends an EST IND message to notify the BSC that a link is successfully established on the air interface. Upon receipt of the UA frame, the MS sends a CHAN MODE MODIFY ACK message to the BSC. When the BSC receives the CHAN MODE MODIFY ACK message from the MS, the counter R4429B:TRX.CHMD.CMPL.TCH is incremented by one for the TRX where the TCH seized by the MS resides.

Measurement point

As shown in Figure 1, each time the BSC receives a CHANMOD MODIFY ACK message from the MS (see measurement point E2), the counter is incremented by one.
