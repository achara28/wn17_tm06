Description

This measurement provides the retransmission rate of uplink EGPRS RLC data blocks in UAS7-UAS11 coding schemes in a cell in a granularity period. If the measurement value is high, it indicates that the signal quality is poor. In this case, the network needs to be optimized.

Measurement point

See measurement points of the counters in the formula.

Formula

Retransmission Rate of Uplink EGPRS UAS7 RLC Data Block = ([Total Number of Uplink EGPRS UAS7 RLC Data Blocks] - [Total Number of Valid Uplink EGPRS UAS7 RLC Data Blocks]) x 100/[Total Number of Uplink EGPRS UAS7 RLC Data Blocks]

"Retransmission Rate of Uplink EGPRS UASx RLC Data Block" is similar as [Retransmission Rate of Uplink EGPRS UAS7 RLC Data Block]

Note: UASx indicates UAS8-UAS11.

Unit

%
