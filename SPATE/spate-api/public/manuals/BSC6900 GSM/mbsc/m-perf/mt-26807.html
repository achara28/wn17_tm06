Outgoing Inter-RAT Inter-Cell Handover Commands

Measurement Counters

ID Counter Description
1278083471 CELL.INTERRAN.OUTCELL.HO.CMD CH351:Outgoing Inter-RAT Inter-Cell Handover Commands

Description

In the outgoing inter-RAT inter-cell handover procedure, the BSC sends the MS the HO CMD message through the originating cell after receiving the HO CMD message from the MSC. This measurement provides the number of the HO CMD messages sent by the BSC to the MS. The measurement is performed based on the originating cell and involves handovers on the SDCCH, TCHF, and TCHH.

Measurement point

As shown in Figure 1 and Figure 2, the counter is measured at B when the BSC sends the HO CMD message to the MS.

Figure 1 Measurement of the counters related to outgoing inter-RAT inter-cell handover (excluding directed retry)

Measurement points Description
A Measurement point of Outgoing Inter-RAT Inter-Cell Handover Requests (Excluding Directed Retry)
B Measurement point of Outgoing Inter-RAT Inter-Cell Handover Commands (Excluding Directed Retry)
C Measurement point of Successful Outgoing Inter-RAT Inter-Cell Handovers (Excluding Directed Retry)
Figure 2 Measurement of the counters related to outgoing inter-RAT inter-cell handover (directed retry)

Measurement points Description
A Measurement point of Outgoing Inter-RAT Inter-Cell Handover Requests (Directed Retry)
B Measurement point of Outgoing Inter-RAT Inter-Cell Handover Commands (Directed Retry)
C Measurement point of Successful Outgoing Inter-RAT Inter-Cell Handovers (Directed Retry)

Formula

Outgoing Inter-RAT Inter-Cell Handover Commands = [Outgoing Inter-RAT Inter-Cell Handover Commands (SDCCH) (Excluding Directed Retry)] + [Outgoing Inter-RAT Inter-Cell Handover Commands (TCHF)] + [Outgoing Inter-RAT Inter-Cell Handover Commands (TCHH)]
