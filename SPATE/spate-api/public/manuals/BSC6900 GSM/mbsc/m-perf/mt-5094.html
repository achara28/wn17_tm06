Description

These counters provide the number of RLSD messages received by the SCCP with different causes.Services may be affected if multiple connections are abnormally released.

The following table describes the meaning of each counter.

Counter Description
VS.SCCP.Rx.RLSD.SubsFail This counter provides the number of RLSD messages received by the SCCP with the cause of subsystem faults.
VS.SCCP.Rx.RLSD.MtpFail This counter provides the number of RLSD messages received by the SCCP with the cause of message transfer part (MTP) faults.
VS.SCCP.Rx.RLSD.Cong This counter provides the number of RLSD messages received by the SCCP with the cause of network congestion.
VS.SCCP.Rx.RLSD.Other This counter provides the number of RLSD messages received by the SCCP for other abnormal causes except for subsystem faults, MTP faults, and network congestion.
VS.SCCP.RX.RLSD.NORMAL This counter provides the number of RLSD messages for normal causes.

Measurement point

In the measurement period, these counters are incremented by one when the SCCP receives the RLSD messages with different causes.
