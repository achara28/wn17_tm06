Description

In the outgoing inter-RAT inter-cell handover procedure, the BSC sends the MS the HO CMD message through the originating cell after receiving the HO CMD message from the MSC. The specific counter provides the number of outgoing inter-RAT inter-cell handover commands sent by the BSC in the originating cell. Together with the Outgoing Inter-RAT Inter-Cell Handover Requests and the Failed Outgoing Inter-RAT Inter-Cell Handovers, this counter determines the performance of the outgoing inter-RAT inter-cell handover.

The following counter provides the number of outgoing inter-RAT inter-cell handover commands (excluding directed retry), when the BSC sends the HO CMD message to the MS and the MS seizes an SDCCH before the handover:

  • H3510:CELL.INTERRAN.OUTCELL.HO.CMD.NOT.INCLUDE.DR.SD
The following counter provides the number of outgoing inter-RAT inter-cell handover commands (including directed retry), when the BSC sends the HO CMD message to the MS and the MS seizes an SDCCH before the handover:
  • H3510A:CELL.INTERRAN.OUTCELL.HO.CMD.DR.SD
The following two counters provide respectively the number of outgoing inter-RAT inter-cell handover commands when the MS seizes a TCHF or a TCHH before the handover. In this measurement, both the handovers initiated by directed retry and those initiated according to the handover algorithms based on the measurement reports are included.
  • H3517:CELL.INTERRAN.OUTCELL.HO.CMD.TCHF
  • H3518:CELL.INTERRAN.OUTCELL.HO.CMD.TCHH
If the BSC enables the local switching function, then the calls that meet the local switching condition of the BSC use the local switching function of the BSC for the conversation and those that meet the local switching condition of the BTS use the local switching function of the BTS for the conversation.

The following counters provide the numbers of outgoing inter-RAT inter-cell handover commands respectively for the calls using the BSC local switch and those using the BTS local switch. They are measured when the BSC sends the HO CMD message to the MS.

  • H351Aa:CELL.INTERRAN.OUTCELL.HO.CMD.BSC.LOOP
  • H351Ab:CELL.INTERRAN.OUTCELL.HO.CMD.BTS.LOOP
