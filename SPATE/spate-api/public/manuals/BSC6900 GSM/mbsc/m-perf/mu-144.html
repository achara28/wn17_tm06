PS MS Radio Access Capability(PSCALL.MSCAP.CELL)

Overview

This counter unit provides the number of times RAT-capable MSs processing PS services in a cell. This counter can be used to determine whether certain optimization functions can be enabled based on the RAT capability of MSs, and guide network planning, To measure this counter, run the SET GCELLSOFT command with MS Capability Statistic Switch set to ON. The RAT capability of MSs includes the followings:

Supporting GPRS extended dynamic allocation

Supporting EGPRS extended dynamic allocation

Supporting modulation based multislot class

Supporting multiple TBF procedures in A/Gb mode

Supporting Downlink Advanced Receiver Performance

Supporting extended RLC/MAC control message segmentation

Supporting PS handover
