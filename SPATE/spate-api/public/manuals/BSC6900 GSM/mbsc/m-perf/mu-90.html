Measurement of MS Capability per Cell(CALL.MSCAP.CELL)

Overview

When the MS sends the CLASSMARK CHANGE message, or when the MSC sends the CLASSMARK UPDATE message or the incoming BSC handover request message, the message may contain the classmark information (classmarks 2 and 3). When the MS sends the SETUP or CALL CONFIRM message, the message contains Bearer Capability information. This measurement unit measures the MS capability based on the classmark information and Bearer Capability information in the message. The following items are involved in the measurement of MS capability per cell: version support, support for early classmark sending, support for A5 ciphering algorithm, reception capability of short messages, VGCS support, VBS support, multiband support, 3G access support, GERAN support, SAIC support, support for repeated ACCH, DTM support, and bearing capability.Before starting measurement of MS capability per cell, set the MS Capability Statistic Switch parameter to ON.
