Description

VS.E1T1.ES.ESR: This counter measures the proportion of the received errored seconds (ESs) in the specified measurement period.

VS.E1T1.ES.TESR: This counter measures the proportion of the transmitted errored seconds (TESs) in the specified measurement period.

Measurement point

Samples are taken every second in the specified measurement period.

Formula

None

Unit

%
