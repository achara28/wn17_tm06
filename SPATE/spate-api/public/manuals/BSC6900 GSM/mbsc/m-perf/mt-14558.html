Number of Uplink DTM TBF Abnormal Releases Due to N3101 Overflow

Measurement Counters

ID Counter Description
1282433006 DTM.UP.TBF.ABNORM.REL.N3101.OVERFLOW A9819:Number of Uplink DTM TBF Abnormal Releases Due to N3101 Overflow

Description

This counter provides the number of uplink TBF abnormal releases for DTM MSs due to N3101 overflow in a measurement period. This counter as well as the counter Number of Successful Uplink DTM TBF Establishments indicates the uplink TBF call drop rate for DTM MSs in a measurement period. If the value of this counter is large, the radio quality is poor in the cell.

Measurement point

During uplink TBF transmission for DTM MSs, the BSC reserves uplink blocks for the MS and allocates a valid Uplink State Flag (USF) for scheduling these data blocks. In addition, the BSC attaches the USF to downlink blocks that the MS listens to, instructing the MS to upload data on the related uplink blocks. If the BSC receives valid uplink data blocks on the data blocks reserved for the MS within a period specified by a time on the BSC, the counter N3101 is reset. Otherwise, the counter N3101 is incremented by one. If N3101 overflows, the BSC releases the uplink TBF for DTM MSs. This counter is incremented by one when N3101 overflows.
