CS Pagings Discarded on LAPD Link

Measurement Counters

ID Counter Description
1277871418 LAPDLNK.DISC.CS.PAGS A531:CS Pagings Discarded on LAPD Link

Description

This measurement provides the number of times when the CS paging is discarded on the LAPD link.

Measurement point

If the BSC decides to discard the CS paging because of an overload on the Abis interface board or on the LAPD link is detected, this counter is incremented by one at A4, as shown in Figure 1.

Figure 1 CS paging procedure

Measurement points Description
A1 Measurement point of MSC-Initiated Paging Requests for CS Service received by the BSC from the MSC over the A interface
A2 Measurement point of SGSN-Initiated Paging Requests for CS Service received by the BSC from the MSC over the Gs interface
A3 Measurement point of Delivered Paging Messages for CS Service and Delivered to BTS for CS Service
A4 Measurement point of SM Pagings Discarded on LAPD Link and CS Pagings Discarded on LAPD Link
