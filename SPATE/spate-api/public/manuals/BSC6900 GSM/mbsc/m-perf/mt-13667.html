Number of Calls with Medium SNR of Input Signals

Measurement Counters

ID Counter Description
1282432115 CELL.TRAU.ANC.MEDIUMSNR.INPUT A3336:Number of Calls with Medium SNR of Input Signals

Description

This counter provides the number of calls in which the level of the downlink input signal-to-noise ratio (SNR) ranges from 3 dB to 10 dB in the measurement period. The downlink input SNR is the ratio of the downlink voice received by the BSC transcoder (TC) to the uplink background noise received by the BSC TC. The higher the downlink SNR, the more clearly a local receiver can hear the voice of a remote speaker. The larger the value of this counter, the more the calls in which the downlink voice received by the BSC TC is a little higher than the uplink background noise received by the BSC TC.

Measurement point

This counter is incremented by one when the level of the downlink input SNR in a call ranges from 3 dB to 10 dB in the measurement period.
