Description

AR3010A:CELL.CH.NUM.SD.900.AVR provides the average number of configured SDCCHs in the GSM900, GSM900/DCS1800, GSM850, GSM850/DCS1800, GSM850/PCS1900 and TGSM810 cells controlled by the BSC in a measurement period. The channel types can be SDCCH8, SDCCH+CBCH, combined BCCH, and BCCH+CBCH. This number includes the number of SDCCHs that are dynamically converted from TCHs.

AR3010B:CELL.CH.NUM.SD.1800.AVR provides the average number of configured SDCCHs in the DCS1800 and PCS1900 cells controlled by the BSC in a measurement period. The channel types can be SDCCH8, SDCCH+CBCH, combined BCCH, and BCCH+CBCH. This number includes the number of SDCCHs that are dynamically converted from TCHs.

CR3010:CELL.CH.NUM.SD.AVR provides the average number of configured SDCCHs in the cells on all the frequency bands controlled by the BSC in real-time measurement. The channel types can be SDCCH8, SDCCH+CBCH, combined BCCH, and BCCH+CBCH. This number includes the number of SDCCHs that are dynamically converted from TCHs.

Measurement point

The number of dynamically configured channels is sampled and accumulated in the sampling period of 5s. The channel type can be SDCCH8, SDCCH+CBCH, combined BCCH, and BCCH+CBCH. At the end of the measurement period, the value of this counter is obtained through dividing the accumulated value by the number of sampling times.
