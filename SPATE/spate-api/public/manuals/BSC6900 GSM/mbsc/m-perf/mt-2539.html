Delivered Paging Messages for CS Service

Measurement Counters

ID Counter Description
1277870417 CELL.PAGES.CS A330:Delivered Paging Messages for CS Service

Description

When an MS is paged for CS services, the BSC receives a CS paging message from the MSC, PCU, or SGSN and sends a PAGING COMMAND message to the corresponding cell based on the CI contained in the message. This counter is measured based on the cell. This counter provides the number of pagings over the Abis interface. Refer to GSM Rec. 08.58 and 04.08.

Measurement point

This counter is measured when the BSC sends a PAGING COMMAND for CS service to the cell under the BTS, at A3 as shown in Figure 1.

Figure 1 CS paging procedure(PCU)

Measurement points Description
A1 MSC-Initiated Paging Requests for CS Service received by the BSC from the MSC over the A interface
A2 SGSN-Initiated Paging Requests for CS Service received by the BSC over the Gs interface
A3 Delivered Paging Messages for CS Service and Delivered to BTS for CS Service
A4 SM Pagings Discarded on LAPD Link and CS Pagings Discarded on LAPD Link
