Description

These counters provide the numbers of valid MRs or preprocessing MRs with the downlink receive quality rank ranging from 0 to 7 on the TCHFs in one location group.

Measurement point

These counters are measured when the BSC receives valid MRs or preprocessing MRs from the BTS based on the downlink receive level on the TCHFs in one location group.
