Description

During the incoming internal inter-cell handover procedure, if the handover fails because of various causes before the BSC receives HO CMP message from the MS, the specific counter is measured based on the target channel types, handover failure causes, and handover initiation causes. With Incoming Internal Inter-Cell Handover Requests and Incoming Internal Inter-Cell Handover Responses, this counter determines the performance of incoming internal inter-cell handovers. With the counters related to the outgoing internal inter-cell handovers, this counter determines the performance of internal inter-cell handovers.

The following counters provide the number of incoming internal inter-cell handover failures based on the target channel types and the frequency bands between which the handover is performed. The channel types include the SDCCH and TCH.

  • H3220W:CELL.INTRABSC.INCELL.HO.FAIL.SD.900.900
  • H3220X:CELL.INTRABSC.INCELL.HO.FAIL.SD.1800.1800
  • H3220Y:CELL.INTRABSC.INCELL.HO.FAIL.SD.900.1800
  • H3220Z:CELL.INTRABSC.INCELL.HO.FAIL.SD.1800.900
  • H3229W:CELL.INTRABSC.INCELL.HO.FAIL.TCH.900.900
  • H3229X:CELL.INTRABSC.INCELL.HO.FAIL.TCH.1800.1800
  • H3229Y:CELL.INTRABSC.INCELL.HO.FAIL.TCH.900.1800
  • H3229Z:CELL.INTRABSC.INCELL.HO.FAIL.TCH.1800.900
The following counters are measured based on the specific handover failure causes.
1) The following counters are measured when the incoming internal inter-cell handover fails because of failed allocation of relevant resources.
  • H3220A:CELL.INTRABSC.INCELL.HO.FAIL.CONG.SD provides the number of incoming internal inter-cell handover failures because no channel resource is available when the target channel is an SDCCH.
  • H3229A:CELL.INTRABSC.INCELL.HO.FAIL.CONG.TCH provides the number of failed incoming internal inter-cell handovers because no channel resource is available when the target channel is a TCH.
  • H322H:CELL.INTRABSC.INCELL.HO.FAIL.NO.SPEECH.VER provides the number of incoming internal inter-cell handover failures because no proper speech version is available based on the speech versions carried in the ASS REQ message from the MSC, the speech versions supported by the target cell, and the speech versions permitted by the A interface phase tag. This counter is measured when the BSC fails to obtain the speech version.
  • H322M:CELL.INTRABSC.INCELL.HO.FAIL.NO.IDLE.ABIS provides the number of incoming internal inter-cell handover failures because no Abis CS resource is available in the target cell when the Abis dynamic allocation is enabled.
  • H322B:CELL.INTRABSC.INCELL.HO.FAIL.APPLY.TRSL.FAIL provides the number of incoming internal inter-cell handover failures because the BSC fails establish the terrestrial link.
2) The following counters are measured when the incoming internal inter-cell handover fails because the MS fails to access the new channel on the Um interface and also fails to return to the old channel.
  • H3220C:CELL.INTRABSC.INCELL.HO.FAIL.EXP.SD
  • H3227Cb:CELL.INTRABSC.INCELL.HO.FAIL.EXP.TCHF.SIG
  • H3228Cb:CELL.INTRABSC.INCELL.HO.FAIL.EXP.TCHH.SIG
  • H3227Ca:CELL.INTRABSC.INCELL.HO.FAIL.EXP.TCHF.TRAF
  • H3228Ca:CELL.INTRABSC.INCELL.HO.FAIL.EXP.TCHH.TRAF
After successfully allocates and activates the channel in the target cell, if the BSC releases the call because T3103 expires before the BSC receives the HO CMP message from the MS, the specific counter is measured in the target cell based on the target channel, that is, signaling channel (SDCCH/TCHF/TCCH) or traffic channel (TCHF/TCHH).
  • H322D:CELL.INTRABSC.INCELL.HO.FAIL.RECONN.SUCCESS: After the BSC successfully allocates and activates the channel in the target cell, it sends an HO CMD message to the MS, and starts the T3103 to wait for the HO CMP message. This counter is measured in the target cell if the BSC receives an HO FAIL message on the old channel after the MS returns to the old channel before timer T3103 expires.
3) The following counters are measured when the incoming internal inter-cell handover fails because of equipment faults.
  • H322I:CELL.INTRABSC.INCELL.HO.FAIL.CHACT.FAIL: After the BSC successfully performs channel allocation and speech version confirmation in the target cell, it sends a CH ACT message to the BTS for activating the channel, and starts the corresponding timer to wait for the response. This counter is measured if the BSC receives a CH ACT NACK or no response from the BTS before the timer expires.
  • H322G:CELL.INTRABSC.INCELL.HO.FAIL.A.INTERF.FAIL provides the number of failed outgoing internal inter-cell handovers because the BSC detects an SS7 link failure on the A interface and releases the call.
4) The following counters are measured based on the loop types. The loop types include the BSC loop and BTS loop.
  • H322a:CELL.INTRABSC.INCELL.HO.FAIL.BSC.LOOP
  • H322:CELL.INTRABSC.INCELL.HO.FAIL.BTS.LOOP

  • H322D:CELL.INTRABSC.INCELL.HO.FAIL.RECONN.SUCCESS
As shown in Figure 1, the following counter is measured at A, B, E, F, G, and H if the incoming internal inter-cell handover fails because the BSC detects an SS7 link failure on the A interface and releases the call:
  • H322G:CELL.INTRABSC.INCELL.HO.FAIL.A.INTERF.FAIL
