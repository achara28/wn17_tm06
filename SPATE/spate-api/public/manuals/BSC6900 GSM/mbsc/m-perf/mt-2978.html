Description

This counter provides the total number of valid uplink EGPRS RLC data blocks using MCS1 to MCS9 coding schemes in a cell in a measurement period. A high-rate coding scheme indicates large data volume transmitted on each data block. A large proportion of valid uplink RLC data blocks indicates fewer retransmissions of uplink RLC data blocks.

Measurement point

This counter is incremented by one when an EGPRS RLC data block received by the BSC from the MS is defined as a valid uplink data block in a measurement period. The data block is defined as valid when all the following conditions are met: 1. The data block uses an MCSx coding scheme. 2. The block sequential number of the data block is within the range of an RLC receive window. 3. The data block receiving state in the receive state array V(N) on the BSC side is NACKED, UNRECEIVED, or RECEIVED. (MCSx indicates MCS1 to MCS9)
