Description

After receiving the measurement report from the MS, the BSC determines whether to initiate a handover and allocates the channels in the target cell, and then sends an HO CMD to the MS in the originating cell.

The specific counter provides the number of HO CMD messages sent by the BSC to the MS during the incoming internal inter-cell handover procedure. With the Incoming Internal Inter-Cell Handover Requests and the Failed Incoming Internal Inter-Cell Handovers, these counters determine the incoming internal inter-cell handover performance. With the counters related to the outgoing internal inter-cell handover, these counters determine the internal inter-cell handover performance.

The following counters provide the number of incoming internal inter-cell handover requests based on the target channel types and the frequency bands between which the handover is performed.

  • H3210W:CELL.INTRABSC.INCELL.HO.CMD.SD.900.900
  • H3210X:CELL.INTRABSC.INCELL.HO.CMD.SD.1800.1800
  • H3210Y:CELL.INTRABSC.INCELL.HO.CMD.SD.900.1800
  • H3210Z:CELL.INTRABSC.INCELL.HO.CMD.SD.1800.900
  • H3217W:CELL.INTRABSC.INCELL.HO.CMD.TCHF.900.900
  • H3217X:CELL.INTRABSC.INCELL.HO.CMD.TCHF.1800.1800
  • H3217Y:CELL.INTRABSC.INCELL.HO.CMD.TCHF.900.1800
  • H3217Z:CELL.INTRABSC.INCELL.HO.CMD.TCHF.1800.900
  • H3218W:CELL.INTRABSC.INCELL.HO.CMD.TCHH.900.900
  • H3218X:CELL.INTRABSC.INCELL.HO.CMD.TCHH.1800.1800
  • H3218Y:CELL.INTRABSC.INCELL.HO.CMD.TCHH.900.1800
  • H3218Z:CELL.INTRABSC.INCELL.HO.CMD.TCHH.1800.900
The following counters are measured based on the loop types. The loop types include the BSC local loop and BTS local loop.
  • H321a:CELL.INTRABSC.INCELL.HO.CMD.BSC.LOOP
  • H321b:CELL.INTRABSC.INCELL.HO.CMD.BTS.LOOP
