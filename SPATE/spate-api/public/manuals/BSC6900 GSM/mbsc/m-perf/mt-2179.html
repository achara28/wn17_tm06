Sampled Measurement of Initially Configured Channels (CBCH)

Measurement Counters

ID Counter Description
1278469431 CELL.CH.CFG.NUM.CBCH.900 R3009A:Number of Initially Configured Channels (CBCH) (900/850/810 band)
1278469432 CELL.CH.CFG.NUM.CBCH.1800 R3009B:Number of Initially Configured Channels (CBCH) (1800/1900 Band)

Description

Cell broadcast channels (CBCHs) are used by the BSS system to forward cell broadcast messages from the CBC to MSs. For the areas where cell broadcast services are enabled, you can configure CBCHs by configuring BCCH+CBCH or SDCCH+CBCH.

R3009A:CELL.CH.CFG.NUM.CBCH.900 provides the number of configured CBCHs in the GSM900, GSM900/DCS1800, GSM850, GSM850/DCS1800, GSM850/PCS1900 and TGSM810 cells controlled by the BSC at each sampling point.

R3009B:CELL.CH.CFG.NUM.CBCH.1800 provides the number of configured CBCHs in the DCS1800 and PCS1900 cells controlled by the BSC at each sampling point.

Measurement point

The BSC takes the samples of the channel allocation status every five seconds. When the sampling timer expires, R3009A:CELL.CH.CFG.NUM.CBCH.900 or R3009B:CELL.CH.CFG.NUM.CBCH.1800 are measured according to the frequency band of the cell and the number of configured BCCH+CBCH or SDCCH+CBCH channels.At the end of the granularity period, the value reported within the last five seconds is used as the value of the counter.
