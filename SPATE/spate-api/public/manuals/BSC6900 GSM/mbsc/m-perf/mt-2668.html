Number of Failed Outgoing Inter-RAT Inter-Cell PS Handovers Between 2G and 4G

Measurement Counters

ID Counter Description
1282429810 CELL.2G4GINTERRAN.OUTCELL.PSHO.FAIL H9522:Number of Failed Outgoing Inter-RAT Inter-Cell PS Handovers Between 2G and 4G

Description

This measurement provides the number of failed outgoing inter-RAT inter-cell PS handovers between 2G and 4G in a granularity period.

This counter indicates the condition that the system fails to hand over PS services on the local cells to other 4G cells.

Together with "Number of Outgoing Inter-RAT Inter-Cell PS Handover Attempts Between 2G and 4G", this counter is used to calculate the outgoing 2G/4G inter-RAT PS handover success rate. If the handover fails, the reason may be that no channel is available in the target cell, the originating cell does not receive any response message after sending the PS handover command, or the originating cell receives the PS handover failure message from the MS after sending the PS handover command.

Together with "Number of Outgoing Inter-RAT Inter-Cell PS Handover Attempts Between 2G and 4G", this counter is used to analyze the performance of outgoing 2G/4G inter-RAT PS handovers.

Measurement point

The BSC determines whether to initiate an outgoing inter-cell PS handover, based on the measurement report information from the MS, the Service UTRAN CCO IE from the CN, the priority of the radio access technology (RAT) used in the cell, and the status of the cell where the MS belongs to. This counter is incremented by one when the BSC receives a non acknowledged message upon the PS handover request, or the BSC fails to receive a response after sending a PS handover command to the MS, or the BSC receives a network-controlled cell reselection failure message, after the outgoing inter-cell PS handover procedure is triggered with the target cell being a 4G cell.
