Downlink Bandwidth Usage of the BC

Measurement Counters

ID Counter Description
1279369426 BC.TRAN.DOWN.UTILIZATION.RATE RL9610:Downlink bandwidth usage of the BC

Description

This counter provides the rate of the actually used downlink bandwidth to the configured bandwidth of the BC in a measurement period. It indicates the downlink usage of the BC. If the usage exceeds 80%, you can infer that the downlink traffic is heavy and the BC bandwidth needs to be expanded; otherwise, you can infer that the downlink traffic is light and the bandwidth may be wasted. This counter can be measured after the BC is configured.

Measurement point

This counter is calculated by using the following formula.

Formula

Downlink bandwidth usage of the BC = [Downlink bandwidth actually used on the BC] * 100/[Configured bandwidth of the BC]

Unit

%
