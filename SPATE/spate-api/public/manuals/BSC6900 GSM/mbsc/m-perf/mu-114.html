Channel Assignment Concentric Cell Measurement per Cell(CHAN.AllocIuoOrig.CELL)

Overview

In the concentric cell, the number of TRXs in the underlaid subcell and overlaid subcell should be properly configured. In a wide-coverage area, the TRX in the underlaid subcell provides the remote coverage and that in the overlaid subcell

The measurement unit Channel Assignment Concentric Cell Measurement per Cell is used to measure the specifications, such as channel assignment requests and channel assignment overflows. Based on these specifications, you can analyze the channel assignment of the concentric cell and check whether handover parameters of the concentric cell are correctly set.
