Description

The MS measures the information of the serving cell and neighbor cells, and sends the information to the BTS in an MR. The BTS then sends the information to the BSC in an MR. This counter is used to measure the number of times that the signal strength difference between the serving cell and each neighbor cell contained in the MRs is between two successive Neighbor Cell Interference Level Thresholds.

Measurement point

After receiving the valid MR from the MS, the BSC compares the receive level of the serving cell with the receive level of each neighbor cell. A specific counter is measured in the following situations: N ranges from 1 to 7.

The counter S364:CELL.SCELL.NCELL.LEV.GT.THRES1.MR.NUM is incremented by one when the following formula is met: serving cell receive level +64 >= neighbor cell receive level + Neighbor Cell Interference Level Threshold 1.

The counter S365:CELL.SCELL.NCELL.LEV.GT.THRES2.MR.NUM is incremented by one when the following formula is met: serving cell receive level +64 >= neighbor cell receive level + Neighbor Cell Interference Level Threshold 2.

The counter S366:CELL.SCELL.NCELL.LEV.GT.THRES3.MR.NUM is incremented by one when the following formula is met: serving cell receive level +64 >= neighbor cell receive level + Neighbor Cell Interference Level Threshold 3.

The counter S367:CELL.SCELL.NCELL.LEV.GT.THRES4.MR.NUM is incremented by one when the following formula is met: serving cell receive level +64 >= neighbor cell receive level + Neighbor Cell Interference Level Threshold 4.

The counter S368:CELL.SCELL.NCELL.LEV.GT.THRES5.MR.NUM is incremented by one when the following formula is met: serving cell receive level +64 >= neighbor cell receive level + Neighbor Cell Interference Level Threshold 5.

The counter S369:CELL.SCELL.NCELL.LEV.GT.THRES6.MR.NUM is incremented by one when the following formula is met: serving cell receive level +64 >= neighbor cell receive level + Neighbor Cell Interference Level Threshold 6.

The counter S370:CELL.SCELL.NCELL.LEV.GT.THRES7.MR.NUM is incremented by one when the following formula is met: serving cell receive level +64 >= neighbor cell receive level + Neighbor Cell Interference Level Threshold 7.

The measurement object of this counter is configurable. If the measurement object is not configured, this counter is not measured.
