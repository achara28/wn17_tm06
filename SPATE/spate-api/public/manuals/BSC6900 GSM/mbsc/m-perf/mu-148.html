Uplink EGPRS RLC Data Transfer Capability Measurement(PSCALL.UpERlcDataTran.Cell)

Overview

The measurement unit Uplink EGPRS RLC Data Transfer Capability Measurement provides the measurement counters related to the uplink EGPRS RLC transmission in a cell in a granularity period. This measurement unit indicates the PS service load of a cell and is used as a reference for PS service model establishment.
