SCTPIP Measurement(SCTPIP)

Overview

Based on the IP protocol, the SCTP link provides reliable point-to-point transmission of signaling messages for the Base Station Control. SCTPIP measurement provides the counters related to the traffic on the SCTPIP link.

For details about the SCTP protocol, see the protocol RFC4960.
