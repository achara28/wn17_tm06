Incoming Inter-RAT Inter-Cell Handover Measurement per Cell (TDD)(CALL.InterRanTDDIntoCellHo.CELL)

Overview

In this document, the MSC, BSC, and BTS in the GSM system is called MSC1, BSC1, and BTS1 respectively; the MSC in the TD-SCDMA (TD) system is called MSC2. When MSC1 receives a handover request message from MSC2, MSC1 finds BSC1 based on both the target cell information in the message and the cell-BSC relation among the data configuration of itself. Then, MSC1 sends an HO REQ message to BSC1.

Upon the receipt of the message, BSC1 allocates channel resources based on the target cell information in the message. If the channel is successfully allocated, BSC1 sends a CHAN ACTIV message to the target cell. After receiving the CHAN ACTIV ACK message from the target cell, BSC1 requests circuit service resources. If the request is successful, BSC1 sends an HO REQ ACK message to MSC1. After receiving an HO CMD message from the TD system, the MS sends an HO ACC message to BTS1 to access the new system. BTS1 sends an HO DETECT message to MSC through BSC1 and sends a PHY INFO message to the MS.

The MS sends a FIRST SABM frame to access the network. BTS1 sends a EST IND message to BSC1 and a UA frame to the MS for acknowledgement.

After the MS accesses the network, it sends an HO CMP message to MSC1. The inter-RAT incoming cell handover (TDD) is complete.
