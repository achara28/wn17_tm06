Description

This counter provides the number of times that the timeslot occupation rate is N/M in a measurement period. N indicates the number of timeslots actually allocated to a downlink TBF in the transmission state, and M indicates the number of timeslots that can be allocated according to the calculation based on the service type and MS multislot capability. A larger N indicates higher timeslot usage. The value range of N and M is from 1 to 5, and N is equal to or less than M.

Measurement point

This counter is incremented by one when the number of actually allocated downlink timeslots is N and the maximum number of downlink timeslots can be allocated (which is calculated based on multislot capability and service type of the MS) is M for a downlink TBF in the transmission state in a measurement period. The BSC periodically samples all the downlink TBFs, and the sampling period is 5s.
