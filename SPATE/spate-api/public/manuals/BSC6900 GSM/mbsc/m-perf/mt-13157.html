Successful Channel Conversions (PDCH-SDCCH)

Measurement Counters

ID Counter Description
1282431605 CELL.CH.CONVERT.SUCC.NUM.PDCH.SD R3525C:Successful Channel Conversions (PDCH-SDCCH)

Description

In situations where dynamic SDCCH adjustment is allowed, the BSC attempts to convert a TCH or PDCH to an SDCCH if the BSC detects that the number of idle SDCCHs in the cell is smaller than a predefined threshold during SDCCH assignment. If no idle TCH is available, the BSC attempts to convert a PDCH to an SDCCH. This counter provides the number of successful times of converting the PDCH to the SDCCH.

Measurement point

This counter is measured at the BSC when the BSC successfully converts the PDCH to the SDCCH.
