Counters Related to the Transmit Rate of a PPP Link Queue

Measurement Counters

ID Counter Description
67204582 VS.PPP.QUEUE.PEAK.TXRATE T7386:Maximum Transmit Rate of a PPP Link Queue
67204583 VS.PPP.QUEUE.MEAN.TX T7387:Average Transmit Rate of a PPP Link Queue

Description

These counters provide the transmit rate of a PPP link queue in the specified measurement period. The rates measured here include the spending of the header and the payload of the PPP frame.

Measurement point

Samples are taken every five seconds in the specified measurement period. The system obtains the number of bytes transmitted on the link once every five seconds. With this, the system obtains the maximum number of bytes transmitted in five seconds, and the total number of one measurement period. The maximum speed is calculated with the maximum bytes in five seconds, and the average speed is calculated with the total bytes in one measurement period.

Formula

None

Unit

kbit/s
