BTS Power Consumption by Adding Static Power Consumption

Measurement Counters

ID Counter Description
1282449256 SITE.EnerConsump.ADDING CR2791:BTS Power Consumption by Adding Static Power Consumption

Description

This counter provides the BTS power consumption calculated by adding up the static power consumption of each module in a measurement period. The static power consumption of BTS modules consists of two parts: 1. Some modules report their power consumption to the BTS. 2. The typical static power consumption of each type of board of a BTS is obtained by querying the static power consumption table. The BSC accumulates the preceding power consumption to obtain the power consumption of the entire BTS. This method does not have any special requirements for BTS hardware (for example, does not require a PMU or smart electricity meter) and is applicable to all types of base stations. The power consumption of a single-mode base station is indicated by this counter. The power consumption of a multi-mode base station is indicated by this counter plus other counters corresponding to the base station mode.

Measurement point

The modules (including TRX modules) that are capable of measuring digital power or software report their power consumption to the BBU. The BBU queries the power consumption of the modules that are incapable of measuring digital power or software in the static power consumption table. Then, the BBU adds up the power consumption of each module and obtains the power consumption of the entire BTS. The BBU reports the power consumption of the entire BTS to the BSC every 5 minutes, and then the BSC accumulates the power consumption and obtains the value of this counter.
