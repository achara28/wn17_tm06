Description

AR3018A:CELL.CH.NUM.HR.900.AVR provides the average number of configured TCHHs in the GSM900, GSM900/DCS1800, GSM850, GSM850/DCS1800, GSM850/PCS1900 and TGSM810 cells controlled by the BSC in a measurement period. This number includes the number of TCHHs that are dynamically converted from TCHFs.

AR3018B:CELL.CH.NUM.HR.1800.AVR provides the average number of configured TCHHs in the DCS1800 and PCS1900 cells controlled by the BSC in a measurement period. This number includes the number of TCHHs that are dynamically converted from TCHFs.

CR3018:CELL.CH.NUM.HR.AVR provides the average number of configured TCHHs in the cells on all the frequency bands controlled by the BSC in real-time measurement. This number includes the number of TCHHs that are dynamically converted from TCHFs and from dynamic PDCHs.

Measurement point

The number of TCHH dynamically configured channels is sampled and accumulated in the measurement period. At the end of the measurement period, the value of this counter is obtained through the division of the accumulated value by the number of sampling times.
