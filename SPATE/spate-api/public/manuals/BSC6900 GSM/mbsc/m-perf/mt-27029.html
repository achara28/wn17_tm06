Description

AR3015A:CELL.CH.NUM.EDGE.900.AVR provides the average number of configured EDGE PDCHs in the GSM900, GSM900/DCS1800, GSM850, GSM850/DCS1800, GSM850/PCS1900 and TGSM810 cells controlled by the BSC in a measurement period. This number includes the number of EDGE PDCHs that are dynamically converted from dynamic EDGE PDCHs.

AR3015B:CELL.CH.NUM.EDGE.1800.AVR provides the average number of configured EDGE PDCHs on the DCS1800 and PCS1900 frequency bands in DCS1800, PCS1900, GSM900&DCS1800, GSM850&DCS1800, GSM850&PCS1900 cells controlled by the BSC in a measurement period. The value of this counter includes the number of EDGE PDCHs that are dynamically converted from dynamic EDGE PDCHs.

CR3015:CELL.CH.NUM.EDGE.AVR provides the average number of configured EDGE PDCHs in the cells on all the frequency bands controlled by the BSC in real-time measurement. This number includes the number of EDGE PDCHs that are dynamically converted from dynamic EDGE PDCHs.

Measurement point

The number of EDGE dynamically configured channels is sampled and accumulated in the measurement period. At the end of the measurement period, the value of this counter is obtained through the division of the accumulated value by the number of sampling times.
