Number of VAMOS Channel Multiplexing Attempts in Overlaid Subcell (Assignment)

Measurement Counters

ID Counter Description
1282432166 CELL.VAMOS.ASS.TRY.OLCELL H3064:Number of VAMOS Channel Multiplexing Attempts in Overlaid Subcell (Assignment)

Description

This counter provides the total number of attempts to multiplex a new call during assignment and an established call onto one VAMOS channel in an overlaid subcell. This counter can be used to calculate other counters, such as the proportion of the number of channel multiplexing attempts during assignment to the total number of channel multiplexing attempts and the success rate of channel multiplexing attempts during assignment. If the value of this counter is much smaller than that of Number of Successful VAMOS Candidate Call Decisions (Assignment), more calls can be multiplexed by increasing the thresholds for path loss offset and VAMOS overload in a channel multiplexing decision during assignment.

Measurement point

This counter is incremented by one when VAMOS channel multiplexing is triggered during assignment in an overlaid subcell. (VAMOS channel multiplexing during assignment is triggered when the following conditions are met: The VAMOS channel multiplexing function is enabled, the cell load is higher than the specified load threshold, a new call that requires a half-rate channel can be selected as a VAMOS candidate call, and the new call is successfully assigned a VAMOS channel.)
