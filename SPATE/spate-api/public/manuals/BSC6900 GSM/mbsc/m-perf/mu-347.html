Channel Assignment Success Measurement per TRX(CHAN.AllocSucc.TRX)

Overview

Channel Assignment Success Measurement per TRX is used to measure the successful channel assignments in the following procedures based on different channel types:

  • Immediate assignment
  • Assignment
  • Internal intra-cell handover
  • Incoming internal inter-cell handover
  • Incoming external inter-cell handover
  • Local Call
