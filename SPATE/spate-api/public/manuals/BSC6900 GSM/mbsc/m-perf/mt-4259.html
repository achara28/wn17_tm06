Number of discarded NS PDUs on NSVC

Measurement Counters

ID Counter Description
1279469419 NSVC.TRAN.DISCARD.NS.PDU L9703:Number of discarded NS PDUs on NSVC

Description

This measurement provides the total number of NS PDUs discarded on an NSVC in Gb over FR in a granularity period.

Measurement point

In a granularity period, the counter is incremented by one every time the BSC discards an uplink or a downlink NS PDU message on an NSVC. If the value of the counter is great, check whether each NSVC configured for the NSE is in normal state. Then, check other counters to determine whether the great value of the counter is caused by the heavy traffic. If the traffic is too heavy, the capacity of the Gb interface is recommended to be expanded.
