Description

This Item measures the number of successful TCH (traffic channel) seizures by the MS during TCH handovers.

Measurement point

1.Internal intra-cell or inter-cell handover procedure

The BSC measures the counter in the target cell based on the channel type when it receives a HANDOVER DETECT message from the BTS, as shown by point E in Figure 1 and Figure 2.
