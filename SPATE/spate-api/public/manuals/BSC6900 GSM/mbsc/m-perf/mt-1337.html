Description

If the incoming inter-RAT inter-cell handover (TDD) fails during the handover procedure, the specific counter is measured in the target cell under control of the target BSC, based on the type of the target channel and the cause of the failure. Together with Incoming Inter-RAT Inter-Cell Handover Requests (TDD) and Incoming Inter-RAT Inter-Cell Handover Responses (TDD), this counter indicates the performance of the incoming inter-RAT inter-cell handover (TDD).

The following counters provide the numbers of failed incoming inter-RAT inter-cell handovers (TDD), based on the specific type of the target channel: SDCCH, TCHF, or TCHH. They are measured upon the termination of the incoming inter-RAT inter-cell handover (TDD).

  • H36201:CELL.INTERRAN.TDD.INCELL.HO.FAIL.SD
  • H36271:CELL.INTERRAN.TDD.INCELL.HO.FAIL.TCHF
  • H36281:CELL.INTERRAN.TDD.INCELL.HO.FAIL.TCHH
The following counters provide the number of failed incoming inter-RAT inter-cell handovers (TDD) due to the specific cause, and are measured upon the termination of the incoming inter-RAT inter-cell handover (TDD).

1) The following counter provides the number of a failed incoming inter-RAT inter-cell handover (TDD) because the BSC fails to decode an HO REQ message with invalid contents from the MSC:

  • H362J1:CELL.INTERRAN.TDD.INCELL.HO.FAIL.INVALID.CONTENTS
2) The following counters provide the numbers of failed incoming inter-RAT inter-cell handovers (TDD) due to failures in allocating relevant resources.

The following counters provides respectively the number of failed incoming inter-RAT inter-cell handovers (TDD) because the SDCCH or TCH is unavailable in the target cell. The specific counter is measured when the channel allocation fails.

  • H3620A1:CELL.INTERRAN.TDD.INCELL.HO.FAIL.CONG.SD
  • H3629A1:CELL.INTERRAN.TDD.INCELL.HO.FAIL.CONG.TCH
The following counters provides the number of failed incoming inter-RAT inter-cell handovers because no proper speech version is available based on the speech versions carried in the HO REQ message from the MSC, the speech versions supported by the target cell, and the speech versions permitted by the A interface phase tag.
  • H362L1:CELL.INTERRAN.TDD.INCELL.HO.FAIL.NO.SPEECH.VER
The following counters provides the number of failed incoming inter-RAT inter-cell handovers (TDD) because the BSC fails to establish the terrestrial link.
  • H362B1:CELL.INTERRAN.TDD.INCELL.HO.FAIL.APPLY.TRSL.FAIL
The following counters provides the number of failed incoming inter-RAT inter-cell handovers (TDD) because the specified CIC on the A interface is unavailable, after the BSC receives the HO REQ from the MSC.
  • H362E1:CELL.INTERRAN.TDD.INCELL.HO.FAIL.NO.CIC
The following counters provides the number of failed incoming inter-RAT inter-cell handovers (TDD) because the specified CIC on the A interface has already been allocated to another MS, after the BSC receives the HO REQ from the MSC.
  • H362F1:CELL.INTERRAN.TDD.INCELL.HO.FAIL.CIC.ALLOC
3) The following counters provide the numbers of failed incoming inter-RAT inter-cell handovers (TDD) because the MS fails to access the new channel on the Um interface.

The following counters provides respectively the number of failed incoming inter-RAT inter-cell handovers (TDD), based on the channel type and service type of the target channel (SDCCH, TCH as signaling channel, and TCH as traffic channel), because timer T3121 expires after the BSC allocates and activates the channel in the target cell and sends an HO REQ ACK message to the MSC.

  • H3620C1:CELL.INTERRAN.TDD.INCELL.HO.FAIL.EXP.SD
  • H3629Cb1:CELL.INTERRAN.TDD.INCELL.HO.FAIL.EXP.TCH.SIG
  • H3629Ca1:CELL.INTERRAN.TDD.INCELL.HO.FAIL.EXP.TCH.TRAF.CH
4) The following counter provides the number of failed incoming inter-RAT inter-cell handovers (TDD) because the BSC releases the call after receiving the CLR CMD message from the MSC.

The following counters provides the number of failed incoming inter-RAT inter-cell handovers (TDD) because the BSC releases the call after receiving the CLEAR CMD message from the MSC.

  • H362H1:CELL.INTERRAN.TDD.INCELL.HO.FAIL.MSC.CLR.CMD
The main causes of failed incoming inter-RAT inter-cell handovers (TDD) include:

1) Causes related to message decoding

The BSC fails to decode the HO REQ message.

2) Causes related to resource allocation

No traffic channel is available.

No speech version is available.

The BSC fails to obtain the terrestrial resources.

No CIC is available.

The CIC is already allocated.

3) Causes related to access on the Um interface

The timer for the BSC to wait for an HO CMP message expires.

4) Other causes

The BSC releases the call after receiving the CLR CMD from the MSC.

As shown in Figure 1, during the incoming inter-RAT inter-cell handover (TDD) procedure, the following counters are measured at point A when the incoming inter-RAT inter-cell handover (TDD) fails because the message has an invalid content, no suitable channel is available in the target cell, or no proper speech version is available after the BSC receives an HO REQ message from the MSC:

  • H362J1:CELL.INTERRAN.TDD.INCELL.HO.FAIL.INVALID.CONTENTS
  • H3620A1:CELL.INTERRAN.TDD.INCELL.HO.FAIL.CONG.SD

  • H362L1:CELL.INTERRAN.TDD.INCELL.HO.FAIL.NO.SPEECH.VER
As shown in Figure 1, the following counters are measured at point B if the incoming inter-RAT inter-cell handover (TDD) fails because the CIC specified in the HO REQ message is unavailable or has already been allocated to another MS, the BSC fails to apply for the terrestrial resources, or no circuit resource is available:
  • H362B1:CELL.INTERRAN.TDD.INCELL.HO.FAIL.APPLY.TRSL.FAIL
  • H362E1:CELL.INTERRAN.TDD.INCELL.HO.FAIL.NO.CIC
  • H362F1:CELL.INTERRAN.TDD.INCELL.HO.FAIL.CIC.ALLOC
As shown in Figure 1, during the incoming inter-RAT inter-cell handover (TDD) procedure, the following counters are measured at point C if T3121 expires after the BSC sends the HO REQ ACK message to the MSC.
  • H3620C1:CELL.INTERRAN.TDD.INCELL.HO.FAIL.EXP.SD
  • H3629Cb1:CELL.INTERRAN.TDD.INCELL.HO.FAIL.EXP.TCH.SIG
  • H3629Ca1:CELL.INTERRAN.TDD.INCELL.HO.FAIL.EXP.TCH.TRAF.CH
As shown in Figure 1, the following counter is measured at point D when the BSC releases the call after receiving the CLEAR CMD message from the MSC.
  • H362H1:CELL.INTERRAN.TDD.INCELL.HO.FAIL.MSC.CLR.CMD
As shown in Figure 1, the following counters are measured at points A, B, C, and D when the incoming inter-RAT inter-cell handover (TDD) fails.

  • H36271:CELL.INTERRAN.TDD.INCELL.HO.FAIL.TCHF
  • H36281:CELL.INTERRAN.TDD.INCELL.HO.FAIL.TCHH
