Description

In the outgoing inter-RAT inter-cell handover (TDD) procedure, after receiving the HO CMD message from the MSC, the BSC sends the HO CMD message to the MS in the originating cell. The specific counter provides the number of outgoing inter-RAT inter-cell handover commands (TDD) sent by the BSC in the originating cell. Together with the Outgoing Inter-RAT Inter-Cell Handover Requests (TDD) and the Failed Outgoing Inter-RAT Inter-Cell Handovers (TDD), this counter determines the performance of the outgoing inter-RAT inter-cell handover (TDD).

The following counter provides the number of outgoing inter-RAT inter-cell handover commands (TDD) (excluding directed retry), when the BSC sends the HO CMD message to the MS and the MS seizes an SDCCH before the handover:

  • H35101:CELL.INTERRAN.TDD.OUTCELL.HO.CMD.NOT.INCLUDE.DR.SD
The following counter provides the number of outgoing inter-RAT inter-cell handover commands (TDD) (including directed retry), when the BSC sends the HO CMD message to the MS and the MS seizes an SDCCH before the handover:
  • H35101A: CELL.INTERRAN.TDD.OUTCELL.HO.CMD.DR.SD
The following two counters provide respectively the number of outgoing inter-RAT inter-cell handover commands when the MS seizes a TCHF or a TCHH before the handover. In this measurement, both the handovers initiated by directed retry and those initiated according to the handover algorithms based on the measurement reports are included.
  • H35171:CELL.INTERRAN.TDD.OUTCELL.HO.CMD.TCHF
  • H35181:CELL.INTERRAN.TDD.OUTCELL.HO.CMD.TCHH
