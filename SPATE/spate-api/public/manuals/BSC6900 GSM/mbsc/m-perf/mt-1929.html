1282429905 CELL.TRAU.TFO.EST.AMR.TCHH A3305:Number of Successful TFO Establishments (AMR HR)

Description

This counter provides the number of successful Tandem Free Operation (TFO) establishments during calls that use the AMR HR speech version in the measurement period. Using the TFO feature to bypass encoding/decoding helps improve the voice quality. Therefore, the larger the value of this counter, the greater the improvement in the average voice quality of AMR HR calls with the TFO feature enabled. The speech versions can be categorized into the following types: HR, FR, EFR, AMR FR, AMR HR, and AMR WB.

Measurement point

This counter is incremented by one when a TFO establishment is successful during AMR HR calls in the measurement period. This counter is measured by cell when a handover or call ends.
