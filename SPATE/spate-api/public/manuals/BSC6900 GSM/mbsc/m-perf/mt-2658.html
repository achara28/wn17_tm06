Number of Outgoing Inter-Cell PS Handover Attempts

Measurement Counters

ID Counter Description
1282429799 CELL.OUTCELL.PSHO.ATMPT H9712:Number of Outgoing Inter-Cell PS Handover Attempts

Description

This measurement provides the number of outgoing inter-cell PS handover attempts in a granularity period. The value of this counter is related to cell reselection policies such as the radio access technology (RAT) used over the Um interface, downlink transmission quality, cell load, neighboring cell level, and priority of the RAT over the Um interface.

This counter indicates the condition that the system attempts to hand over PS services on the local cells to other cells.

Together with "Number of Failed Outgoing Inter-Cell PS Handovers", this counter is used to calculate the outgoing inter-cell PS handover success rate. If the handover fails, the reason may be that no channel is available in the target cell, the originating cell does not receive any response message after sending the PS handover command, or the originating cell receives the PS handover failure message from the MS after sending the PS handover command.

Together with "Number of Failed Outgoing Inter-Cell PS Handovers", this counter is used to analyze the performance of outgoing inter-cell PS handovers.

Measurement point

The BSC determines whether to initiate an outgoing inter-cell PS handover, based on the measurement report information from the MS, the Service UTRAN CCO IE from CN, the priority of the radio access technology (RAT) used in the cell, and the status of the cell where the MS belongs to. This counter is incremented by one when the outgoing inter-cell PS handover procedure is triggered due to any reasons.
