PDCH Resource Capability Measurement(PSCHAN.PdchRes.Cell)

Overview

PDCH Resource Capability Measurement is used to measure the usage of PDCHs in a cell. Available PDCHs are those bound with transmission resources and stayed in normal state. Occupied PDCHs are those carrying TBFs.
