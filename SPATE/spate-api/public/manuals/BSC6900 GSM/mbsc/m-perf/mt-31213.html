5-Minute Measurement of the Highest Transmit Rates of of the Logic Port

Measurement Counters

ID Counter Description
73441533 VS.LGCPRT.FIVE.MINUTES.PEAK.TXRATE T7540:Maximum Transmit Rate of Logical Port(5-Minute)

Description

These measurement counters provide the transmission rate of the logic port in the 5-Minute measurement period. The rates measured here include the spending of the header of the Link layer frame(could be MAC, PPP or MP frame depending on the bearing type of the logic port) and the payload.

Measurement point

1.Within a specified measurement period, the system measures the number of transmitted bytes every 5 seconds and accumulates the number of bytes at the end of every 5-minute sampling period to obtain the total number of bytes transmitted within the sampling period.

2.The system calculates the transmit rate within every 5-minute sampling period with the following formula: Transmit rate within a 5-minute sampling period = Total number of bytes transmitted within a 5-minute sampling period/300

3.The system compares all of the collected transmit rates to obtain the highest transmit rate within the specified measurement period, which is the value of the corresponding counter.

Formula

None

Unit

kbit/s
