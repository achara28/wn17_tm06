L3 Data Requests Received by BTS on LAPD Link

Measurement Counters

ID Counter Description
1276079425 LAPD.DATA L5109:L3 Data Requests Received by BTS on LAPD Link

Description

This counter provides the number of requests that are sent from LAPD Link L3 to L2 and contain the to-be-confirmed messages in a measurement period. The BTS measures the counter and sends the value to the BSC. The requests sent from L3 consist of the messages that need or need not be confirmed. The former are transferred in L2 I frames and the latter are transferred in L2 UI frames. All the L3 messages sent by the BTS except for the MR need be confirmed. If the BTS L2 receives one request containing the to-be-confirmed message from L3, this Item increases by 1. If the physical uplink is disrupted, the measurement result cannot be sent to the BSC, and this Item does not increase.

Measurement point

The BTS reports the number of to-be-confirmed message transmission requests which are received over the LAPD link on L3 to the BSC every 4 minutes in a measurement period. The BSC accumulates the received numbers in the measurement period to obtain the value of this counter.
