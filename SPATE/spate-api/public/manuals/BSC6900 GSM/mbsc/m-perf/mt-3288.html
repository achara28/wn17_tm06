Failed Incoming Inter-Cell Handovers

Measurement Counters

ID Counter Description
1278084418 CELLCELL.INCELL.HO.FAIL.CONG H382A:Failed Incoming Inter-Cell Handovers (Congestion)
1278084419 CELLCELL.INCELL.HO.FAIL.OTHER H382M:Failed Incoming Inter-Cell Handovers (Other Causes)

Description

In the incoming inter-cell handover procedure, if the handover fails, this counter is measured based on the cause of congestion or others. Along with Incoming Inter-Cell Handover Requests, this counter indicates whether the failure rate of incoming-cell handovers is abnormal. Along with the Outgoing-Cell Handover Measurement concerning the neighboring cells, this counter also indicates the performance of inter-cell handovers.

The handover failure can be caused by congestion or others.

In the measurement of the counter "H382M: Failed Incoming Inter-Cell Handovers (Other Causes)", the cause includes direct retry.

Measurement point

The originating cell and the target cell are GSM cells. For the measurement point, refer to Failed Incoming Internal Inter-Cell Handovers, and Failed Incoming External Inter-Cell Handovers. If the cause of the handover failure is Radio Resource Unavailable, the counter H382A:CELLCELL.INCELL.HO.FAIL.CONG is measured; otherwise, the counter H382M:CELLCELL.INCELL.HO.FAIL.OTHER is measured.
