Total Bytes of Downlink GPRS Background Service LLC PDUs Sent

Measurement Counters

ID Counter Description
1279184427 TX.DOWN.BKGND.LLC.PDU.TOTAL.BYTE L9511:Total Bytes of Downlink GPRS Background Service LLC PDUs Sent

Description

This measurement counter provides the total bytes of downlink GPRS background service LLC PDUs in a granularity period.

Measurement point

Figure 1 shows the procedure in which the BSC checks the received PACKET DOWNLINK ACK/NACK message to determine whether the DOWNLINK LLC PDU is successfully sent to the MS. Each time a DOWNLINK LLC PDU is successfully sent and then deleted from the PDU sending queue (see measurement point A), the bytes of the DOWNLINK LLC PDU are accumulated to the counter. The TBF is a GPRS TBF that neither supports bandwidth guarantee nor supports the ARP and THP.
