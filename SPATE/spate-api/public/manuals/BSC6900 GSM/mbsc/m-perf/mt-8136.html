Number of Bursts with High-Order Modulation Power Increase of 3.6 dB

Measurement Counters

ID Counter Description
1282412294 HIGH.MOD.PWR.UPG.BURST.36.NUM HM318:Number of Bursts with High-Order Modulation Power Increase of 3.6 dB

Description

This counter provides the number of bursts whose increased transmit power is greater than 3.2 dB but is equal to or less than 3.6 dB after the high-order modulation power boost function is enabled.

When the TRX uses the 8PSK, QPSK, 16QAM, or 32QAM high-order modulation scheme to handle data services with a wide bandwidth, employing the high-order modulation power boost function can enhance corresponding TRX output power, expand coverage area, and allow high-speed PS services to reach the same coverage area as CS service. The counters associated with the high-order modulation power boost assess the function's efficiency.

After using the high-order modulation power boost function, the BTS measures the counters according to the TRX. Every five minutes afterwards, the RC Statistics Report message sent through the RSL reports the counters to the BSC for measurement.

Measurement point

This counter is incremented by one when bursts of PS services on the TRX use the high-order modulation power boost function and the power increase is greater than 3.2 dB but equal to or less than 3.6 dB. This counter is measured at the BTS.
