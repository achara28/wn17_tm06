Successful VGCS Uplink Seizures

ID Counter Description
1278174418 CELL.UL.REQ.SUCC A03443:Successful VGCS Uplink Seizures

Description

After receiving a TALKER DETECTION message, the BSC sends an UPLINK REQUEST message to the MSC if an uplink TCH is idle. After receiving the ESTABLISH INDICATION message from the BTS and the UPLINK REQUEST ACKNOWLEDGE message from the MSC, the BSC sends an UPLINK REQUEST CONFIRM message to the MSC. This measurement provides the number of UPLINK REQUEST CONFIRM messages that the BSC replies to the MSC.

Measurement point

This counter is incremented by one when the BSC replies an UPLINK REQUEST CONFIRM message to the MSC after receiving the ESTABLISH INDICATION message from the BTS and the UPLINK REQUEST ACKNOWLEDGE message from the MSC. In the signaling procedure, this counter is measured at point B in Figure 1.
