Number of Successful Incoming Internal Inter-Cell Handovers (Overlaid Subcell to Overlaid Subcell)

Measurement Counters

ID Counter Description
1278078519 CELL.INBSC.INTO.CELL.HO.SUCC.OLCELL.OLCELL H323II:Number of Successful Incoming Internal Inter-Cell Handovers (Overlaid Subcell to Overlaid Subcell)

Description

When a handover between different concentric cells is performed under the same BSC, the performance of the handover is measured. Taking the target cell as the measurement object, this measurement provides the number of successful handovers from the overlaid subcell of the originating cell to the overlaid subcell of the target cell. The handovers involve the handover between traffic channels, handover between signaling channels, and directed retry.

Measurement point

See measurement points of the counters in the formula.

Formula

Number of Successful Incoming Internal Inter-Cell Handovers (Overlaid Subcell to Overlaid Subcell) = [Number of Incoming Internal Inter-Cell Handover Commands (Overlaid Subcell to Overlaid Subcell)] - [Number of Unsuccessful Incoming Internal Inter-Cell Handovers (Overlaid Subcell to Overlaid Subcell)]
