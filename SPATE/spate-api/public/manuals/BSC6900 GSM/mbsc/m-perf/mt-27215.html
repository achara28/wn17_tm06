Description

This measurement provides the retransmission rates of uplink EGPRS RLC data blocks in MCS1-MCS9 coding schemes in a cell in a granularity period. If the measurement value is high, you can infer that the signal quality is poor, thus the network needs to be optimized.

Measurement point

See measurement points of the counters in the formula.

Formula

Retransmission Rate of Uplink EGPRS MCS1 RLC Data Block = ([Total Number of Uplink EGPRS MCS1 RLC Data Blocks] - [Total Number of Valid Uplink EGPRS MCS1 RLC Data Blocks]) x 100/[Total Number of Uplink EGPRS MCS1 RLC Data Blocks]

"Retransmission Rate of Uplink EGPRS MCSx RLC Data Block" is similar as [Retransmission Rate of Uplink EGPRS MCS1 RLC Data Block]

Note: MCSx indicates MCS1-MCS9.

Unit

%
