Number of Uplink 8PSK_MEAN_BEP Different Value Measurement(Cell)(PSCALL.Up8pskMBepDifNum.Cell)

Overview

The PSCALL.Up8pskMBepDifNum.Cell measurement unit indicates the quality of the uplink air interface, which serves as a reference for establishing the service model. The following table lists the measurement counters related to the measurement unit Number of Uplink 8PSK_MEAN_BEP Different Value Measurement (Cell). Except for different values of UPLINK_8PSK_MEAN_BEP, the descriptions, measurement points, and formulas of these measurement counters are the same. Therefore, only 8PSK_MEAN_BEP = 0 is taken as an example and described in details.These counters related to the measurement unit are measured only when "ADJUSTULMCSTYPE" in the SET GCELLEGPRSPARA command is set to 2 (the value can be adjusted based on the uplink quality MR reported by the BTS).
