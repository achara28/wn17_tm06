Incoming Inter-RAT Inter-Cell Handover Requests (TDD)

Measurement Counters

ID Counter Description
1278091435 CELL.INTERRAN.TDD.INCELL.HO.REQ CH3601:Incoming Inter-RAT Inter-Cell Handover Requests (TDD)

Description

When a BSC receives an HO REQ message from the MSC for an MS in a neighbor cell from TD-SCDMA, it is referred to as an incoming inter-RAT inter-cell handover (TDD).

This counter provides the number of incoming inter-RAT inter-cell handover requests (TDD), based on the target cell under the control of the target BSC.

Measurement point

As shown in Figure 1, if the BSC receives an HO REQ from the MSC and determines an inter-RAT inter-cell handover (TDD), this counter is measured at point A based on the type of the originating channel.

Figure 1 Measurement of the counters related to incoming inter-RAT inter-cell handover (excluding directed retry) (TDD)

Measurement points Description
A Request for an incoming inter-RAT inter-cell handover (TDD)
B Response to an incoming inter-RAT inter-cell handover (TDD)
C Detection of an incoming inter-RAT inter-cell handover (TDD)
D Completion of an incoming inter-RAT inter-cell handover (TDD)

Formula

Incoming Inter-RAT Inter-Cell Handover Requests (TDD) = Incoming Inter-RAT Inter-Cell Handover Requests (SDCCH) (TDD) + [Incoming Inter-RAT Inter-Cell Handover Requests (TCH) (TDD)]
