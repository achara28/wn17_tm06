1278083486 CELL.INTERRAN.OUTCELL.HO.REQ.LOAD.BASED.HO H350G:Outgoing Inter-RAT Inter-Cell Handover Requests(load based handover)

Description

The load-based handover decision algorithm is used to select the candidate cells for handover based on the cell load. This counter is measured when the BSC sends a Handover Required message to the MSC with the cause value being Traffic, in the GSM-to-UMTS handover procedure. Together with Outgoing Inter-RAT Inter-Cell Handover Requests and Failed Outgoing Inter-RAT Inter-Cell Handovers, this counter indicates the performance of the outgoing inter-RAT inter-cell handover. Together with the measurement counters of Incoming Inter-RAT Inter-Cell Handover Measurement per Cell, this counter indicates the overall conditions of inter-RAT inter-cell handovers.

Measurement point

As shown in Figure 1, this counter is incremented by one at point A when the BSC sends a Handover Required message to the MSC with the cause value being Traffic during the outgoing inter-RAT inter-cell handover procedure.
