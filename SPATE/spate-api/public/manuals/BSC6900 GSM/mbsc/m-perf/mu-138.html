Downlink EGPRS TBF Establish and Release Capability Measurement(PSCALL.DownETbfEstbRel.Cell)

Overview

The measurement unit PSCALL.DownETbfEstbRel.Cell provides the measurement counters related to the downlink EGPRS TBF establishment and release in a granularity period. This measurement unit indicates the PS services load of a cell and is used as a reference for PS services model establishment.
