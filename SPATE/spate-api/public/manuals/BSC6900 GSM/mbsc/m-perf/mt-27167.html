Description

This measurement provides the retransmission rates of downlink EGPRS RLC data blocks in MCS1-MCS9 coding schemes in a cell in a granularity period. If the retransmission rate is high, you can infer that the signal quality is poor, thus the network needs to be optimized.

See measurement points of the counters in the formula.

Formula

Retransmission Rate of Downlink EGPRS MCS1 RLC Data Blocks = ([Total Number of Downlink EGPRS MCS1 RLC Data Blocks] - [Total Number of Valid Downlink EGPRS MCS1 RLC Data Blocks]) x 100 /[Total Number of Downlink EGPRS MCS1 RLC Data Blocks]

The formula of "Retransmission Rate of Downlink EGPRS MCSx RLC Data Blocks " is similar as "[Retransmission Rate of Downlink EGPRS MCS1 RLC Data Blocks]".

Note: MCSx indicates MCS1-MCS9.
