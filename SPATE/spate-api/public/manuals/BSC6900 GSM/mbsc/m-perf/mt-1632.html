Description

If a handover is required according to the handover algorithm and the source cell (current cell) is also the target cell, the BSC assigns a proper channel in the target cell (the target cell is also the source cell) and then sends a CHAN ACTIV message to the BTS of the target cell (the target cell is also the source cell) to activate the channel of the target cell determined by the handover algorithm. Upon receipt of a CHAN ACTIV ACK message, the BSC requests the relevant resource and sends an HO CMD message to the MS through the source cell after the resource is successfully obtained, instructing the MS to initiate a handover from the source cell to the target cell.

The counters provide the numbers of HO CMD commands sent by the BSC through the source cell (current cell) to the MS based on the type of the channel the MS seizes before the handover and the handover reason. These counters can be combined with the Internal Intra-Cell Handover Requests and the Failed Internal Intra-Cell Handovers to reflect the performance of internal intra-cell handovers.

The counter H3010 indicates the number of internal intra-cell handover commands sent by the BSC when the MS seizes an SDCCH before the internal intra-cell handover. The counter H3017 indicates the number of internal intra-cell handover commands sent by the BSC when the MS seizes a TCHF before the internal intra-cell handover. The counter H3018 indicates the number of internal intra-cell handover commands sent by the BSC when the MS seizes a TCHH before the internal intra-cell handover. Each time an internal intra-cell handover command is initiated, the counter H3010, H3017, or H3018 is incremented by one based on the type of the channel the MS seizes before the handover.
  • H3010:CELL.INTRACELL.HO.CMD.SD
  • H3017:CELL.INTRACELL.HO.CMD.TCHF
  • H3018:CELL.INTRACELL.HO.CMD.TCHH
The counter H3012 indicates the number of internal intra-cell handover commands sent by the BSC to the MS when the MS hands over from the underlaid of a concentric cell to the overlaid of the cell. The counter H3011 indicates the number of internal intra-cell handover commands sent by the BSC to the MS when the MS hands over from the overlaid of a concentric cell to the underlaid of the cell. Each time the BSC sends an internal intra-cell handover command to the MS for a handover from the underlaid to the overlaid or from the overlaid to the underlaid, the counter H3012 or H3011 is incremented by one based on the type of the handover.
  • H3011:CELL.INTRACELL.HO.CMD.OLCELL.ULCELL
  • H3012:CELL.INTRACELL.HO.CMD.ULCELL.OLCELL
The counter H301a indicates the number of internal intra-cell handover commands sent by the BSC to the MS for the calls that use the local switching function of the BSC. The counter H301b indicates the number of internal intra-cell handover commands sent by the BSC to the MS for the calls that use the local switching function of the BTS. Each time the BSC sends an internal intra-cell handover command to the MS for a call that uses the local switching function of the BSC or the BTS, the counter H301a or H301b is incremented by one.
  • H301a: CELL.INTRACELL.HO.CMD.BSC.LOOP
  • H301b: CELL.INTRACELL.HO.CMD.BTS.LOOP
The counter H3015A indicates the number of internal intra-cell handover commands sent by the BSC to the MS for a TCHF-TCHH conversion for an AMR call. The counter H3015B indicates the number of internal intra-cell handover commands sent by the BSC to the MS for a TCHH-TCHF conversion for an AMR call. Each time the BSC sends an internal intra-cell handover command to the MS for a TCHF-TCHH or TCHH-TCHF conversion for an AMR call, the counter H3015A or H3015B is incremented by one based on the type of the conversion.
  • H3015A:CELL.INTRACELL.HO.CMD.AMR.TCHF.TCHH
  • H3015B:CELL.INTRACELL.HO.CMD.AMR.TCHH.TCHF

Measurement point

Each time the BSC sends an HO CMD message through the source cell to the MS, one of the counters given above is incremented by one based on the type of the channel the MS seizes before the handover and the handover reason.
