Description

These counters provide the average numbers of idle TCHHs whose interference levels are in each interference band (1-5) in a location group during a measurement period.

Measurement point

Every five seconds, the system accumulates the number of idle TCHHs whose interference levels are in each interference band in one location group. At the end of a measurement period, the accumulated number is divided by sampling times. The result is the average number of idle TCHHs whose interference levels are in each interference band in one location group.
