Downlink GPRS TBF Establish and Release Capability Measurement(PSCALL.DownGTbfEstbRel.Cell)

Overview

The measurement unit PSCALL.DownGTbfEstbRel.Cell provides the measurement counters related to downlink GPRS TBF establishment and release in a granularity period. This measurement unit indicates the PS services load of a cell and is used as a reference for PS services model establishment.
