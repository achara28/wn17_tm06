Number of Outgoing External Inter-Cell Handover Successes During BSC Local Switch

Measurement Counters

ID Counter Description
1278081572 CELL.INTERBSC.OUTCELL.HO.SUCC.BSC.LOOP CH333a:Number of Outgoing External Inter-Cell Handover Successes During BSC Local Switch

Description

This counter provides the total number of outgoing external inter-cell handover successes during BSC local switching. The counter is calculated by subtracting the Number of Outgoing External Inter-Cell Handover Failures During BSC local switching from the Number of Outgoing External Inter-Cell Handover Requests During BSC local switching.

Measurement point

As shown in Figure 1 and Figure 2, the counter is measured at C when the BSC receives the CLEAR CMD message from the MSC during the outgoing external inter-cell handover and the cause is Handover successful (0x0B).

Figure 1 Measurement of the counters related to outgoing external inter-cell handover (excluding directed retry)

Measurement points Description
A Measurement point of outgoing external inter-cell handover requests (excluding directed retry)
B Measurement point of outgoing external inter-cell handover responses (excluding directed retry)
C Measurement point of successful outgoing external inter-cell handovers (excluding directed retry)

Formula

Number of Outgoing External Inter-Cell Handover Successes During BSC Local Switch = [Number of Outgoing External Inter-Cell Handover Requests During BSC Local Switch] - [Number of Outgoing External Inter-Cell Handover Failures During BSC Local Switch]

None
