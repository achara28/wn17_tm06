Description

This counter provides the total number of valid downlink EGPRS RLC data blocks using MCS1 to MCS9 coding schemes in a cell in a measurement period. A high-rate coding scheme indicates large data volume transmitted on each data block. A large proportion of valid downlink RLC data blocks indicates fewer retransmissions of downlink RLC data blocks.

Measurement point

The measurement is triggered at point A in figure 1 (the process that the BSC sends a downlink EGPRS RLC data block using the MCS1 coding scheme and receives an EGPRS PACKET DOWNLINK ACK/NACK message). After receiving the message from the MS, the BSC determines whether the MS receives the downlink EGPRS RLC data block based on the bitmap in the EGPRS PACKET DOWNLINK ACK/NACK message. This counter is incremented by one the first time the BSC confirms that the MS receives the data block.

Figure 1 Transmission of downlink EGPRS RLC data blocks using MCS-1 coding scheme

The measurement point of the number of downlink EGPRS RLC data blocks using MCS-2 to MCS-9 coding schemes is the same as that of the number of downlink EGPRS RLC data blocks using MCS-1 coding scheme. The measurement is triggered after the downlink data block using the corresponding coding scheme is received and acknowledged.
