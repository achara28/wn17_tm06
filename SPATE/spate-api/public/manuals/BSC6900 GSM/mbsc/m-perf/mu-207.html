MP Queue Measurement(MLPPP.QUEUE)

Overview

MP groups are important resources of the Base Station Controller. The transmission quality of MP groups affects the availability of system functions. Packets carried on MP groups are mapped onto four or eight queues based on their Differentiated Service Code Point(DSCP). The MHF parameter determines the number of queues. A MP group can transmit the services of different types. That is, the services of different types can be transmitted through different queues according to certain rules, and the MP group schedules the data in different queues according to priorities. You can run the following commands to query the correspondence between service types and queues:

1. You can run the LST TRMMAP command to query the correspondence between service types and the Per-Hop Behavior(PHB).

2. You can run the LST PHBMAP command to query the correspondence between PHB and DSCP.

3. You can run the LST PHBMAP command to query the correspondence between DSCP and queues.

MP group queues belong to the data link layer.
