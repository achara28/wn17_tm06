Call Drops on TCH (Signaling Channel)

Measurement Counters

ID Counter Description
1278087428 CELL.KPI.TCH.DROPS.SIG K3022:Call Drops on TCH (Signaling Channel)

Description

In the very early TCH assignment procedure, this measurement provides the number of call drops due to any cause on the established TCH carrying signaling.

Measurement point

In the very early TCH assignment procedure, if the TCH is used to carry signaling, this counter is measured in the cell where a call drop occurs at A1, A2, A3, or A4 in Figure 1. For details, see the measurement point of the counters in the formula.

Figure 1 Call drops in the immediate assignment (including very early assignment) procedure
