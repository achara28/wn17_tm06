Sampled Measurement of Initially Configured Channels (Dynamic PDCH)

Measurement Counters

ID Counter Description
1278469425 CELL.CH.CFG.NUM.DYN.PDCH.900 R3002A:Number of Initially Configured Channels (Dynamic PDCH) (900/850/810 band)
1278469426 CELL.CH.CFG.NUM.DYN.PDCH.1800 R3002B:Number of Initially Configured Channels (Dynamic PDCH) (1800/1900 Band)

Description

R3002A:CELL.CH.CFG.NUM.DYN.PDCH.900 provides the number of configured dynamic PDCHs in the GSM900, GSM900/DCS1800, GSM850, GSM850/DCS1800, GSM850/PCS1900 and TGSM810 cells controlled by the BSC at each sampling point. This number includes the number of dynamic PDCHs that are dynamically converted into TCHFs.

R3002B:CELL.CH.CFG.NUM.DYN.PDCH.1800 provides the number of configured dynamic PDCHs on the DCS1800 and PCS1900 frequency bands in DCS1800, PCS1900, GSM900&DCS1800, GSM850&DCS1800, GSM850&PCS1900 cells controlled by the BSC at each sampling point. The value of this counter includes the number of dynamic PDCHs that have been dynamically converted into TCHFs.

Measurement point

The BSC takes the samples of the channel allocation status every five seconds. When the sampling timer expires, R3002A:CELL.CH.CFG.NUM.DYN.PDCH.900 and R3002B:CELL.CH.CFG.NUM.DYN.PDCH.1800 are measured according to the frequency band of the cell and the number of configured dynamic PDCHs in the cell.At the end of the granularity period, the value reported within the last five seconds is used as the value of the counter. The BSC does not measure this type of counter if dynamic PDCH cannot be configured because the BSC is configured with a built-in PCU.
