Number of SGSN Selections When Valid Subscriber Identities Are Unavailable

Measurement Counters

ID Counter Description
1282429912 CELL.USR.NUM.UNKNOWN.PS R9753:Number of SGSN Selections When Valid Subscriber Identities Are Unavailable

Description

This measurement provides, in an MOCN-enabled cell, the number of subscribers that select SGSN based on IMSI in the case of the routing area update or attachment request when the subscribers do not select SGSN according to the IMSI or NRI. In an MOCN-enabled cell, the subscriber obtains SGSN through load sharing in sequence if the routing area update or attachment request message sent by the subscriber does not contain the IMSI or a valid NRI assigned by the operator.

This counter provides the number of subscribers that select SGSN in an MOCN-enabled cell if the selection is not based on a valid subscriber.

Measurement point

Upon receiving the routing area update acceptance or attachment acceptance message sent by the SGSN, the BSC calculates the counter based on cell and SGSN if the subscriber obtains the SGSN through load sharing in sequence.

Formula

None
