Counters Related to the Number of Rejects to Immediate Assignments (Packet Service)

Measurement Counters

ID Counter Description
1278075429 CELL.IMM.ASS.REJ.PS A302H:Rejects to Immediate Assignments (Packet Service)

Description

This counter provides the number of times that the BSC sends the IMM ASS REJ message to the MS if the access cause is PS services in a measurement period. During the immediate assignment, the BSC sends an IMM ASS REJ message to the MS if no channel is available after receiving a CHAN RQD message for PS services from the BTS. This indicates that the immediate assignment fails.

Measurement point

This counter is incremented by one when the BSC sends an IMM ASS REJ message to the MS and the access cause is PS services in a measurement period. This counter is incremented by one at point A, as shown in Figure 1.
