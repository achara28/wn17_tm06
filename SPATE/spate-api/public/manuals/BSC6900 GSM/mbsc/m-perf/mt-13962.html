Measurement point

The measurement is triggered at point A in figure 1 (the process that the BSC sends a downlink EGPRS RLC data block using MCS1 coding scheme and receives an EGPRS PACKET DOWNLINK ACK/NACK message). This counter is incremented by one for the subsite in which the MS locates each time the BSC receives the EGPRS PACKET DOWNLINK ACK/NACK message from the MS and determines that the MS receives the corresponding downlink EGPRS RLC data block using MCS1 coding scheme based on the bitmap in the message for the first time.

Figure 1 Transmission of downlink EGPRS RLC data blocks using MCS1
