Description

The counters indicate the performance of the outgoing inter-cell handover in a specified cell, for example, the number of requests for the handover from the specified cell to another cell, the number of successful handovers, and the success rate of handovers.

It is recommended that the counters related to Analyzed Measurement of Outgoing Cell Handovers are registered at the initial stage of BSC running or during the analysis of handover performance in a specified cell for network optimization. When the BSC system is running stably, these counters are deleted. Along with the counters related to Analyzed Measurement of Incoming Cell Handovers, these counters indicate the performance of the handover between two specified cells. The recommended period of measuring these counters is 60 minutes.
