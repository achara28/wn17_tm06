Description

This measurement provides the retransmission rate of downlink EGPRS RLC data blocks in DAS5-DAS12 coding schemes in a cell in a granularity period. If the retransmission rate is high, it indicates that the signal quality is poor. In this case, the network needs to be optimized.

See measurement points of the counters in the formula.

Formula

Retransmission Rate of Downlink EGPRS DAS5 RLC Data Blocks = ([Total Number of Downlink EGPRS DAS5 RLC Data Blocks] - [Total Number of Valid Downlink EGPRS DAS5 RLC Data Blocks]) x 100 /[Total Number of Downlink EGPRS DAS5 RLC Data Blocks]

The formula of "Retransmission Rate of Downlink EGPRS DASx RLC Data Blocks" is similar to "[Retransmission Rate of Downlink EGPRS DAS5 RLC Data Blocks]".

Note: DASx indicates DAS6-DAS12.
