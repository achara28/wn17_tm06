Call Drops due to Error Indications in Stable BTS local switching State

Measurement Counters

ID Counter Description
1278089461 CELL.CALL.DROPS.ERR.IND.BTS.LOOP M3800:Call Drops due to Error Indications in Stable BTS local switching State

Description

After seizing a channel, the MS is in the BTS local switching state. The BTS sends the BSC the ERR IND message due to the abnormal connection at the radio link layer, indicating that call drops occur on the Um interface because of the faulty radio links. This counter provides the number of call drops when the cause value carried in the ERR IND message is timer T200 expired (N200+1) times (0x01), unsolicited DM response, multiple frame established state: perform abnormal release(0x05), or sequence error (0x07).

Measurement point

If the BSC receives an ERR IND message, the BSC regards it as a call drop. The release procedure is started. If the MS is detected in the BTS local switch state, this counter is measured.
