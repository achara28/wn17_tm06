Number of Successful Uplink DTM TBF Establishments

Measurement Counters

ID Counter Description
1282433001 DTM.UP.TBF.ESTB.SUCC A9814:Number of Successful Uplink DTM TBF Establishments

Description

This counter provides the number of successful uplink TBF establishments for DTM MSs in a measurement period.

Measurement point

1.The measurement is triggered at point A in figure 1 (the process of establishing an uplink TBF for DTM MSs). Upon receiving the DTM REQUEST message from an MS on the DCCH, the BSC sends a DTM ASSIGNMENT or a PACKET ASSIGNMENT message to the MS. This counter is incremented by one when the uplink TBF for DTM MSs receives uplink data blocks of MSs on the allocated channel and the uplink TBF establishment for a DTM MS is successful.

Figure 1 Process of establishing an uplink TBF for DTM MSs on the DCCH

2.The DTM MS successfully establishes an uplink TBF on the PACCH (uplink TBF establishment in the downlink TBF). After the BSC sends a PACKET UPLINK ASSIGNMENT message to the MS and receives uplink data blocks of the MS, this counter is incremented by one when the uplink TBF establishment for a DTM MS on the PACCH is successful.
