Measurement of IP Layer on User Plane of Transmission Resource Pool Adjacent Node(IPPOOL.ADJNODE.IPLAYER)

Overview

Measurement of IP Layer on User Plane of Transmission Resource Pool Adjacent Node indicates the user-plane traffic of each adjacent node that communicates with the IP addresses in a transmission resource pool. This measurement helps analyze the transmission quality of the system.
