Description

This counter provides the number of EGPRS PACKET DOWNLINK ACK/NACK massages (sent by the MS to the BSC) whose EGPRS BEP Link Quality Measurements information contains a GMSK_MEAN_BEP field of value N. This counter reflects the downlink quality over the Um interface in GMSK modulation mode. The greater the value of N, the better the Um interface quality. The value range for N is from 0 to 31.

Measurement point

The measurement is triggered at point A in figure 1 (the process that the MS sends the BSC an EGPRS PACKET DOWNLINK ACK/NACK message, and the EGPRS PACKET DOWNLINK ACK/NACK message transmitted in GMSK modulation mode contains the GMSK_MEAN_BEP field to indicate the radio measurement report reported by the MS). This counter is incremented by one each time the BSC receives the EGPRS PACKET DOWNLINK ACK/NACK message and the GMSK_MEAN_BEP field of the message is of value N, as shown in point A. The value range for N is from 0 to 31.
