Number of transmitted BLOCK ACK Messages on NSVC

1279469429 NSVC.TRAN.TX.BLOCK.ACK.MSG L9713:Number of transmitted BLOCK ACK Messages on NSVC

Description

This measurement provides the number of BLOCK ACK messages sent from the BSC to the SGSN in a granularity period. When the SGSN initiates the procedure for blocking the NSVC, the BSC sends an NSVC BLOCK ACK message to the SGSN after receiving an NSVC BLOCK message. The value of the counter should be consistent with the number of received BLOCK messages. If not, check the link status and related data configuration of the Gb interface.

Measurement point

In a granularity period, the counter is incremented by one every time the BSC sends an NSVC BLOCK ACK message on an NSVC to the SGSN (see measurement point B in Figure 1).
