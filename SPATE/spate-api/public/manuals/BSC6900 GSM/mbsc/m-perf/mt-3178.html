Maximum Number of EGPRS Downlink Timeslots

Measurement Counters

ID Counter Description
1279188427 EGPRS.TBF.DOWNLINK.MAX.TS.NUM S9311:Maximum Number of EGPRS Downlink Timeslots

Description

This measurement provides the maximum number of timeslots that can be allocated to the EGPRS downlink TBFs in transmission state in a granularity period, based on the service type and multislot capability of the MS. This counter is used to calculate Average Maximum Number of Downlink TBF Timeslots.

Measurement point

In a granularity period, the BSC periodically samples all the downlink EGPRS TBFs in transmission state. Based on the multislot capability and service type of the MS, the BSC calculates the maximum number of timeslots that can be allocated to the TBFs during each sampling. The calculated value is accumulated to Maximum Number of EGPRS Downlink Timeslots. The sampling period is 5s.
