Number of Times TRXs Preempt Baseband Resources

Measurement Counters

ID Counter Description
1282432954 CELL.TRX.PREEMPT.TIMES R3613D:Number of Times TRXs Preempt Baseband Resources

Description

This counter provides the number of times that baseband resources for TRXs in a cell are preempted in a measurement period after baseband resource pool is enabled.

Measurement point

The measurement is triggered when baseband resources for TRXs in a cell are preempted in a measurement period.

None
