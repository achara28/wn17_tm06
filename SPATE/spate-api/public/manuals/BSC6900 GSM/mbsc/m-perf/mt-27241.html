Retransmission Rate of Uplink GPRS RLC Data Blocks

Measurement Counters

ID Counter Description
1279177438 UP.RLC.GPRS.BLK.RETRAN.RATE RL9022:Retransmission Rate of Uplink GPRS RLC Data Blocks

Description

This measurement provides the retransmission rate of uplink GPRS RLC data blocks in a cell in a granularity period. This measurement indicates the quality of the uplink GPRS RLC data transmission in a granularity period. If the quality of the air interface or the data transmission is poor, the uplink data packets may be lost or faulty, which causes the MS to retransmit the packets. Therefore, this measurement indicates the quality of the air interface or data transmission to some extent.

Measurement point

In a granularity period, the number of retransmission data blocks are equal to the total number of uplink GPRS RLC data blocks minus the number of valid uplink RLC data blocks. Then, calculate the rate of the number of retransmission data blocks to the total number of uplink GPRS RLC data blocks to obtain the Retransmission Rate of Uplink GPRS RLC Data Blocks.

Formula

Retransmission Rate of Uplink GPRS RLC Data Blocks = (Total Number of Uplink GPRS RLC Data Blocks - Number of Valid Uplink RLC Data Blocks Using CS1 - Number of Valid Uplink RLC Data Blocks Using CS2 - Number of Valid Uplink RLC Data Blocks Using CS3 - Number of Valid Uplink RLC Data Blocks Using CS4) x 100/Total Number of Uplink GPRS RLC Data Blocks

Unit

%
