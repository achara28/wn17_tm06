RLC Data Transmission Performance Measurement for Cell-level Subsite(PSCALL.RlcDataTran.CELL_LOCGRP)

Overview

This measurement unit provides the measurement counters related to RLC data transmission in a subsite in a measurement period. This measurement unit indicates the PS load of a subsite and is used as a reference for setting up the PS traffic model.
