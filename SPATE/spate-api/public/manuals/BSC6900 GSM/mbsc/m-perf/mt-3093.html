Number of Uplink GPRS TBF Abnormal Releases due to FLUSH

Measurement Counters

ID Counter Description
1279173425 UP.GPRS.TBF.ABNORM.REL.FLUSH A9009:Number of Uplink GPRS TBF Abnormal Releases due to FLUSH

Description

This measurement provides the number of uplink GPRS TBF abnormal releases due to cell flush in a granularity period. A high measurement value is yielded because the cell reselection procedure is frequently initiated by the MS. In this case, the planning of radio network environment is required.

Measurement point

When the MS reselects a new cell, the SGSN initiates the cell updating procedure and sends a FLUSH LL PDU to the BSC.

Accordingly, the BSC suspends all the PS services of the MS in the original cell irrespective of the MS status. When the MS is in transmission state and the uplink GPRS TBF exists, the BSC releases the uplink GPRS TBF if the BSC receives the FLUSH LL PDU from the SGSN.

Figure 1 shows the procedure in which the BSC abnormally releases the uplink GPRS TBF upon receipt of an FLUSH LL PDU. Each time the BSC receives the FLUSH LL PDU message (see measurement point A), the counter is incremented by one.
