Number of Successful TRX Allocations

Measurement Counters

ID Counter Description
1282433153 CELL.TRX.LICENSE.ALLOC.SUC.TIMES R3139D:Number of Successful TRX Allocations

Description

This counter provides the number of successful TRX license allocations to a cell after the intra-BBU carrier pool function is enabled. If this counter is incremented by one, the Number of TRX Releases counter is also incremented by one in a cell where the TRX license is released.

Measurement point

This counter is measured once when the BSC allocates a TRX license to the cell in a measurement period.

None
