Description

These counters provide the number of preparation attempts for PS outgoing inter-RAT handover in the best cell.

Measurement point

The measurement is triggered at point B as shown in the figure, when the RNC sends to the UE a CELL CHANGE ORDER FROM UTRAN message during the PS outgoing inter-RAT handover. In this case, the corresponding counters are measured in the best cell that the UE camps on.
