Number of Positioning Reports Meeting Accuracy Requirement

Measurement Counters

ID Counter Description
67181025 VS.LCS.QoSAccurMet Number of Location Reports with Position Estimate Satisfying the Requested Accuracy for RNC

Description

This measurement item takes statistics of the number of LOCATION REPORT messages from the RNC to the CN which meet the positioning accuracy requirement.

Measurement point

The measurement is triggered at point A as shown in figure. When the RNC sends the CN a LOCATION REPORT message, and if the Accuracy Code is smaller than or equal to that in the LOCATION REPORTING CONTROL message from the CN, it indicates that the LOCATION REPORT message meets the positioning accuracy requirement. In this case, the RNC measures this item.
