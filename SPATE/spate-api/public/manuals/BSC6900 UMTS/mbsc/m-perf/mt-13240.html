Number of RABs Abnormally Released During PTT Calls for Cell

Measurement Counters

ID Counter Description
73423560 VS.PTT.RAB.AbnormRel.DCH Number of RABs Abnormally Released During PTT Calls for Cell

Description

This counter provides the number of RABs abnormally released during PTT calls in a cell.

The counter is used with the counter "Number of Complete PTT Calls for Cell" to analyze the call drop rate of PTT services.

Measurement point

At point A shown in the following figure, when a failure occurs, the RNC sends a RAB RELEASE REQUEST message to the CN. After the RNC receives from the CN a RAB ASSIGNMENT REQUEST message containing a cause value other than normal release values (such as "User Inactivity", "Normal Release", "Successful Relocation", and "Network Optimization", for details, see 3GPP TS 25.413), the RNC sends a RAB ASSIGNMENT RESPONSE message. When this happens, this counter is measured in the best cell that the UE camps on if the UE transfers to the current CELL_DCH state due to a PTT call.
