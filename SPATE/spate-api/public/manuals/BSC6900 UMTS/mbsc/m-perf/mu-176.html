IMA Link Measurement(IMALNK)

Overview

The quality of transmission links directly affects the system performance. IMA link measurement helps analyze system transmission quality so as to maintain system stability and reliability.

Figure 1 ATM Reference Model

IMA Link belongs to the physical layer of the ATM reference model, as figure 1 shows.

Figure 2 Physical bearer of the IMA Link

In the configuration structure, IMA links are between physical ports and logical applications, as shown in the figure 2.
