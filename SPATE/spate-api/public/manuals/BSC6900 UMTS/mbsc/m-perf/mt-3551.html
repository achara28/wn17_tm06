Number of Times That a Fault Occurs in an IMA Link

Measurement Counters

ID Counter Description
67194042 VS.IMALNK.FAULTEMIT Number of Times that a Fault Occurs in an IMA Link

Description

This counter provides the number of times that a fault occurs in an IMA Link in the specified measurement period.

Measurement point

Samples are taken every five seconds in the specified measurement period. If the system is in faulty state at the current sampling point but in normal state at the previous sampling point, the number of faults is added by one.

Formula

None

Unit

times
