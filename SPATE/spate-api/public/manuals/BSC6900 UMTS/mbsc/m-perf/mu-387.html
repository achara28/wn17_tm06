Measurement related to Paging per cell(PAGE.CELL)

Overview

This measurement unit mainly measures the number of paging losses due to congestion and the number of paging times in a cell.

If the paged UE is in idle mode, the CELL_PCH state, or URA_PCH state, the RNC sends a PAGING TYPE 1 message on the Paging Control Channel (PCCH) to the cells in the paging area. If the UE is in the CELL_PCH state and both the network and the UE support EPCH, the RNC sends a PAGING TYPE 2 message to the cells in the paging area.
