Forward Jitter of Transmission Resource Pool IPPM

Measurement Counters

ID Counter Description
73424341 VS.IPPOOL.IPPM.Forward.JitterMax T7825:Maximum Forward Jitter of Transmission Resource Pool IPPM
73441262 VS.IPPOOL.IPPM.Forward.JitterStandardDeviation T7826:Forward Jitter Standard Deviation of Transmission Resource Pool IPPM

Description

This counter provides the jitter standard deviation of IPPM data packets in a measurement period. It indicates the stabilization of the transmission delay of the Transmission Resource Pool. A large counter value indicates that the transmission delay varies significantly. The Jitter Standard Deviation is calculated as follows:

Jitter Standard Deviation = (((Jitter 1 - Mean Jitter)^2+(Jitter 2 - Mean Jitter)^2+......(Jitter n - Mean Jitter)^2)/n)^0.5

Measurement point

The system obtains the delay jitter of every packet and calculates the standard deviation of the IPPM data packet delay jitter based on the collected statistics within one measurement period.

Formula

None

Unit

microsecond
