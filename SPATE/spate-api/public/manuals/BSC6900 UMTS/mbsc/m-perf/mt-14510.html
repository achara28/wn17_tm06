Number of Successful Hard Handovers of PTT Service

Measurement Counters

ID Counter Description
73424830 VS.PTT.HHO.SuccOut Number of Successful Hard Handovers of PTT Service for Cell

Description

This counter measures the number of successful hard handovers (including intra-frequency handovers, inter-frequency handovers, and handovers under the same RNC or between RNCs) for the PTT service in a cell.

Measurement point

During a hard handover, the RNC may send any of the following messages to a UE:

[1]PHYSICAL CHANNEL RECONFIGURATION,

[2]TRANSPORT CHANNEL RECONFIGURATION,

[3]RADIO BEARER RECONFIGURATION,

[4]RADIO BEARER SETUP,

[5]RADIO BEARER RELEASE.

The UE may send one of the following messages to the RNC accordingly:

[1]PHYSICAL CHANNEL RECONFIGURATION COMPLETE,

[2]PHYSICAL CHANNEL RECONFIGURATION FAILURE,

[3]TRANSPORT CHANNEL RECONFIGURATION COMPLETE,

[4]TRANSPORT CHANNEL RECONFIGURATION FAILURE,

[5]RADIO BEARER RECONFIGURATION COMPLETE,

[6]RADIO BEARER RECONFIGURATION FAILURE,

[7]RADIO BEARER SETUP COMPLETE,

[8]RADIO BEARER SETUP FAILURE,

[9]RADIO BEARER RELEASE COMPLETE,

[10]RADIO BEARER RELEASE FAILURE.

Use the RADIO BEARER RECONFIGURATION message as an example. The measurement is triggered at point B shown in the figure when both of the following conditions are met: [1]The RNC sends a RADIO BEARER RECONFIGURATION message to a UE during a hard handover; [2]The RNC receives a RADIO BEARER RECONFIGURATION COMPLETE message from this UE. In this case, this counter is increased by one in the best cell of this UE before the hard handover.
