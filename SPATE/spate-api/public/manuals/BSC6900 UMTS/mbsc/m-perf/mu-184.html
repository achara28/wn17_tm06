Measurement Related to IPPM(IPPATH.PM)

Overview

The IPPATH Performance Monitor (PM) uses Forward Monitoring (FM) and Backward Reporting (BR) to check for packet loss over channels. For measurement on the BSC side, the BSC periodically sends FM messages to indicate the number of outgoing packets, the number of bytes, and sending time. The Base Station responds with BR messages after receiving the FM message to report the number of received packets, the number of received bytes, the receiving time of PM message, the sending time of BR response. The BSC calculates the packet loss rate, transmission delay and jitter according to the BR response returned from the Base Station. The numbers of packets and bytes listed in the FM and BR frames are the number of data packets and the number of bytes included in the data packets, exclusive of the number of FM and BR frames and the bytes included in the FM and BR frames.
