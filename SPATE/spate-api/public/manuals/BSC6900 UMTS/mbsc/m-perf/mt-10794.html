Description

These counters provide the number of RRC CONNECTION REQUEST messages sent from the UE to the RNC that are actually processed by the RNC according to various reasons for RRC connection requests. These counters exclude the number of rejected RRC connection requests caused by service-based redirection.

Measurement point

The measurement is triggered at point A as shown in the figure, when the RNC receives a RRC CONNECTION REQUEST message from the UE. In this case, these counters are measured separately according to the reasons for RRC connection requests in the cell where the UE is located. These counters exclude the number of rejected RRC connection requests caused by service-based redirection. For more details on the reason types, see 3GPP TS 25.331. The measurement is not triggered for a RRC CONNECTION REQUEST message that is retransmitted by the UE.
