RRC Reporting Measurement per cell(MEAR.Report.Cell)

Overview

The RRC measurement reports include the following events: Events 1A, 1B, 1C, 1D related to intra-frequency measurement. Events 2D and 2F related to active set measurement. Events 4A and 4B related to traffic measurement.
