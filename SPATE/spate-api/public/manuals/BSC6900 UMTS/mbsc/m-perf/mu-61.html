Fractional ATM Link Measurement(FRACATMLNK)

Overview

The quality of transmission links directly affects the system performance. Fractional ATM link measurement helps analyze system transmission quality so as to maintain system stability and reliability.

Figure 1 ATM Reference Model

Fractional ATM link belongs to the physical layer of the ATM reference model, as figure 1 shows.

Figure 2 Physical bearer of the Fractional ATM

In the configuration structure, fractional ATM links are between physical ports and logical applications, as shown in the figure 2.
