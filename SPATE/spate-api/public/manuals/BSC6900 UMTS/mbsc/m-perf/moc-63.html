Measurement of MPU Performance(MPU)

Overview

The Main Processing Unit (MPU) measurement is used to assess the performance of the MPU subsystem of a main control SPU board in a Main Processing Subrack (MPS) or Extended Processing Subrack (EPS) of the RNC. SPU boards can be divided into two types: main control SPU boards and non-main control SPU boards. In addition to processing UMTS control-plane services, main control SPU boards also manage the user-plane, control-plane, and transmission resources of the UMTS. Non-main control SPU boards only process UMTS control-plane services.

Subsystem 0 of the main control SPU board is the Main Processing Unit (MPU) subsystem. It manages the user-plane, control-plane, and transmission resources of the UMTS.

Other subsystems of the main control SPU board and all subsystems of non-main control SPU boards are CPUS subsystems. They process control-plane services.

Different types of SPU boards have different number of subsystems. For example, each SPUa board has four subsystems, and each SPUb board has eight subsystems. In addition, an SPUa subsystem and an SPUb subsystem have different processing capabilities.

The MPU measurement is performed only to assess the performance of a single MPU subsystem.
