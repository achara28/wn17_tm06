E1T1 Port Bit Error Measurement(E1T1_ES)

Overview

E1/T1 is a major resource for the base station controller. It indicates E1T1 electrical ports or channelized optical ports. E1T1 Port Bit Error Measurement is used to measure the E1T1 errored seconds and BER. In this way, the transmission quality can be monitored and the stable operation of the system is ensured. ITU G.826 defines the following monitoring items(a1,a2,d1,d2,d3) and HUAWEI defines the following monitoring(a3,d4,d5).Based on the items and their measurement rules, the E1T1 errored seconds and BER can be obtained.

a1: an errored frame alignment signal.a2: an EB as indicated by an EDC.a3: E-Bit error, indicating that the CRC for the transmitted frames fails.d1: loss of signal.d2: alarm indication signal.d3: loss of frame alignment.d4: frame slip, indicating that frame slips occur in the receiving direction.d5: remote alarm indication, indicating that the peer end cannot receive frames properly.
