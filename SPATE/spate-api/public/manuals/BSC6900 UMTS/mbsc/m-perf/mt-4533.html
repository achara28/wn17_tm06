Number of the times that the location in SAS centric mode is used in the RNC

Measurement Counters

ID Counter Description
73394012 VS.LCS.SasMode.Att Number of the positioning attempt of using SAS centric mode for RNC
73394013 VS.LCS.SasMode.Succ Number of the successful positioning of using SAS centric mode for RNC

Description

This counter provides the number of times that the location in SAS centric mode is used in the RNC.

Measurement point

SAS is an abbreviation for Stand-Alone SMLC. It provides auxiliary data for location and performs related calculation. The interface between the RNC and the SAS is the Iu-PS interface. This counter helps assessing the success rate of SAS centric location on the Iu-PC interface. The measurement of VS.LCS.SasMode.Att is triggered at point A as shown in the figure, when the RNC sends the SAS an IUPC POSITION INITIATION REQUEST message. The measurement of VS.LCS.SasMode.Succ is triggered at point B, when the RNC receives from the SAS a POSITION INITIATION RESPONSE message that contains the location result.
