Description

These counters provide the number of times that the RAB is modified successfully for PS services in the RNC based on the service type.

Measurement point

The measurement is triggered at point B, as shown in the figure. When the RNC sends to the CN a RAB ASSIGNMENT RESPONSE message after receiving from the UE a RADIO BEARER RECONFIGURATION COMPLETE message, it measures this counter based on the service type if the RAB is for PS services.
