Description

These counters provide the number of RRC CONNECTION SETUP COMPLETE messages from the UE to the RNC in the cell according to various reasons of RRC connection requests.

Measurement point

The measurement is triggered at point C as shown in the figure, when the RNC receives a RRC CONNECTION SETUP COMPLETE message from the UE. In this case, these counters are measured separately according to the reasons for RRC connection requests in the cell where the UE is located. For details on the reason types, see section 10.3.3.11 in 3GPP TS 25.331.
