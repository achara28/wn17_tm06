ERACH Measurement per cell(ERACH.Cell)

Overview

In ERACH mode, UEs in CELL_FACH state use the E-DCH as the uplink transport channel and use the HS-DSCH as the downlink transport channel.

The ERACH feature is introduced in R8. The E-DCH carries message data, replacing the traditional RACH for a higher uplink peak rate. In this way, delay of UEs in the IDLE mode or in the CELL_FACH, CELL_PCH, or URA_PCH state on the user plane and control plane is reduced and delay in transition from the CELL_FACH, CELL_PCH, or URA_PCH state to the CELL_DCH state is reduced.
