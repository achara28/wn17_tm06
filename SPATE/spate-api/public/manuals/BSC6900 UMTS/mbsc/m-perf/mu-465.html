Transmission Resource Pool IPPM Performance Measurement(IPPOOL.PM)

Overview

The Transmission Resource Pool IPPM Performance Measurement uses Forward Monitoring (FM) and Backward Reporting (BR) to monitor transmission quality. The BSC support both forward measurement and backward mesurement. The forward direction indicates the sending direction, and the backward direction indicates the receiving direction. For forward measurement, the BSC periodically sends FM messages to indicate the number of outgoing packets, the number of bytes, and sending time. The Base Station responds with BR messages after receiving the FM message to report the number of received packets, the number of received bytes, the receiving time of PM message, the sending time of BR response. For backward measurement, the Base Station periodically sends FM messages and the BSC responds with BR messages. The BSC calculates the packet loss rate, transmission delay and jitter according to the FM and BR messages. The numbers of packets and bytes listed in the FM and BR frames are the number of data packets and the number of bytes included in the data packets, exclusive of the number of FM and BR frames and the bytes included in the FM and BR frames.
