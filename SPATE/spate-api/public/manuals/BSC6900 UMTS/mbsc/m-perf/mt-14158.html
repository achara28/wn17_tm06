Number of Successful TRNC Relocations in the CS Domain and PS Domain (UE Involved)

Measurement Counters

ID Counter Description
73424478 VS.TRELOC.SuccExecUEInvolCS Number of Successful CS Relocations for TRNC (UE Involved)
73424479 VS.TRELOC.SuccExecUEInvolPS Number of Successful PS Relocations for TRNC (UE Involved)

Description

These counters measure the number of successful target RNC (TRNC) relocations.

Measurement point

The measurement is triggered at point B shown in figure 1 when the TRNC sends a RELOCATION COMPLETE message to the CN upon completing a relocation. In this case, these counters are measured based on the service type.
