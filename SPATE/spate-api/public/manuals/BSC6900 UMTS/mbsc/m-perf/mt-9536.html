Description

These counters provide the number of CELL UPDATE messages received by the RNC according to different cause values.

Measurement point

The measurement is triggered at point A as shown in the figure. When the RNC receives a CELL UPDATE message from the UE and the UE is not involved in a RRC connection release procedure, the RNC measures the items based on different cell update causes in the cell that the UE chooses during the cell update. The table describes these measurement items.This counter is not measured for a CELL UPDATE message retransmitted by the UE in the same cell.
