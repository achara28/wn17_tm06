Description

These counters provide the average number of RBs for PS R99 streaming services with different UL and DL rates in the cells in the active set.

Measurement point

The RNC takes a sample every 5 seconds from the number of PS streaming service RBs at different bit rates. At the end of the measurement period, the RNC divides the accumulated number of streaming service RBs at each bit rate by the number of samples to obtain the mean number of PS streaming service RBs. In this case, the RNC measures the item in each cell that the UE camps on.
