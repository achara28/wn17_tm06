Measurement related to RLC per cell(RLC.Cell)

Overview

Radio Link Control (RLC) measurement in a cell is to measure the number of RLC PDUs sent, retransmitted and discarded in a cell. These RLC PDUs include traffic RLC PDUs and signaling RLC PDUs.
