67196201 VS.RRC.SuccConnEstab.HSDSCH Number of Successful RRC Connection Setups for Cell (DL Carried on HS_DSCH)

Description

These counters provide the number of RRC CONNECTION SETUP COMPLETE messages sent by the UE to the RNC in the cell. Among them, the counter of VS.RRC.SuccConnEstab.HSDSCH includes both the case of SRB over HSDPA and the case of EFACH.

Measurement point

The measurement is triggered at point C as shown in the figure, when the RNC receives a RRC CONNECTION SETUP COMPLETE message from the UE. In this case, these counters are measured in the cell where the UE is located.
