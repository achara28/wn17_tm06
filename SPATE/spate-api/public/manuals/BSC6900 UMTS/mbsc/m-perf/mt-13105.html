Sum of Sampled Values of Downlink BE Service Throughputs for All DCH PS UEs for a Best Cell

Measurement Counters

ID Counter Description
73423425 VS.Reserve.Counter6 Reserved counter 6 for cell

Description

This counter provides the sum of sampled values of the downlink BE service throughputs for all DCH PS UEs within a best cell, in the unit of kbit/s.

Measurement point

The RNC periodically samples the downlink throughputs for BE services for all DCH PS UEs within the sampling period of 500 ms. In the statistical period, the sum of sampled values of the downlink BE service throughputs for all DCH PS UEs in a best cell is obtained by accumulating the throughputs whose sampled values are not 0. RLC header data and retransmission data are excluded during calculation of the sampled downlink throughput values. When some UEs initiate multiple BE services, the RNC uses only one RAB for each UE to measure this counter.
