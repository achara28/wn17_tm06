Description

These counters provide the number of AMR WB users with different UL and DL rates in the cells in the active set.

Measurement point

The number of AMR WB users with each UL or DL rate is sampled every 30 seconds in the cells in the active set. At the end of the measurement period, the value of each counter is obtained as follows: dividing the accumulated value of each sampling point by the number of sampling times.
