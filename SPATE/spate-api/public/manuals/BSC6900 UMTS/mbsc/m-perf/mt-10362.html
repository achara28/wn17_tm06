Description

These measurement items take statistics of the number of CS RABs abnormally released according to different types of service in the best cell.

Measurement point

The measurement is triggered at point A as shown in the figure. The RNC sends to the CN a RAB RELEASE REQUEST message due to exception. If the RNC receives a RAB ASSIGNMENT REQUEST message(with the cause other than "User Inactivity", "Normal Release", "Successful Relocation", "Network Optimization", or "UE Generated Signalling Connection Release", reference to 3GPP TS 25.413), the RNC measures the items according to the service types in the best cell that the UE camps on if the released RABs belong to CS domain.
