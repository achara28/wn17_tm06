Description

These measurement items take statistics of the number of failed radio link in RADIO LINK FAILURE INDICATION messages sent by DRNC on the Iur interface according to different causes.

The following table describes the meaning of each counter.

Counter Description
VS.SHO.RLFail.OM.Tx Number of failed radio links on Iur Interface Because of OM Intervention (OM Intervention)
VS.SHO.RLFail.HW.Tx Number of failed radio links on Iur Interface Because of Hardware Failure (Hardware Failure)
VS.SHO.RLFail.SyncFail.Tx Number of failed radio links on Iur Interface Because of Synchronization Failure (Synchronization Failure)
VS.SHO.RLFail.CfgUnsup.Tx Number of failed radio links on Iur Interface Because of Improper Configurations (UL SF not supported, DL SF not supported, Downlink Shared Channel Type not supported, Uplink Shared Channel Type not supported, CM not supported, Number of DL codes not supported, Number of UL codes not supported)
VS.SHO.RLFail.Other.Tx Number of failed radio links on Iur Interface Because of Other Cause (cause other than OM Intervention, Hardware Failure, Synchronization Failure, and Improper Configurations)

Measurement point

The measurement is triggered at point A as shown in the figure. When the DRNC sends the SRNC a RADIO LINK FAILURE INDICATION message with different failure causes as shown below, these items shall be measured separately on each Iur interface.
