RRC connection release measurement per cell(RRC.Rel.Cell)

Overview

When the RNC releases a RRC connection, it sends a RRC CONNECTION RELEASE message to the UE to release the resources occupied by the UE.

RRC connection release measurement provides the number of RRC connection release messages with different causes in a specified cell.
