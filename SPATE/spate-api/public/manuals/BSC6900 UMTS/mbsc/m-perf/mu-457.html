Algorithm Measurement2 for Cell(ALGO2.Cell)

Overview

Radio Resource Management (RRC) algorithm is an important function of the WCDMA system. It is used to manage cell resources effectively and to increase resource utilization. This measurement unit(ALGO2.Cell) provides measurements related to the radio resource management algorithm.
