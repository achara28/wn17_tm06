Number of Redirection to LTE Through RRC Connection Release for Cell

Measurement Counters

ID Counter Description
73423419 VS.U2LTEHO.RRCRelease.Service Number of Service-Based Redirection to LTE Through RRC Connection Release for Cell

Description

This counter provides the number of times that the RNC instructs a UE to redirect to the LTE by sending an RRC Connection Release message to the UE in the best cell.

Measurement point

The measurement is triggered at point A shown in the figure, when the RNC sends the UE an RRC Connection Release message, indicating the UE to redirect to the LTE. In this case, this counter is measured in the best cell under the SRNC.
