Description

These counters provide the failed requests for different resources in the RAB setup procedure. These counters will be measured in the cells that fail to request the resources.For DC-HSDPA users, this counter is measured only in the cell with the primary carrier.

Measurement point

The measurement is triggered at point A as shown in the Figure1.After the CN delivers the RAB ASSIGNMENT REQUEST message, if the RNC fails to request the resources, this counter will be measured in the cells that fail to request the resources according to different failure causes.
