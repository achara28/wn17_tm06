E1/T1 Electronical Port Measurement(ELECT)

Overview

The E1/T1 Electronical Ports are important resources of the Base Station Controller system. Each E1/T1 electronical port is a measurement object of the ELECT. Therefore, the E1/T1 Electronical ports can be measured for the analysis of the transmission quality of the running Base Station Controller so that the stable and reliable operation of the Base Station Controller can be maintained.
