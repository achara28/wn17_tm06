Description

These counters provide the number of RAB release requests for CS services in the best cell based on the release cause.

Measurement point

The measurement is triggered at point A as shown in the figure. When the RNC receives a RAB ASSIGNMENT REQUEST message from the CN and the RAB is for CS services, the corresponding counter is measured in the best cell that the UE camps on based on the release cause.
