Number of RRCs Released due to PS Data session end

Measurement Counters

ID Counter Description
73391137 VS.RRC.SigConnRelInd.UEPSDataEnd Number of RRCs Release Due to Receiving SIGNALLING CONNECTION RELEASE INDICATION message with cause of "UE Requested PS Data session end" for cell

Description

The measurement item takes statistics of the number of RRCs Released due to Receiving SIGNALLING CONNECTION RELEASE INDICATION message with cause of "UE Requested PS Data session end" in the best cell which the UE camps on.

Note: This counter is not used in this version but will be reused in the R15 version.

Measurement point

The measurement is triggered at point A as shown in the figure.After the UE checks that no PS data is transmitted, it sends to the RNC the SIGNALLING CONNECTION RELEASE INDICATION message with the cause value "UE Requested PS Data session end". This counter measures the numbers of RRCs Release on this situation.
