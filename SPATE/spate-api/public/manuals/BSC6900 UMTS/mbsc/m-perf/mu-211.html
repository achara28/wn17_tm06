MTP3 Link Measurement(MTP3.LNK)

Overview

MTP3 link measurement provides the counters related to MTP3(Message Transfer Part) link. The MTP3 serves as the signaling transport network layer for an Base Station Control. It is a signaling transport link between two signaling points in direct connection. There are up to 16 MTP3 links between two signaling points.

For details about the M3UA function, see the protocol ITU-T Q.704.
