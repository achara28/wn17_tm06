Number of Successful Cell Resource Requests During Channel Type Switch

Measurement Counters

ID Counter Description
67181080 VS.RAC.TrChSwitchCallAcc Number of Successful Cell Resource Requests During Channel Type Switch for Cell

Description

When the type of the channel where the UE occupies in a cell switches, the RNC requests cell resources for the UE again. In this case, the involved channel types are D2H, H2D, D2F, H2F, E2D, D2E, E2F and F2E. This counter provides the number of successful cell resource requests for the UE in a cell during the channel type switch. In addition, only the number of successful cell resource requests in a cell during the channel type switch is measured.

Measurement point

When the type of channel where the UE occupies in a cell switches, the RNC measures the counter in the cell where the UE camps on if the cell resource request succeeds.
