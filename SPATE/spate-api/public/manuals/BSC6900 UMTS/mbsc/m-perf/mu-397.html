Measurement of CS RAB release per cell(RAB.RelCs.Cell)

Overview

If a CS RAB is not needed or an abnormal release occurs after the RAB setup, the CN sends to the RNC a RAB ASSIGNMENT REQUEST message, indicating the RNC to release the RAB. After receiving this message, the RNC release the corresponding RAB and then sends to the CN a RAB ASSIGNMENT RESPONSE message.

For details, see 3GPP TS 25.331 and 3GPP TS 25.413.
