Description

These counters provide the number of cells with error codes in the E1/T1 electronical port in the measurement period.

Measurement point

Samples are taken every five seconds in the specified measurement period. The system obtains the number of cells with error codes on the E1/T1 electronical port once every five seconds and obtains the total number of cells with error codes on the E1/T1 electronical port in one measurement period.
