Measurement related to SPI per cell(SPI.Cell)

Overview

This measurement unit is used to monitor and assess the performance of the feature of Differentiated Service Based on SPI Weight. It provides the following information: (1) Number of times that the downlink throughput of HSDPA services is within each range based on the service level. (2) Number of times that the uplink throughput of HSUPA services is within each range based on the service level. (3) Number of bytes sent in downlink on HSDPA MAC-d flows and the mean downlink throughput of a single HSDPA MAC-d flow based on the service level. (4) Number of bytes received in uplink on HSUPA MAC-d flows and the mean uplink throughput of a single HSUPA MAC-d flow based on the service level.The measurement item is measured only when the SPI measure function is activated using the SET UDPUCFGDATA command with CellCountersMeasOBJCtrlSwitch set to Spi or QosAndSpi.
