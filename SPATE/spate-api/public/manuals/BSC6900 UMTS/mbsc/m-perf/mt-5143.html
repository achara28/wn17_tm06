Counters Related to Number of Bytes That the SCTP Link Receives from or Transmits to the IP Layer

Measurement Counters

ID Counter Description
67191610 VS.SCTP.RX.BYTES T6001:Number of IP Bytes Received on the SCTP Link
67191611 VS.SCTP.TX.BYTES T6002:Number of IP Bytes Transmitted on the SCTP Link
67191614 VS.SCTP.RX.MAXBYTES T6005:Maximum Number of IP Bytes Received on the SCTP Link
67191615 VS.SCTP.TX.MAXBYTES T6006:Maximum Number of IP Bytes Transmitted on the SCTP Link

Description

These counters provide the number of payload bytes of data packets that the SCTP link receives from or transmits to the IP layer in the specified measurement period.

The following table describes the meaning of each counter.

Counter Description
VS.SCTP.RX.BYTES This counter provides the number of payload bytes of data packets that the SCTP link receives from the IP layer
VS.SCTP.TX.BYTES This counter provides the number of payload bytes of data packets that the SCTP link transmits to the IP layer
VS.SCTP.RX.MAXBYTES This counter provides the maximum number of payload bytes of data packets that the SCTP link receives from the IP layer
VS.SCTP.TX.MAXBYTES This counter provides the maximum number of payload bytes of data packets that the SCTP link transmits to the IP layer

Measurement point

Number of payload bytes of data packets that the SCTP link receives from or transmits to the IP layer is sampled every five seconds. In the measurement period, the maximum sample is taken as the maximum value of this counter.At the end of the measurement period, the maximum sampling value is taken as the maximum payload and the accumulated sampling value is taken as the total payload.

Formula

None

Unit

byte
