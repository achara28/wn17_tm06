Description

These counters provide the bit rate of the data packets (including IP header and payload, but excluding FM and BR frames) in a specified measurement period. These counters are measured at the transport layer.

Measurement point

Samples are taken every five seconds in the specified measurement period. The system obtains the number of bytes received and transmitted on the link once every five seconds. With this, the system obtains the maximum number of bytes received or transmitted in five seconds, and the total number of packets in one measurement period. The maximum speed is calculated with the maximum bytes in five seconds, and the average speed is calculated with the total bytes in one measurement period.

None
