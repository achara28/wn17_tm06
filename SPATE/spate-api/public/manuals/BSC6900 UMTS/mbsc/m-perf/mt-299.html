The measurement items provide the number of RABs requested to establish according to different types of service for each operator, as shown below.

The following table describes the meaning of each counter.

Counter Description

VS.RAB.AttEstab.CS64.PLMN.RNC Number of CS 64kbits/s conversational service RABs requested to establish for each operator
VS.RAB.AttEstab.PS.PLMN.RNC Number of PS RABs requested to establish for each operator
