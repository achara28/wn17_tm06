Counters Related to the Number of CPICH Power Adjustments Due to Cell Breathing

Measurement Counters

ID Counter Description
67190846 VS.CellBreath.CPICHUp Number of Upward CPICH Power Adjustments Due to Cell Breathing for Cell
67190847 VS.CellBreath.CPICHDown Number of Downward CPICH Power Adjustments Due to Cell Breathing for Cell

Description

These counters provide the number of CPICH power adjustments due to the cell breathing in a cell of the RNC.

Measurement point

When the TCP is greater than the upper threshold configured for cell breathing or is smaller than the lower threshold configured for cell breathing, the RNC needs to adjust the power of CPICH. The RNC measures the number of increases and decreases in the CPICH power in a measurement period on a cell basis. Note: If the cell breathing algorithm switch is disabled in the measurement period, the values of these counters are zero.
