Measurement of IP Transmission Quality of Transmission Resource Pool Adjacent Node(IPPOOL.ADJNODE)

Overview

Measurement of IP Transmission Quality of Transmission Resource Pool Adjacent Node indicates the transmission quality of each adjacent node that communicates with the IP addresses in a transmission resource pool. This measurement helps analyze the transmission quality of the system.
