URA Update Measurement per cell(URAUPDATE.Cell)

Overview

A UE in the URA_PCH state may send a URA UPDATE message to the RNC for different causes, such as "change of URA" and "periodic URA update." This measurement unit is used to assess the URA update procedure by measuring the number of periodically URA update attempts, number of successful periodically URA updates, number of URA update attempts caused by URA change, and number of successful URA updates caused by URA change.
