SCCP Measurement(SCCP)

Overview

SCCP measurement provides the counters related to the SCCP originating signaling point. The SCCP provides connection-oriented and connectionless signaling transmission functions. Therefore, SCCP measurement provides the counters related to the number of connection-oriented and connectionless messages received or sent by the SCCP. It also provides the counters related to the errors and failures handled by the SCCP.

For details about the SCCP function, see the protocol ITU-T Q.704.
