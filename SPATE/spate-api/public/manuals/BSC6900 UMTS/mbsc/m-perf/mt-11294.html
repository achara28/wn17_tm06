Number of CS Outgoing Inter-RAT Handover Preparation Attempts According to Different Causes Between Neighboring Cells

Measurement Counters

ID Counter Description
73394035 VS.IRATHO.AttRelocPrepOutCS.DR.GCell Number of CS Outgoing Inter-RAT Handover Preparation Attempts (Directed retry)
73394036 VS.IRATHO.AttRelocPrepOutCS.RF.GCell Number of Failed CS Outgoing Inter-RAT Handover Preparations (Relocation desirable for radio reasons)

Description

This counters provide the number of CS outgoing inter-RAT handover preparations according to different causes.

Measurement point

The measurement is triggered at point A as shown in the figure. When the RNC sends the CN a RELOCATION REQUIRED message during the CS outgoing inter-RAT handover, the RNC measures these items for each pair of source and target cells according to different causes. Here, the best cell that the UE camps on before the inter-RAT handover is the source cell, and the cell that the UE wants to hand over to is the target cell.
