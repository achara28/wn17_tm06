Number of URA Updates due to Periodic URA Update

Measurement Counters

ID Counter Description
67180452 VS.AttURAUpd.Prd Number of URA Updates Due to Periodic URA Update for Cell

Description

This measurement item provides the number of URA UPDATE messages with the cause value of "periodic URA update" received by the SRNC in a cell.

Measurement point

The measurement is triggered at point A as shown in the figure, when the SRNC receives a URA UPDATE message with the cause value of "periodic URA update" from an UE and the UE is not involved in the RRC connection release procedure. In this case, the RNC measures the item in the cell selected by the UE during the URA update procedure.
