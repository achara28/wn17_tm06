Measurement between a cell and a intRAT cell(IRATHO.Gcell)

Overview

This measurement unit is used to measure inter-RAT handovers between neighboring cells. For details about inter-RAT handover procedure, refer to the following sections in "RNC Performance Measurement (RNC)" :

"CS Domain Inter-RAT Handover Measurement";

"PS Domain Inter-RAT Handover Measurement".

The two cells involved in the inter-RAT handover are called source cell and target cell. They belong to different communication systems. For the source cell, the RNC measures each of the following items in the relationship of "source cell - target cell".

Suppose the UE camps on CELL1 at first. After inter-RAT handover, the UE camps on CELL2. Then CELL1 is the source cell and CELL2 is the target cell. This indicates one inter-RAT handover between CELL1 and CELL2.

The RNC takes statistics for source cells that belong to the WCDMA system.

Note:

1. Using scenes of this kind of items are usually described as follows: evaluate neighboring cell priority rationality, provide evidence for deleting redundant neighboring cell, which is generally used for network optimization.

2. The measurement of this kind of item needs to be started through the M2000. The measurement is not started by default.
