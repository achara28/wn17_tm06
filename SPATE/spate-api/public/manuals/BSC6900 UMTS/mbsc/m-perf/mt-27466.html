Mean RTT of Packets Received over a Single IP Path

Measurement Counters

ID Counter Description
67192710 VS.IPPATH.RTP.AverageRtt RTT at Local End of IP Path

Description

This measurement item takes statistics of the Round Trip Time (RTT) of packets received over a single IP path.

Measurement point

When IP transport is applied to the Iu/A interface and RTCP is supported, after an RTCP packet is transmitted from the local end, RTT of the received RTCP packet can be calculated each time an RTCP packet with Reception Report Block is received in the downlink. The RTT is calculated on the basis of the time of transmitting that RTCP packet, the time of receiving this RTCP packet, and the DLSR field of the received RTCP packet. The formula is as follows: RTT = the time of receiving this RTCP packet - the time of transmitting that RTCP packet - DLSR in the received RTCP packet Then, the average RTT is calculated as follows: Average RTT =Sum of RTT/ Total times

Formula

None

Unit

ms
