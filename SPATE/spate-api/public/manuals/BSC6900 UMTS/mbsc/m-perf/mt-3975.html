Description

The measurement items provide the number of radio links unsuccessfully reconfigured by the SRNC on the Iur interface for different causes.

The following table describes the meaning of each counter.

Counter Description
VS.SHO.FailRLRecfgIur.HW.Rx Number of failed radio link reconfigurations by hardware failure on Iur interface for SRNC (cause value: Hardware Failure)
VS.SHO.FailRLRecfgIur.Cong.Rx Number of failed radio link reconfigurations caused by hardware failure on Iur interface for SRNC (cause value: Cell not Available, UL Scrambling Code Already in Use, DL Radio Resources not Available, UL Radio Resources not Available, Combining Resources not Available, Measurement Temporarily not Available, Cell Reserved for Operator Use, Transport Resource Unavailable, Control Processing Overload, or Not enough User Plane Processing Resources)
VS.SHO.FailRLRecfgIur.CfgUnsupp.Rx Number of failed radio link reconfigurations caused by insufficient RNC capability on Iur interface for SRNC (cause value: Cell not Available, UL Scrambling Code Already in Use, DL Radio Resources not Available, UL Radio Resources not Available, Combining Resources not Available, Measurement Temporarily not Available, Cell Reserved for Operator Use, Transport Resource Unavailable, Control Processing Overload, or Not enough User Plane Processing Resources)

Measurement point

The measurement is triggered at point B, as shown in the figure. When the SRNC receives a RADIO LINK RECONFIGURATION FAILURE message from the DRNC, the SRNC measures the item according to different failure causes on each Iur interface.
