Counters Related to the Number of Bytes Received, Transmitted or Discarded at the IP Layer of an IPPATH

Measurement Counters

ID Counter Description
67195400 VS.IPPATH.IPLAYER.RXBYTES T7406:Number of Bytes Received at the IP Layer of an IP Path
67195396 VS.IPPATH.IPLAYER.TXBYTES T7402:Number of Bytes Transmitted at the IP Layer of an IP Path
67195398 VS.IPPATH.IPLAYER.TXDROPBYTES T7404:Number of Bytes Discarded in Transmission at the IP Layer of an IP Path
67195402 VS.IPPATH.IPLAYER.RXDROPBYTES T7408:Number of Bytes Discarded in Reception at the IP Layer of an IP Path

Description

VS.IPPATH.IPLAYER.RXBYTES and VS.IPPATH.IPLAYER.TXBYTES provide the number of User Plane bytes received or transmitted at the IP layer of an IPPATH. VS.IPPATH.IPLAYER.RXDROPBYTES and VS.IPPATH.IPLAYER.TXDROPBYTES provide the number of bytes discarded over IP paths due to system internal faults. They indicate the traffic volume of the IPPATH at the IP layer. The number of bytes measured includes the number of bytes in the header of the IP packet and the payload.

Measurement point

Samples are taken every five seconds in the specified measurement period. The system obtains the number of bytes transmitted, received or discarded on the link once in every five seconds and obtains the total number of one measurement period.

Formula

None

byte
