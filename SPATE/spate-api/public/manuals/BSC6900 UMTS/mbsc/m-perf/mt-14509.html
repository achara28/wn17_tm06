Number of Serving Cell Change Attempts of PTT Service in Soft Handover Status

Measurement Counters

ID Counter Description
73424829 VS.PTT.SHO.ServCellChg.AttOut Number of Serving Cell Change Attempts of PTT Service in Soft Handover Status for Cell

Description

This counter measures the number of HSDPA or HSUPA serving cell change attempts (regardless of whether the new serving cell is under the same RNC or under another RNC) for PTT service in soft handover status in a cell. If a UE performs both HSDPA and HSUPA services, this counter counts the serving cell change attempts of the UE only once.

Measurement point

During an HSDPA or HSUPA serving cell change, the RNC may send any of the following messages to a UE:

[1]PHYSICAL CHANNEL RECONFIGURATION,

[2]TRANSPORT CHANNEL RECONFIGURATION,

[3]RADIO BEARER RECONFIGURATION,

[4]ACTIVE SET UPDATE.

The UE may send one of the following messages to the RNC accordingly:

[1]PHYSICAL CHANNEL RECONFIGURATION COMPLETE,

[2]PHYSICAL CHANNEL RECONFIGURATION FAILURE,

[3]TRANSPORT CHANNEL RECONFIGURATION COMPLETE,

[4]TRANSPORT CHANNEL RECONFIGURATION FAILURE,

[5]RADIO BEARER RECONFIGURATION COMPLETE,

[6]RADIO BEARER RECONFIGURATION FAILURE,

[7]ACTIVE SET UPDATE COMPLETE,

[8]ACTIVE SET UPDATE FAILURE.

Use the PHYSICAL CHANNEL RECONFIGURATION message as an example. The measurement is triggered at point A shown in the figure when the RNC sends a PHYSICAL CHANNEL RECONFIGRATION message to a UE during an HSDPA or HSUPA serving cell change. In this case, this counter is increased by one in the original serving cell of this UE. The measurement result of this counter includes serving cell changes in soft handover status instead of the hard handover status.
