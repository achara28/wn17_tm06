RRC connection release measurement per RNC(RRC.Rel.RNC)

Overview

When the RNC release a RRC connection, it sends a RRC CONNECTION RELEASE message to the UE to release the resources related to the UE. The RRC connection release measurement counts the RRC connection release messages of the entire RNC by the RRC connection release cause.
