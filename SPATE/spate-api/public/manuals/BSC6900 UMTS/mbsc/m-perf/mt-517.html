Description

These counters provide the number of user cells received, transmitted or Discarded on an Fractional IMA Link in the measurement period to indicate the traffic volume of the Fractional IMA Link.

The following table describes the meaning of each counter.

Counter Description
VS.FRACIMALNK.TXUSERCELLS Number of user cells transmitted on an Fractional IMA Link
VS.FRACIMALNK.TXSTUFFCELLS Number of filling cells transmitted on an Fractional IMA Link
VS.FRACIMALNK.RXUSERCELLS Number of user cells received on an Fractional IMA Link
VS.FRACIMALNK.RXSTUFFCELLS Number of filling cells received on an Fractional IMA Link
VS.FRACIMALNK.RXDROPPEDCELLS Number of received cells discarded on an Fractional IMA Link due to errors in the link
VS.FRACIMALNK.RXHCSERRCELLS Number of cells with error HCS (Header Check Sequence) received on an Fractional IMA Link

Measurement point

Samples are taken every five seconds in the specified measurement period. The system obtains the number of IEs transmitted, received or discarded on the link once every five seconds and obtains the total number of IEs in one measurement period.
