Description

These counters provide the number of RRC CONNECTION RELEASE messages with different causes on CCCH sent by the RNC to the UE.

Measurement point

The measurement is triggered at point A, as shown in the figure. When the RNC sends a RRC CONNECTION RELEASE message with a certain cause to the UE to release a RRC connection, the corresponding counter is measured in the cell that the UE camps on. For details on the different causes, see section 10.3.3.32 in 3GPP TS 25.331.
