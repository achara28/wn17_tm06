Number of Outgoing SRNS Relocation Executions with HHO Between Neighboring Cells

Measurement Counters

ID Counter Description
73394026 VS.HHO.AttOutInterRNCCN.CS.NCell Number of Outgoing UE-Involved SRNS Relocation Executions Triggered by CS HHO
73394030 VS.HHO.AttOutInterRNCCN.PS.NCell Number of Outgoing UE-Involved SRNS Relocation Executions Triggered by PS HHO

Description

This counters provide the number of outgoing UE-involved SRNS relocation executions triggered by hard handover. Relocation execution is the process that the SRNC Sends to the UE reconfiguration messages after the SRNC relocation prepration succeeds. The reconfiguration involves the following messages: RADIO BEARER SETUP, RADIO BEARER RECONFIGURATION, RADIO BEARER RELEASE, TRANSPORT CHANNEL RECONFIGURATION, and PHYSICAL CHANNEL RECONFIGURATION. The originating cell and the target cell of the hard handover should under different RNCs.

Measurement point

Take the RADIO BEARER RECONFIGURATION message as an example, the measurement is triggered at point A as shown in the figure. When the SNRC sends the UE a RADIO BEARER RECONFIGURATION message during the UE-involved SRNS relocation triggered by hard handover, the counter is measured in both the best cells that the UE camps on before and after the hard handover if the originating cell and the target cell are under different RNCs.
