Description

This counter provides the Forward Packet Loss Rate in the specified measurement period.

The forward packet loss rate is calculated as follows:

(Number of packets transmitted by BSC - Number of packets received by Base Station) / Number of packets transmitted by BSC The backward packet loss rate is calculated as follows:

(Number of packets transmitted by Base Station - Number of packets received by BSC) / Number of packets transmitted by Base Station

Measurement point

Samples are taken every five seconds in the specified measurement period. The system obtains the number of discarded packtes once every five seconds. With this, the system obtains the total number of discarded packtes in one measurement period. The loss rate is calculated by dividing the discarded packtes with total packtes.

None
