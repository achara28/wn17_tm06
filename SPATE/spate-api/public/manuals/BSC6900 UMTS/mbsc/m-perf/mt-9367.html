Counters Related to the Number of Qoffset Updates Due to Load Level Change

Measurement Counters

ID Counter Description
67189856 VS.PUC.High.Offset.Updt Number of Qoffset Updates Due to Heavy Load for Cell
67189857 VS.PUC.Light.Offset.Updt Number of Qoffset Updates Due to Light Load for Cell
67189858 VS.PUC.Norm.Offset.Updt Number of Qoffset Updates Due to Normal Load for Cell

Description

The selection/reselection parameter Qoffset in system information block (SIB) 11 defines the selection/reselection offset between two cells. By Qoffset, the network can direct a UE to camp on one of the cells. When the load of the serving cell is light, the RNC can set greater values for Qoffset between the serving cell and the neighboring cells by updating system information. Thus, the UE in idle mode is not apt to reselect another cell in heavy load. When the load of the serving cell is heavy, the RNC can set smaller values for Qoffset between the serving cell and the neighboring cells by updating system information. Thus, the UE in idle mode is apt to reselect another cell in light load, which can reduce the potential load of the current cell. These counters provide the number of Qoffset updates when the load of the serving cell becomes light, normal, and heavy respectively.
