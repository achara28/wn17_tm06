Measurement Related to VCL PM on an AAL2 Path(AAL2PATH.VCLPM)

Overview

This measurement uses FPM and BR cells to monitor the ATM transmission quality of a path. In the source direction, FPM cells are sent periodically to indicate the number of transmitted cells and the number of error cells that failed the Bit Interleaved Parity 16 (BIP16) check. In the sink direction, BR cells are transmitted as response to the FPM cells. The BR cells calculate the number of received cells and the number of error cells that fail the BIP16 check. In the source direction, BR cells are used to calculate the number of discarded cells and the number of cells that are inserted by mistake. In the sink direction, FPM cells are used to calculate the number of discarded cells and the number of cells that are inserted by mistake.
