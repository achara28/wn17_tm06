Description

These counters provide the number of failed resource allocations for TRNC relocation according to different failure causes.

Measurement point

The measurement is triggered at point A as shown in the following figure. During the relocation preparation, when the resource allocation for TRNC relocation fails, the TRNC sends a RELOCATION FAILURE message to the CN. In this case, the TRNC measures the related counter according to different failure causes.
