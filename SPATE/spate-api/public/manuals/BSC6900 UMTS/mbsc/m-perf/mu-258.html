SAAL Link Measurement(SAAL.LNK)

Overview

The SAAL link measurement provides the reliable function of transmitting the signaling message in point-to-point mode for the RNC. The signaling messages from the RNC to a NodeB, CN, or neighboring RNC (NRNC) are all transmitted through the SAAL link. The SAAL link to a NodeB is called User-to-Network Interface (UNI) SAAL link. The SAAL link to the CN or an NRNC is called Network-to-Network Interface (NNI) SAAL link.

For details about the M3UA function, see the protocol ITU-T Q.2110.
