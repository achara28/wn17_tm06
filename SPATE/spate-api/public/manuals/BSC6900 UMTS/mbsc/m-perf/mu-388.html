Measurement related to QOS per cell(QOS.Cell)

Overview

This measurement unit contains the following contents:1)Delay from RRC establishment to completion of RAB establishment.2)Delay of service establishment by service types and CN domains.3)Mean throughput count by user class and HSDPA download throughput.4)Mean throughput count by user class and HSUPA upload throughput.
