Description

These counters provide the average number of AMR users with different UL and DL rates in the RNC.

Measurement point

The number of AMR users with each UL or DL rate is sampled every five seconds in the RNC. At the end of the measurement period, the value of each counter is obtained as follows: dividing the accumulated value of each sampling point by the number of sampling times.
