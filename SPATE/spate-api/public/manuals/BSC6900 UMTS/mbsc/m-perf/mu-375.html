Cell Update Measurement per cell(CELLUPDATE.Cell)

Overview

An UE may send a CELL UPDATE message to the RNC for different causes, such as uplink data transmission, paging response, radio link failure, re-entering service area, RLC unrecoverable error, cell reselection, and periodical cell update.

This measurement unit is used to assess the cell update procedure by measuring the number of cell updates triggered by different causes, and the number of successful or failed cell updates triggered by each cause.
