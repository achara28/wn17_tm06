Transmission Resource Pool Local IP Address IP Layer Measurement(IPPOOL.SIP.IPLAYER)

Overview

Transmission Resource Pool Local IP Address IP Layer Measurement indicates the IP layer traffic of each Local IP address that communicates with the IP addresses in a transmission resource pool. This measurement helps analyze the transmission quality of the system.
