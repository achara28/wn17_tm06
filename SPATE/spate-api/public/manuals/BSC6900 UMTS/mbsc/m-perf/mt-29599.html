Description

These counters provide the number of cells received or transmitted on a VC cross connection at the ATM layer in the specified measurement period.

Measurement point

Samples are taken every five seconds in the specified measurement period. The system obtains the number of cells transmitted, received or discarded on the link once every five seconds and obtains the total number of one measurement period.
