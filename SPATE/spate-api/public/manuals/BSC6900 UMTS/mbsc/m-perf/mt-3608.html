Counters Related to the Number of Packets Received, Transmitted or Dropped at the IP Layer of an IPPATH

Measurement Counters

ID Counter Description
67195399 VS.IPPATH.IPLAYER.RXPACKETS T7405:Number of Packets Received at the IP Layer of an IP Path
67195395 VS.IPPATH.IPLAYER.TXPACKETS T7401:Number of Packets Transmitted at the IP Layer of an IP Path
67195397 VS.IPPATH.IPLAYER.TXDROPPACKETS T7403:Number of Packets Discarded in Transmission at the IP Layer of an IP Path
67195401 VS.IPPATH.IPLAYER.RXDROPPACKETS T7407:Number of Packets Discarded in Reception at the IP Layer of an IP Path

Description

VS.IPPATH.IPLAYER.RXPACKETS and VS.IPPATH.IPLAYER.TXPACKETS provide the number of User Plane packets received or transmitted at the IP layer of an IPPATH in the measurement period. VS.IPPATH.IPLAYER.RXDROPPACKETS and VS.IPPATH.IPLAYER.TXDROPPACKETS provide the number of packets discarded over IP paths due to system internal faults. They indicate the traffic volume of the IPPATH at the IP layer.

Measurement point

Samples are taken every five seconds in the specified measurement period. The system obtains the number of packets transmitted, received or discarded on the link once in every five seconds and obtains the total number of one measurement period.

Formula

None

packet
