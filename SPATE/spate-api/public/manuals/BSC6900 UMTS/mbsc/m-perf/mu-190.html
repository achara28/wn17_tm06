Measurement related to PS Iu interface signalling(PS.SIG.IU)

Overview

This measurement unit aims at the connectionless signaling procedure and the signaling connection setup and release procedures on the Iu interface. In Iu Flex connection mode, the RNC measures the counters according to the PS CN nodes.
