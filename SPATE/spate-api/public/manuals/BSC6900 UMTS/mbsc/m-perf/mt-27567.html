Number of HSDPA Signaling Bytes on Iur Interface in SRNC

Measurement Counters

ID Counter Description
73410479 VS.SRNCIurBytesSigHSDPA.Tx Number of HSDPA Signaling Bytes on Iur Interface in SRNC

Description

This counter provides the number of the DL signaling bytes that are sent to the DRNC from the SRNC over the HSDSCH FP on the Iur interface.

Measurement point

The measurement is triggered when the SRNC sends DL signaling over the HSDSCH FP carrying the signaling on the Iur interface to the DRNC. The FP header is not included. The measurement is performed respectively on the basis of Iur interfaces.

Formula

None

Unit

byte
