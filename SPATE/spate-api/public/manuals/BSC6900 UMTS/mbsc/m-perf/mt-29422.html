Number of Downlink Signaling PDUs Discarded on AM RLC

Measurement Counters

ID Counter Description
67204864 VS.RLC.AM.Disc.HsdpaSig.PDU Number of HSDPA Downlink Signaling PDUs Discarded on AM RLC for Cell
67204811 VS.RLC.AM.Disc.R99Sig.PDU Number of R99 Downlink Signaling PDUs Discarded on AM RLC for Cell

Description

These counters provide the number of downlink signaling PDUs that are discarded on the AM RLC.

The Number of DL HSDPA Signaling PDUs Discarded on AM RLC for Cell is measured for the HSDPA serving cell, and the Number of DL R99 Signaling PDUs Discarded on AM RLC for Cell is measured for cells in the active set.

Measurement point

The measurement is triggered when a downlink signaling PDU is discarded on the AM RLC.

Formula

None
