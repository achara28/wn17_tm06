Measurement between a cell and a UMTS cell(CELL.Ncell)

Overview

Neighboring Cell Measurement is used to measure the handover procedures between neighboring cells. These handover procedures include soft handover procedures and hard handover procedures. For the details about the procedures, refer to "Soft Handover Measurement" and "Hard Handover Measurement" in "RNC Performance Measurement (RNC)".

The two cells involved in neighboring cell measurement are called source cell and target cell. For the source cell, the RNC measures each of the following items in the relationship of "source cell - target cell". If there are several target cells, the RNC measures the items in "source cell - target cell n".

For example, the UE is in CELL1 then moves to CELL2 and CELL3 after soft handover. In this scenario, CELL1 is the source cell, CELL2 and CELL3 are the target cells. You can measure the soft handovers as follows: one soft handover for "CELL1- CELL2" and another for "CELL1-CELL3".

Neighboring cell measurement is used to measure only the items with the source cell under a Source Radio Network Controller (SRNC) and the target cell under an SRNC or a Target Radio Network Controller (TRNC).

Note:

1. Using scenes of this kind of items are usually described as follows: evaluate neighboring cell priority rationality, provide evidence for deleting redundant neighboring cell, which is generally used for network optimization.

2. The measurement of this kind of item needs to be started through the M2000. The measurement is not started by default.

3. In BSC6900 versions, there is no restriction on the number of measurement objects.
