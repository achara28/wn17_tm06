Description

This counter provides the number of times that the mean uplink throughput of the BE services falls within each range based on the service priority in a measurement period in the HSUPA serving cell.For DC-HSUPA users, this counter is measured only in the primary cell based on the total throughput on the primary and secondary carriers.

Measurement point

In the HSUPA serving cell, the mean uplink throughput of services whose SPIs are set to 8 are measured in a measurement period. When the throughput falls within a certain range, the corresponding counter is incremented by one. The value of each counter is obtained.
