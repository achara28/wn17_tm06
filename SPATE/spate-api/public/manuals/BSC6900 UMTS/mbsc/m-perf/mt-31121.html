Description

These counters provide the number of bytes that are sent on downlink HSDPA MAC-d flows carrying BE services with different SPIs. For DC-HSDPA users, this counter is measured only in the primary cell based on the total number of bytes sent on the primary and secondary carriers.

Measurement point

A counter is measured when the data of a BE service with the corresponding SPI is sent at the RLC layer on the downlink. The measured data excludes the RLC headers and retransmitted data.
