Description

These counters provide the number of failed PS relocation preparations (UE not involved) initiated by the SRNC according to different failure causes.

Measurement point

The measurement is triggered at point A as shown in the following figure. After the SRNC initiates a relocation (UE not involved), the outgoing relocation preparation fails if the SRNC receives a RELOCATION PREPARATION FAILURE message from the CN in the PS domain. In this case, the SRNC measures the related counter according to different failure causes.
