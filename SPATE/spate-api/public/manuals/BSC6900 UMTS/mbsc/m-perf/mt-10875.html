Number of RRC Connection Rejects Due to Radio Link Setup Failure

Measurement Counters

ID Counter Description
67179525 VS.RRC.Rej.RL.Fail Number of RRC Connection Rejects Due to Radio Link Setup Failure for Cell

Description

This counter provides the number of RRC CONNECTION REJECT messages sent by the RNC to the UE due to RL setup failure on the Iub interface after the RNC receives a RRC CONNECTION REQUEST message from the UE. Note that if the NodeB responses with a radio link configuration failure message with the cause of CE resource insufficiency, this counter is not measured.

Measurement point

The measurement can be triggered at point B as shown in the figure. When the RNC receives a RRC CONNECTION REQUEST message from the UE, the RNC starts a radio link setup procedure. After receiving a RADIO LINK SETUP FAIL message from the NodeB, the RNC sends a RRC CONNECTION REJECT message to the UE. In this case, this counter is measured in the cell where the UE is located.

Figure 1 Radio link setup failure

The measurement can be also triggered at point B as shown in the figure. When the RNC receives a RRC CONNECTION REQUEST message from the UE, the RNC starts a radio link setup procedure. If the RNC fails to receive a RADIO LINK SETUP RESPONSE message from the NodeB before the timer expires, the RNC sends a RRC CONNECTION REJECT to the UE. In this case, this counter is measured in the cell where the UE is located.

Figure 2 Radio link setup timeout
